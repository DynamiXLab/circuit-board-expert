# 项目2 · 直流电机驱动模块

> **封装类型**：🔌 全插件（直插/Through-Hole）  
> **难度**：★★☆☆☆  
> **画板时间**：1周（约5天，每天2-3小时）  
> **核心教学目标**：理解功率电路与信号电路在同一块板上的设计差异，学会计算走线宽度、处理散热。

---

## 1. 项目简介

流水灯是纯信号电路（电流都是mA级别）。这次我们面对一个真正有"力气"的电路——驱动直流电机。

电机需要的电流远大于LED（小电机空载几十mA，带载几百mA到几A），所以这块板子上同时存在：
- **信号部分**：控制逻辑，电流小、对干扰敏感
- **功率部分**：电机电流，电流大、会产生干扰

如何让这两部分在同一块板子上和平共处，是项目2的核心课题。

我们选用 **L298N** 作为驱动芯片——它是经典的插件H桥驱动器，MultiWatt-15封装（竖直安装，自带散热片安装孔），单个芯片可以驱动两个直流电机。

### 1.1 你将学到什么

- H桥驱动直流电机的原理（正转/反转/刹车/滑行）
- 功率走线宽度与电流的关系
- 逻辑电源和电机电源分离的设计
- 续流二极管的作用
- 插件时代最后一次手工焊接——准备过渡到贴片

## 2. 认识新元件

### 2.1 L298N 双H桥驱动器

**实物**：黑色长方形塑料封装，15个引脚排成一排（竖直焊接在PCB上），背面有金属散热片（带安装孔，可加散热器）。

**L298N引脚功能**（只需知道接什么）：

| 引脚 | 名称 | 功能 |
|------|------|------|
| 1 | SEN_A | A路电流检测——本项目直接接地 |
| 2 | OUT1 | A路输出1——接电机A一端 |
| 3 | OUT2 | A路输出2——接电机A另一端 |
| 4 | VS | **电机电源**（最高46V，本项目用5~12V） |
| 5 | IN1 | A路控制输入1（接MCU GPIO或开关） |
| 6 | EN_A | A路使能（PWM调速接这里，全速就接高电平） |
| 7 | IN2 | A路控制输入2 |
| 8 | GND | 地（**功率地和逻辑地在芯片内部已经连通**） |
| 9 | VSS | **逻辑电源** 5V（给芯片内部逻辑供电） |
| 10 | IN3 | B路控制输入3 |
| 11 | EN_B | B路使能 |
| 12 | IN4 | B路控制输入4 |
| 13 | OUT3 | B路输出3 |
| 14 | OUT4 | B路输出4 |
| 15 | SEN_B | B路电流检测——本项目直接接地 |

**真值表**（以A路为例，B路同理）：

| EN_A | IN1 | IN2 | 电机状态 |
|------|-----|-----|---------|
| H | H | L | **正转** |
| H | L | H | **反转** |
| H | L | L | **刹车**（快速停止） |
| H | H | H | **刹车** |
| L | × | × | **滑行**（自由停） |

H=高电平(5V)，L=低电平(0V)，×=任意

> **一句话**：EN决定转不转，IN1/IN2决定往哪边转。

### 2.2 续流二极管

**为什么要续流二极管？** 电机是一个电感——断电时电感中的电流不能突变，会产生很高的反向电压（反电动势），可能击穿L298N内部的开关管。

每个输出脚对VS、对GND各接一个二极管，一共需要8个。常用的型号是 **1N5819（肖特基二极管，正向压降低、开关速度快）** 或 **1N4007**。

本项目用 **1N5819**——插件封装 DO-41，外观和1N4007类似但性能更适合。

### 2.3 大容量电解电容

电机启动瞬间需要很大的冲击电流，如果电源响应不过来，电压会瞬间跌落导致芯片复位。

解决方案：在电机电源（VS）和GND之间并一个 **大电解电容（470μF~1000μF）**——它像一个蓄水池，电机要电时先从电容取。

**注意耐压**：如果电机用12V，电容耐压至少选25V。

### 2.4 接线端子

电机线比较粗，不适合直接焊在排针上，应该用**螺钉式接线端子**（插件，脚距5.08mm）。电源输入同理。

## 3. 电路原理

### 3.1 总体框图

```
        逻辑电源(5V)           电机电源(5~12V)
            │                      │
            ▼                      ▼
        ┌──────┐              ┌──────┐
  MCU──→│ L298N │─────────────→│ 电机A │
  GPIOs  │      │─────────────→│ 电机B │
         └──────┘              └──────┘
            │
           GND (两路电源共地)
```

### 3.2 电源部分

**为什么要两路电源？**

- VSS（逻辑电源，5V）：给L298N内部逻辑电路供电，电流很小（几十mA）
- VS（电机电源，5~12V）：给电机供电，电流大（几百mA~2A）

两路电源必须**共地**（GND连在一起），否则控制信号没有参考。

但在PCB上，两路电源的走线要分开——功率电流回路独立，不要走信号地经过。

### 3.3 控制接口

把IN1~IN4、EN_A、EN_B用排针引出来。本阶段先不接MCU（项目5才首次碰MCU），控制信号可以用：
- 排针+杜邦线+跳线帽手动接高/低电平
- 或者用拨码开关+上拉电阻

一个简单的测试方案：

```
IN1: 接VCC（通过跳线帽）     → 正转
IN2: 接GND（通过跳线帽）
EN_A: 接VCC（通过跳线帽）    → 全速
```

## 4. 原理图绘制

### 4.1 放置L298N

在立创EDA搜索 `L298N`，选 MultiWatt-15 封装的版本。注意封装名通常是 `MULTIWATT-15` 或 `Multiwatt-15_Vertical`——竖直焊接型。

### 4.2 连接续流二极管

8个二极管分布在4个输出脚周围：

```
每个输出脚（OUTx）：
- 一个二极管阳极接OUTx，阴极接VS（→ 当OUTx电压>VS+0.3V时导通，保护上管）
- 一个二极管阳极接GND，阴极接OUTx（→ 当OUTx电压<-0.3V时导通，保护下管）
```

在原理图上画成4组，每组两个二极管反向并联在输出脚和电源轨之间。

### 4.3 电源去耦

- VS引脚旁并一个 **470μF 电解** + 一个 **0.1μF 瓷片**（大电容储能、小电容滤高频噪声）
- VSS引脚旁并一个 **100μF 电解** + **0.1μF 瓷片**

### 4.4 控制信号排针

- IN1~IN4、EN_A、EN_B → 排针H1（6脚，一排在板边）
- VSS(5V)和GND → 排针H2（2脚）
- SEN_A、SEN_B → 直接接地

## 5. PCB布局布线

### 5.1 布局

```
┌──────────────────────────────────────┐
│  [电源端子]  [电机A端子]  [电机B端子]  │  ← 功率区域，板子一侧
│                                      │
│    L298N（竖直安装）                  │
│    续流二极管围在L298N旁边            │
│    大电解电容紧靠VS引脚               │
│                                      │
│  [控制排针]  [逻辑电源排针]           │  ← 信号区域，板子另一侧
└──────────────────────────────────────┘
```

**布局要点**：
- L298N放在中间偏上，留出散热器空间
- 功率端子（电源进、电机出）都在板子同一侧
- 控制排针在板子另一侧——物理隔离
- 8个续流二极管紧挨L298N的4个输出脚
- 大电解电容紧靠L298N的VS和GND引脚

### 5.2 走线宽度计算

**关键字：铜厚1oz（35μm），线宽1mm可承载约1A电流（温升10°C）**

| 走线 | 最大电流估算 | 推荐线宽 |
|------|------------|---------|
| VS电源线 | ~2A（双电机满载时） | **2mm** 以上 |
| OUTx输出线 | ~1A（单个电机） | **1mm** |
| GND（功率部分） | ~2A | **2mm** 或直接铺铜 |
| INx信号线 | <1mA | 0.3mm |
| VSS逻辑电源 | ~50mA | 0.3mm |

### 5.3 铺铜策略

- 不在L298N底下整块铺铜——留出散热空间（开窗或不铺）
- 功率GND大面积铺铜在底层
- 如果担心单点接地不好处理，可以直接把功率地和逻辑地铺成一块铜——L298N内部已经把两个地连起来了

### 5.4 散热处理

- L298N竖直安装，背面金属片可以加散热器（买L298N模块时通常带一个铝散热片）
- PCB上在L298N周围不要铺铜太密，留出空气流通空间
- 如果板子较大，可以在L298N附近打几个大过孔帮助散热

## 6. 打样与焊接

打样流程同项目1——双层板、绿油白字、1oz铜厚。

焊接注意事项：
- L298N的15个引脚间距较大，手工焊接不难
- 先焊矮的元件（二极管、电阻），再焊接线端子，最后焊L298N
- 8个1N5819注意方向——**色环端是阴极**
- 电解电容注意极性
- 上电前必测VCC-GND是否短路

## 7. 测试

### 7.1 静态检查

用万用表测：
- VS和GND之间阻值（应该很大，不是短路）
- VSS和GND之间阻值
- 每个二极管的方向是否正确

### 7.2 上电测试

1. 接VSS=5V（逻辑电源），VS先不接
2. 测L298N的VSS脚是否有5V
3. 接VS=5V~12V（电机电源），先不接电机
4. 用跳线帽设置 IN1=H, IN2=L, EN_A=H
5. 万用表电压档测OUT1和OUT2之间是否有电压（应≈VS）
6. 断电，接上电机，重新上电，电机应该转起来
7. 改变IN1/IN2的电平，验证正反转

## 8. 常见问题

| 现象 | 可能原因 |
|------|---------|
| 电机不转 | VS没电/EN没拉高/接线端子接触不良 |
| L298N发烫严重 | 电机电流太大（>2A）/没加散热器/供电电压太高 |
| 电机只往一个方向转 | IN1或IN2有一路没接对 |
| 接上电机后MCU复位 | 电机干扰反弹到逻辑电源——VSS加粗去耦电容 |
| 芯片烧了 | VS和VSS接反/没有续流二极管/电机堵转电流过大 |

## 10. 拓展思考：H桥详解

### 10.1 什么是H桥？

H桥由4个开关（MOSFET或BJT）组成，形状像字母H——上下两个开关控制电机的一端。

```
     VS
     │
  ┌──┴──┐     ┌──┴──┐
  │ S1  │     │ S3  │
  └──┬──┘     └──┬──┘
     ├── 电机 ──┤
  ┌──┴──┐     ┌──┴──┐
  │ S2  │     │ S4  │
  └──┬──┘     └──┬──┘
     │
    GND
```

- S1+S4导通 → 电流从左到右 → **正转**
- S2+S3导通 → 电流从右到左 → **反转**
- S1+S3导通（或S2+S4）→ 电机两端同为VS或同为GND → **刹车**
- 全关 → **滑行**

### 10.2 为什么不能同时导通S1+S2（或S3+S4）？

那会把VS直接短路到GND——**上下管直通**，瞬间大电流烧芯片。L298N内部有逻辑防止这一点，但自己搭分立H桥时一定要注意。

### 10.3 L298N vs 现代驱动芯片

L298N是BJT工艺的老芯片，导通压降大（约2V），发热厉害，效率不高。但它耐压高（46V）、皮实耐操，适合教学。

现代方案（如TB6612、DRV8833）用MOSFET，导通电阻低（<0.5Ω），效率高得多。等你们过渡到贴片时代会接触到。
