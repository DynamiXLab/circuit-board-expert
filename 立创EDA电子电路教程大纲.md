# 基于立创EDA的电子电路设计入门教程大纲

> **适用对象**：刚从高中进入大学实验室的零基础同学  
> **核心目标**：学会画PCB，能独立完成一块电路板从原理图到焊接调试的全流程  
> **最终产出**：一块基于单片机的多传感器巡线小车主板

---

## 课程总览

```
第0周   ──  项目0: 立创EDA基础操作与快捷键  🖥️ 工具入门
第1-2周 ──  项目1: NE555流水灯              🔌 插件元件 · 详细元件认知
第3-4周 ──  项目2: 直流电机驱动模块     🔌 插件元件
第5-6周 ──  项目3: 灰度传感器模块       📐 贴片元件(0603/SOP)
第7-8周 ──  项目4: 电源管理模块（LDO & DC-DC）  📐 贴片元件
第9-10周──  项目5: 单片机最小系统板     📐 贴片元件(LQFP/0603)
第11-16周── 毕业项目: 多传感器巡线小车  📐 全贴片系统板
```

> **封装路线**：前两个项目全部使用**插件（直插/Through-Hole）**元件——引脚粗、间距大、手工焊接友好，适合零基础入门。从项目3开始全面切换到**贴片（SMD）**元件，学习0603阻容、SOP芯片的焊接。毕业项目使用全贴片设计，部分芯片进入LQFP封装。

**每个项目的节奏**

```
周一~周五（画板周）    原理图 + PCB Layout + 打样下单
周六~下周五（等板周）  理论学习 + 上一个项目焊接调试
下周末收板            焊接 + 上电测试 + 复盘
```

等板期间不开启新项目的PCB绘制——手上有且只有一块板在设计、一块板在打样。

## 项目1 · NE555流水灯电路（第1-2周）

> **选它的理由**：纯分立元件，不需要单片机/编程，电路经典、现象直观、成功率高。第一次焊接点亮十个LED跑起来的成就感是入门最重要的事。

### 第1周：原理图 + PCB

| 天 | 内容 |
|----|------|
| 周一 | 讲解NE555多谐振荡器原理 + CD4017十进制计数器原理；在原理图上画出时钟发生 + 计数输出两级 |
| 周二 | 完成完整原理图：10路LED + 限流电阻 + 电源接口 + 滤波电容；学习添加封装（LED 5mm直插、DIP-8、DIP-16） |
| 周三 | 原理图转PCB，讲解布局三原则（信号流从左到右、接插件靠边、LED按圆周/直线排布）；开始手动布局 |
| 周四 | 布线教学——自动布线只做演示，手动布线为主线：走线宽度规则（信号线0.3mm、电源线0.5mm）、过孔放置、泪滴 |
| 周五 | DRC检查 + 铺铜（GND平面）+ 丝印调整 + 导出Gerber，在立创下单助手下单（选嘉立创打样，双层板5片约5元包邮） |

**电路知识点**：

- NE555 多谐振荡器（频率公式 f = 1.44 / ((R1+2R2)·C)）
- CD4017 十进制计数器/脉冲分配器
- LED 限流电阻计算
- 去耦电容的作用

### 第2周：等板期安排

| 天 | 内容（不超前学下一项目） |
|----|--------------------------|
| 周一 | 焊接工具培训：烙铁使用、焊锡选择、助焊剂、吸锡带；在洞洞板上练习焊点（先焊电阻再焊IC座） |
| 周二 | 元器件辨识：色环电阻读数、电容（瓷片/电解/钽）、二极管正负极、IC方向标记 |
| 周三 | 万用表使用：通断档、电压档、电阻档；讲解"上电前先测电源与地是否短路"的铁律 |
| 周四~五 | 消化项目1原理：用面包板搭出流水灯并验证（可选）；复盘PCB设计中的问题和改进空间 |

**收板后**：焊接NE555流水灯板，上电测试，看到LED依次点亮即算通过。

## 项目2 · 直流电机驱动模块（第3-4周）

> **选它的理由**：电机是最常见的执行器。H桥驱动涉及功率电路、大电流走线、散热——第一次面对"信号"和"功率"在同一块板上的设计。

### 第3周：原理图 + PCB

| 天 | 内容 |
|----|------|
| 周一 | 讲解直流电机原理和H桥原理（4个MOSFET如何控制正反转和调速）；对比分立H桥 vs 集成驱动芯片（L298N vs TB6612 vs DRV8833），本项目选用TB6612 |
| 周二 | 完成原理图：TB6612 + 电机端子 + 逻辑电源/电机电源分离 + 大电容滤波 + 续流二极管 |
| 周三 | 讲解PCB关键规则：功率走线宽度的计算（铜厚1oz时1mm≈1A）、星形接地、大电流回路面积最小化 |
| 周四 | 布局布线：TB6612放在板边利于散热、电机线和信号线排在不同侧、铺铜区域划分 |
| 周五 | DRC + 丝印标注Vmotor/Vlogic/A相/B相 + 下单打样 |

**电路知识点**：

- H桥的四种状态（正转/反转/刹车/滑行）
- 电机反电动势与续流二极管
- 逻辑电源与功率电源隔离
- PWM调速的基本概念（占空比）

### 第4周：等板期安排

| 天 | 内容 |
|----|------|
| 周一 | 焊接项目1的板子（如果还没焊完）+ 测试电机驱动板的理论波形 |
| 周二 | 讲解PWM深入原理：频率、占空比、死区时间；用示波器看PWM波形（如有条件） |
| 周三 | 常见电机类型：直流有刷、步进电机、舵机、无刷电机的特点和适用场景 |
| 周四~五 | 用面包板+现成模块搭一个小车雏形，体会"控制器→驱动→电机"的信号链 |

**收板后**：焊接电机驱动模块，用信号发生器或MCU输出PWM驱动电机正反转，验证功能。

## 项目3 · 灰度传感器模块（第5-6周）

> **选它的理由**：这是巡线小车的核心传感器。涉及模拟电路——红外发射、光电接收、比较器阈值——让学生第一次接触"模拟信号调理链"。

### 第5周：原理图 + PCB

| 天 | 内容 |
|----|------|
| 周一 | 讲解红外对管原理：IR LED发射 → 地面反射 → 光电三极管接收；白线/黑线反射率差异 |
| 周二 | 完成原理图：红外发射管 + 限流电阻 + 光电三极管 + 上拉电阻 + LM393比较器 + 电位器阈值调节 + 指示灯LED |
| 周三 | 讲解比较器滞回（迟滞）原理——避免临界值振荡；单路传感器完成再阵列化为4路/5路 |
| 周四 | PCB布局：传感器头伸出板边（或做独立小板+排线），电位器放在便于调节的位置，模拟地与数字地分离 |
| 周五 | DRC + 下单 |

**电路知识点**：

- 红外光的发射与接收特性
- 光电三极管的工作区（截止区/放大区/饱和区）
- 比较器的开环增益与响应时间
- 滞回比较器（Schmitt Trigger）的上下阈值计算
- 为什么需要电位器调节阈值（不同地面、不同高度）

### 第6周：等板期安排

| 天 | 内容 |
|----|------|
| 周一 | 焊接项目2的板子 + 测试电机 |
| 周二 | 讲解传感器信号链：传感器→调理电路→ADC/GPI/O→MCU→决策→执行器 |
| 周三 | AD/DA基础：分辨率、采样率、量化误差；为什么比较器输出是数字量而光电三极管是模拟量 |
| 周四~五 | 用现成灰度模块+Arduino开发板快速搭建巡线原型，理解"读到黑线→对应轮子减速"的控制逻辑 |

**收板后**：焊接灰度模块，调节电位器，在白纸黑线上测试输出信号跳变。

## 项目4 · 电源管理模块（LDO & DC-DC）（第7-8周）

> **选它的理由**：电源是每一个电路的基础。两种最常用的稳压方案——LDO（线性稳压）和DC-DC（开关稳压）——各有长短，理解它们的原理和区别是设计任何电子系统的基本功。这个模块将在巡线小车上直接使用（电池→5V→3.3V），同时它也是全教程第一个**闭环控制系统**的实物案例。

### 第7周：原理图 + PCB

| 天 | 内容 |
|----|------|
| 周一 | 先补基础知识：欧姆定律、电容充电、电感惯性、PWM、开环vs闭环（结合流水灯/电机驱动讲）。然后讲两种稳压方案：LDO原理（智能可变电阻/水龙头比喻）vs Buck原理（高速开关+电感/飞轮比喻） |
| 周二 | LDO电路设计：662K（SOT-23-3小封装LDO）原理图——两个1μF电容就够了、压差与功耗计算（公式逐个汉字注释）、与AMS1117的对比选型。DC-DC Buck原理详解——四要素的"分阶段工作过程"图解 |
| 周三 | 完成MP1584完整原理图：电感选型、反馈电阻分压计算（教学生自己算）、输入输出电容、续流二极管、软启动、补偿网络。讲解Buck电路的闭环反馈回路 |
| 周四 | **重点**：开关电源PCB布局四铁律——(1)输入电容紧贴IC (2)SW节点面积最小化 (3)FB走线远离SW和电感 (4)底部完整地平面。对比LDO的简单布局（几乎不用注意什么）vs Buck的严格布局，体会差异 |
| 周五 | DRC + 导出Gerber + 下单 |

**电路知识点**：

- LDO原理：智能可变电阻比喻 + 闭环反馈（测量→比较→调整）
- LDO效率公式：η = Vout/Vin，功耗 P = (Vin-Vout)×I（带数字计算示例）
- Buck原理：分阶段工作过程（Ton阶段电感储能/Toff阶段电感续流）+ 飞轮比喻
- Buck占空比：D = Vout/Vin（理论），实际由反馈环路动态调节
- 反馈回路的概念——LDO和Buck都是闭环系统（关联电路专题04闭环控制）
- 电感的基础知识：电流不能突变、磁场储能、"有惯性的水流"
- 开关电源PCB布局四铁律 vs LDO的"随便画"（对比理解）

### 第8周：等板期安排

| 天 | 内容 |
|----|------|
| 周一 | 焊接项目3的板子 + 测试灰度模块 |
| 周二 | 讲解电池供电系统：锂电池（3.7V/4.2V）的特性、保护板、充放电曲线、串联 vs 并联——计算2S（7.4V）/3S（11.1V）电池组的电压范围和容量 |
| 周三 | 小车供电架构实战：计算所有模块总功耗（MCU+舵机+传感器+蓝牙），验证为什么需要两级降压（Buck→LDO）而不是一级LDO到底 |
| 周四~五 | 用稳压电源+电子负载（或功率电阻）测试成品电源模块——负载从0→2A变化，记录负载调整率曲线；用示波器对比LDO输出和Buck输出的纹波差异

**收板后**：焊接电源模块，用万用表和示波器测输出电压和纹波。**注意安全：第一次上电串一个限流电阻或使用带限流的稳压电源。**

## 项目5 · 单片机最小系统板（第9-10周）

> **选它的理由**：巡线小车的大脑。最小系统板只包含MCU跑起来必要的电路，USB转串口下载、晶振、复位、电源——复杂度刚好够，又是巡线小车主板的前身。

### 第9周：原理图 + PCB

| 天 | 内容 |
|----|------|
| 周一 | MCU选型讨论：STM32F103C8T6（ARM Cortex-M3）vs STC8H8K64U（51内核）vs ATmega328P（Arduino生态）；综合考虑性价比和教程丰富度，**推荐STM32F103C8T6** |
| 周二 | 讲解最小系统组成：MCU + 晶振(8MHz)+负载电容 + 复位电路(上拉+按键+电容) + 电源去耦(每个VDD脚一个100nF) + BOOT选择 + SWD调试接口 + USB转串口(CH340N) |
| 周三 | 完成完整原理图：把所有GPIO引到排针（方便后续插传感器和驱动模块）、LED+按键作为板载调试外设 |
| 周四 | PCB布局：晶振紧靠MCU、去耦电容背面放置（最短回路）、USB走差分线(90Ω)、数字地区域划分 |
| 周五 | DRC + 下单 |

**电路知识点**：

- 单片机最小系统的五要素：电源、时钟、复位、程序下载、I/O引出
- 晶振的负载电容计算（CL = (C1·C2)/(C1+C2) + Cstray）
- 去耦电容为什么每个电源脚都要放（抑制芯片内部开关噪声）
- SWD vs JTAG调试接口
- BOOT0/BOOT1配置与启动模式

### 第10周：等板期安排

| 天 | 内容 |
|----|------|
| 周一 | 焊接项目4的板子 + 测试电源模块 |
| 周二 | 搭建STM32开发环境（Keil MDK或STM32CubeIDE或PlatformIO + VSCode） |
| 周三 | 第一个程序：GPIO点亮LED（HAL库 vs 寄存器操作）、GPIO输入读取按键 |
| 周四 | 定时器与PWM输出——用定时器产生PWM驱动项目2的电机驱动板 |
| 周五 | ADC读取——用ADC读取项目3的灰度传感器模拟输出（如果做了模拟输出版本） |

**收板后**：焊接最小系统板，烧录Blink程序，验证USB串口通信，验证SWD调试。

## 毕业项目 · 多传感器智能巡线小车（第11-16周）

> **目标**：整合前面所有模块，设计一块"巡线小车主板"，把MCU、电机驱动、传感器接口、电源管理集成到一块PCB上。不是简单堆叠——要重新设计，考虑系统级的问题。

| 天 | 内容 |
|----|------|
| 周一 | 需求定义：巡线（必须）+ 避障（加分）+ 测速（加分）+ 无线调试（加分）+ OLED显示（加分） |

## 附录B：每个项目的关键检查清单

**原理图阶段：**
- [ ] 所有元件都有正确的封装
- [ ] 电源网络没有短路（VCC和GND没有直接连接）
- [ ] IC的电源引脚都接了去耦电容
- [ ] 输入/输出接口有明确的丝印标注

**PCB阶段：**
- [ ] 运行DRC并通过（线宽、间距、过孔尺寸）
- [ ] 电源线和地线的宽度满足电流需求
- [ ] 晶振/差分信号等关键走线优先处理
- [ ] 板框尺寸和安装孔位置正确
- [ ] 丝印不重叠、方向一致、信息完整

**上电前：**
- [ ] 万用表通断档测试VCC与GND没有短路
- [ ] 目检焊接质量（虚焊、连锡）
- [ ] 电解电容和二极管极性正确
- [ ] IC方向正确

## 附录C：常见失败模式与对策

| 现象 | 可能原因 | 排查方法 |
|------|----------|----------|
| 上电后IC发烫 | 电源反接/短路/VCC-GND倒置 | 断电立刻测VCC-GND阻值 |
| LED不亮 | 极性反了/限流电阻太大/焊接虚焊 | 万用表二极管档逐级查 |
| 串口不识别 | CH340驱动未装/晶振不起振/焊接问题 | 检查晶振两端对地电压 |
| 电机不转 | 电源功率不足/TB6612损坏/STBY引脚未拉高 | 先查供电电压再查控制信号 |
| 传感器输出不变 | 电位器阈值不对/供电电压不对 | 用万用表测比较器输入端电压 |
| 电源模块输出不对 | 反馈电阻焊错/电感饱和 | 先测反馈引脚电压是否=Vref |
