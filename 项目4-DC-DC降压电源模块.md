# 项目4 · DC-DC降压电源模块

> **封装类型**：📐 全贴片（SMD）——0603阻容 + SOP-8芯片 + SMD电感  
> **难度**：★★★☆☆（难点不在焊，而在PCB布局——画不好可能完全不工作）  
> **画板时间**：1周（约5天，每天2-3小时）  
> **核心教学目标**：理解开关电源的工作原理，掌握开关电源PCB布局的铁律。这块模块将直接用在巡线小车上（电池→5V）。

---

## 1. 项目简介

前面三个项目我们都用5V电源——要么USB供电，要么稳压电源。但巡线小车用的是电池（锂电池7.4V或11.1V），电池电压会随着电量降低而下降，而且电压比5V高——我们需要一个**降压电路**把电池电压稳定地变成5V。

降压有两种方案：

| 方案 | 芯片举例 | 效率 | 发热 | 适用场景 |
|------|---------|:---:|:---:|---------|
| **线性稳压（LDO）** | LM7805、AMS1117 | 低（~50%） | 大 | 输入输出压差小、电流小（<500mA） |
| **开关稳压（DC-DC Buck）** | MP1584、LM2596 | 高（~85-95%） | 小 | 输入输出压差大、电流大、电池供电 |

如果电池是7.4V，输出5V/2A，用LDO的话：功耗=(7.4-5)×2=4.8W——得加很大的散热片。用Buck的话：效率90%，损耗≈0.5W——几乎不热。

所以我们选 **MP1584**——这是一颗经典的SOP-8封装Buck芯片，输入4.5~28V，输出可达3A，开关频率1.5MHz（频率高→外围电感/电容可以很小）。

### 1.1 你将学到什么

- 开关电源Buck拓扑的基本原理（开关→电感→电容→二极管循环）
- 开关电源PCB布局的四大铁律
- SMD功率电感、输入输出电容的选型
- 反馈电阻分压计算输出电压
- 用示波器看开关波形和输出纹波

## 2. 电路原理

### 2.1 Buck拓扑——用"水龙头"理解

想象你有一个高压水龙头（电池），想要稳定的低压水流（5V）：

- **不聪明的办法（LDO）**：只开一点点水龙头，多余的压力变成热量。水压越高压差越大，浪费越多。
- **聪明的办法（Buck）**：水龙头快速开-关-开-关（每秒150万次！），后面接一个水桶（电感+电容），水桶把脉冲水流"抚平"成平稳的低压水流。调整开和关的时间比例，就能控制出水压力。

### 2.2 Buck电路的四要素

```
Vin ──→ [开关管 SW] ──→ [电感 L] ──→ Vout
              │            │          │
              ├──→ 二极管D ←┤          │
              │   (续流)   │          │
             GND          GND    [输出电容 Cout]
                                     │
                                    GND
```

| 要素 | 功能 | MP1584中的对应 |
|------|------|---------------|
| **开关管** | 高频通断，把直流变成脉冲 | 集成在MP1584内部（NMOS） |
| **续流二极管** | 开关管断开时给电感提供续流路径 | 外部肖特基二极管 |
| **电感** | 储能+平滑电流，Buck的心脏 | 外部功率电感 |
| **输出电容** | 平滑输出电压，减小纹波 | 外部MLCC+电解 |

### 2.3 MP1584引脚说明

| 引脚 | 名称 | 功能 |
|------|------|------|
| 1 | PG | Power Good——输出电压正常时开漏输出低。本项目悬空 |
| 2 | EN | 使能脚——>1.5V启动，<1.2V关断。接Vin或通过分压电阻设定欠压保护 |
| 3 | COMP | 补偿脚——外接RC网络到GND（稳定反馈环路） |
| 4 | FB | 反馈脚——内部基准0.8V，通过外部分压电阻设定Vout |
| 5 | GND | 地（芯片底部还有散热焊盘——必须接地！） |
| 6 | SW | 开关节点——接电感和续流二极管 |
| 7 | IN | 电源输入——4.5~28V |
| 8 | SS | 软启动——外接电容到GND设定启动时间 |

### 2.4 外围元件选型

**输出电压设定（反馈电阻）**：

$$V_{OUT} = 0.8V \times \left(1 + \frac{R_1}{R_2}\right)$$

> R1是上分压电阻（FB到Vout），R2是下分压电阻（FB到GND）  
> 要得到5V输出：选R2=10kΩ → R1 = (5/0.8 - 1) × 10k = **52.5kΩ** → 实际选标称值 52.3kΩ（1%精度）或 51k+1.5k 串联

**电感选型**：

$$L = \frac{V_{OUT} \cdot (V_{IN} - V_{OUT})}{V_{IN} \cdot f_{SW} \cdot \Delta I_L}$$

不用手算——参考MP1584数据手册推荐值：输入7.4~12V、输出5V/2A → **10μH**。

电感额定电流要大于最大负载电流的1.3倍，所以2A输出至少选 **2.6A以上** 的电感。本项目选10μH/3A的SMD功率电感（如CD54或CD75型号，尺寸约5.8×5.2mm或7.3×7.3mm）。

**输入电容**：22μF MLCC（贴片陶瓷）×2，耐压25V以上  
**输出电容**：22μF MLCC ×2 + 100μF电解（可选，降低纹波）  
**续流二极管**：SS34（肖特基，3A/40V，SMA/SMB封装）  
**软启动电容**：0.1μF  
**补偿网络**：RC串联（约10kΩ + 3.3nF）——照数据手册参考电路取值

## 3. 开关电源PCB布局——四大铁律

> 这可能是整个教程中最重要的PCB布局章节。开关电源的PCB画不好，电路可能完全不工作、输出纹波巨大、甚至烧芯片。

### 铁律一：输入电容必须紧贴芯片

输入电容（C1/C2）的 **正极到MP1584的IN脚、负极到MP1584的GND脚**，距离越短越好——最好≤5mm。

**为什么？** 开关管以1.5MHz的频率在IN和SW之间快速通断，每次导通瞬间需要从输入电容抽取大电流尖峰。如果电容离得远，走线电感会在IN脚产生电压尖峰，可能击穿芯片。

```
       ┌──────────┐
       │ C1  C2   │ ← 输入电容
       └──────────┘
       ││      ││
       │└──────┘│
       ▼   GND  ▼
    ┌──────────────┐
    │ IN    GND    │  ← MP1584
    │   MP1584     │
    │ SW           │
    └──────────────┘
```

### 铁律二：SW节点面积最小化

SW引脚连接电感L1和续流二极管D1——这个节点的电压在0V和Vin之间以1.5MHz高速跳变，是板上最强的**EMI辐射源**。

SW节点的铜箔面积越小越好——把电感、二极管紧挨着SW引脚放，走线短且粗。

### 铁律三：反馈走线远离SW节点和电感

FB引脚是高阻抗输入，非常容易被SW节点耦合过来的噪声干扰——一旦FB脚上叠加了开关噪声，输出电压就会抖动。

**FB走线规则**：
- 从Vout引一条细线，经过R1顶端到FB引脚，路径远离电感和SW区域
- R2靠近FB引脚放置，FB到芯片的走线尽量短
- FB走线**不要在电感或SW走线的正下方或正上方**

### 铁律四：底部完整连续地平面

在MP1584下方保持完整连续的地平面（GND铺铜），芯片底部的散热焊盘必须焊接到地平面（散热+电气连接）。

续流二极管的阳极（接GND）和输入电容的GND、输出电容的GND，都应该通过大面积铺铜低阻抗连通。尽量不要用地走线串起来——这就是"星形接地"的反面教材。对于开关电源，**一整块连续的地铜皮才是最好的地**。

## 4. PCB布局步骤

### 4.1 元件摆放

```
  [输入端子] ── C1/C2 ── MP1584 ── L1 ── C3/C4 ── [输出端子]
                  ├── D1(SS34) ──┤
                  └── 大面积GND ──┘
                  
  FB分压电阻：靠近MP1584的FB脚
  补偿网络RC：靠近COMP脚
  软启动电容：靠近SS脚
```

### 4.2 走线宽度

| 走线 | 电流 | 线宽 |
|------|------|------|
| IN → 电感路径 | ~2A (峰值更高) | **1.5~2mm** |
| SW → 电感、SW → 二极管 | 同上 | **粗且短**，面积尽量小 |
| 电感 → Vout | ~2A | **1.5~2mm** |
| 反馈分压走线 | <1mA | 0.25mm（细线反而不容易耦合噪声） |
| GND | 回路电流 | **大面积铺铜** |

### 4.3 过孔

- 输入/输出大电容的GND焊盘各打2~3个过孔到底层GND铺铜（降低过孔电阻）
- MP1584底部散热焊盘打一个过孔矩阵（3×3或4×4的小过孔）到底层GND
- FB走线不打过孔（一条线走到底）

## 5. 原理图注意事项

- MP1584底部有一个**裸露焊盘（Exposed Pad）**——必须在原理图上连接GND，封装才会正确
- EN脚如果直接接Vin（>1.5V）就总是使能；如果要设欠压保护，加分压电阻
- COMP补偿网络的RC取值参考数据手册——不要随便改
- PG脚不接，悬空即可

---

## 6. 焊接注意

- MP1584的SOP-8 EP封装——除了8个引脚，底部还有散热焊盘。手工焊接时，**先在散热焊盘上预锡，然后用热风枪或从底面加热让锡熔化**
- 如果没有热风枪：可以在PCB的散热焊盘位置打一个大过孔，从PCB背面灌锡进去
- SS34二极管注意极性——**阴极（有横线标记的一端）接SW（或电感+输入端）**
- 功率电感不分极性，但要注意焊接牢固（电流大，虚焊会发热）

## 7. 测试（⚠ 安全第一）

### 7.1 上电前的检查

- 万用表通断档测输入端正负极之间不短路
- 测输出端正负极之间不短路
- 测MP1584的IN和GND之间不短路
- 目检所有焊点

### 7.2 首次上电

**强烈建议：第一次上电用稳压电源（带限流功能），限流设在100mA。或者在输入串一个10Ω/2W电阻。**

1. 输入端接7.4V（或12V）电源
2. 万用表电压档测输出——应该≈5V（±0.1V）
3. 用示波器看输出纹波（AC耦合，20MHz带宽限制）——应该<50mV
4. 如果输出不对，立刻断电查反馈电阻

### 7.3 带载测试

1. 接一个功率电阻做负载：R=V/I=5V/1A=5Ω（至少5W），或用电子负载
2. 输出电压应保持5V，纹波略有增加但不超100mV
3. 摸芯片和电感——温热正常，烫手异常（>60°C需检查）

## 8. 常见问题

| 现象 | 可能原因 |
|------|---------|
| 输出=0V | EN没拉高/输入电容短路/芯片损坏 |
| 输出=输入电压 | 开关管击穿（SW-IN短路）/FB开路导致芯片以最大占空比工作 |
| 输出纹波巨大 | 输出电容太小/电感饱和/布局太差（SW干扰FB） |
| 输出不稳定、跳动 | 补偿网络取值不对/FB走线被SW干扰/电感额定电流不够 |
| 芯片上电就烧 | 输入电容离太远导致IN脚电压尖峰/输入电压超过28V |
| 轻载正常、重载掉电压 | 电感饱和/续流二极管电流不够/输入功率不足 |

## 10. 拓展思考

### 10.1 Buck的连续/不连续模式

当负载电流较大时，电感电流始终>0——这叫**连续导通模式（CCM）**。当负载很轻时，电感电流会降到0然后开始新的周期——这叫**不连续导通模式（DCM）**。DCM下效率更高（开关损耗小），但输出纹波大一些。MP1584会自动切换。

### 10.2 同步Buck vs 异步Buck

MP1584是**异步Buck**——续流元件是外部肖特基二极管。更现代的方案是**同步Buck**——续流元件是一个MOSFET（集成在芯片内），导通电阻极低（几十mΩ），效率更高（因为MOSFET压降<<二极管压降）。比如SY8120（SOT-23-6封装，同步整流），但本项目用MP1584因为它外围元件多、学习价值更大。

### 10.3 输出电容的ESR与纹波

输出纹波近似等于电感纹波电流 × 电容ESR（等效串联电阻）。MLCC的ESR极低（几mΩ），所以用MLCC做输出滤波纹波很小。电解电容ESR较大（几十到几百mΩ），但容量大——通常MLCC+电解并联使用，MLCC滤高频、电解储能。
