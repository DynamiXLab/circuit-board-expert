# 电路原理图专题 — 稳压部分

> **学习目标**：掌握稳压电路的设计原理，能根据需求选择合适的稳压方案（LDO / DC-DC），并画出对应的原理图。

---

## 一、为什么需要稳压？

大多数电子电路需要**稳定的直流电压**才能正常工作。但电源来源往往不稳定：
- 电池：电压随电量下降而降低（4.2V → 3.0V）
- USB：标称 5V，但实际可能在 4.5V~5.5V 之间波动
- 变压器整流：纹波大，需要滤波和稳压

**稳压电路的作用**：无论输入电压如何变化，输出一个恒定的电压。

---

## 二、两种主流稳压方案

| 方案 | 特点 | 效率 | 噪声 | 适用场景 |
|------|------|------|------|---------|
| **LDO（低压差线性稳压）** | 降压、简单、外围元件少 | 低（η≈Vout/Vin） | 极低（μV级） | 模拟电路、音频、射频 |
| **DC-DC（开关稳压）** | 可升压/降压、效率高 | 高（85%~95%） | 较高（mV级纹波） | 数字电路、电池供电、大电流 |

### 2.1 LDO 线性稳压

**原理**：相当于一个自动调节的可变电阻，把多余的电压变成热量消耗掉。

```
  Vin ──┬── [调整管] ──┬── Vout
        │              │
        │         [反馈电阻]
        │              │
        [基准电压]─────┤
        │              │
        └── GND ──────┴── GND
```

**典型电路（AMS1117-3.3）：**

```
  Vin (5V) ──┬── 3 (Vin) ── 2 (Vout) ──┬── Vout (3.3V)
             │    AMS1117-3.3           │
             │    ┌──────────────┐      │
        C1 ──┤├── │ 1 (GND)      │      ├── C2 ──┤├── GND
        10μF   │  └──────────────┘       │   10μF
               │                         │
              GND                       GND
```

**选型关键参数：**

| 参数 | 含义 | 选型建议 |
|------|------|---------|
| **Vdropout（压差）** | 输入比输出至少高多少 | 普通 LDO 约 1V，低压差（如 XC6206）约 0.2V |
| **Iout（输出电流）** | 最大能提供多大电流 | 至少留 30% 余量 |
| **PSRR（电源抑制比）** | 对输入噪声的抑制能力 | 模拟电路选 PSRR > 60dB @1kHz |
| **静态电流 Iq** | LDO 自身消耗的电流 | 电池供电选 < 10μA |

**功率计算：**
```
P = (Vin - Vout) × Iout
例：5V→3.3V，带 500mA 负载
P = (5 - 3.3) × 0.5 = 0.85W → 需要散热焊盘或 SOT-223 封装
```

### 2.2 DC-DC 开关稳压

**原理**：通过高速开关（几十 kHz~几 MHz）配合电感和电容，高效转换电压。

**三种拓扑：**

| 拓扑 | 功能 | 典型芯片 |
|------|------|---------|
| **Buck（降压）** | Vin > Vout | MP1584、TPS5430、XL1509 |
| **Boost（升压）** | Vin < Vout | MT3608、XL6009 |
| **Buck-Boost** | 升降压均可 | TPS63020 |

**典型 Buck 电路（MP1584 5V→3.3V）：**

```
  Vin (5V~12V) ──┬── Vin              SW ──┬── L1(10μH) ──┬── Vout (3.3V)
                  │    MP1584              │               │
            Cin ──┤├── EN ── Vin（使能拉高） │               ├── Cout ──┤├── GND
            10μF   │    ┌──────────┐       │               │   22μF
                   │    │          │       │              GND
                   │    │  FB      │───────┴── R1(10k) ───┤
                   │    │          │                      │
                  GND   └──────────┘                 R2(3.24k) ── GND
                                                    Vout = 0.8V × (1 + R1/R2)
```

**DC-DC 关键设计要点：**

1. **电感选择**：通常 10~47μH，电流值 ≥ Iout × 1.2
2. **输入/输出电容**：陶瓷电容（X5R/X7R），低 ESR
3. **FB 反馈分压**：Vout = Vref × (1 + R1/R2)，查手册找 Vref
4. **布局**：SW 节点是主要噪声源，尽量缩小面积；FB 走线远离 SW
5. **续流二极管**：异步 Buck 需要肖特基二极管，如 SS34

| 芯片 | 类型 | Vin | Vout | Iout | 封装 | 用途 |
|------|------|-----|------|------|------|------|

| MT3608 | Boost | 2~24V | 最高 28V | 2A | SOT-23-6 | 锂电池升 5V |

## 四、稳压电路设计 checklist

- [ ] 输入电压范围是否覆盖了所有可能情况？
- [ ] 最大负载电流是否在芯片额定范围内（余量 30%）？
- [ ] LDO 的功耗是否在可接受的范围内？需要散热吗？
- [ ] 输入输出电容是否按数据手册推荐值选取？
- [ ] 电容耐压值是否足够（至少 1.5 倍额定电压）？
- [ ] 电容类型是否选对？MLCC 要注意 DC Bias 压降
- [ ] DC-DC 的电感饱和电流是否大于峰值电流？
- [ ] FB 分压电阻精度是否满足要求？（反馈用 1% 精度）
- [ ] 布局上，输入回路和输出回路面积是否最小化？

## ✏️ 实战作业

设计一个**5V→3.3V 的稳压电路**，要求：
1. 输出电流至少 500mA
2. 用 AMS1117-3.3（LDO 方案）画一张原理图
3. 输入输出各加 10μF 电解电容 + 100nF 陶瓷电容
4. 用 MP1584（DC-DC 方案）再画一张，对比两者优缺点
5. 在原理图上标注：电容耐压值、电阻精度要求
