## 一、什么是"芯片控制部分"？

任何智能电子产品都有一个"大脑"——**单片机（MCU, Microcontroller Unit）**。芯片控制部分就是围绕 MCU 搭建的所有外围电路，让它能正常工作并与外界交互。

一个典型的 MCU 控制系统框图：

```
┌─────────────────────────────────────────────────────────────┐
│                      MCU 最小系统                            │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────────┐   │
│  │ 电源电路 │  │ 晶振时钟 │  │ 复位电路 │  │ 下载调试接口 │   │
│  └─────────┘  └─────────┘  └─────────┘  └─────────────┘   │
│                            │                               │
│  ┌────────────────────────────────────────────────────┐    │
│  │                     MCU 芯片                        │    │
│  │         (STM32 / ESP32 / ATmega / CH32V)           │    │
│  │         GPIO │ UART │ I²C │ SPI │ ADC │ PWM        │    │
│  └────────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────────┘
```

## 二、MCU 最小系统

最小系统 = 让 MCU 能运行起来的最少外围电路。

### 2.1 电源电路

MCU 需要稳定的供电，常见电压等级：

| MCU 类型 | 典型电压 | 说明 |
|----------|---------|------|
| STM32F103 | 2.0~3.6V（通常 3.3V） | 内部有 LDO，电源引脚多 |
| ESP32 | 2.3~3.6V（通常 3.3V） | WiFi 工作时电流大（~300mA） |
| ATmega328P | 1.8~5.5V | 宽电压，Arduino 用 |
| CH32V003 | 2.4~5.5V | 国产低价 RISC-V |

**电源引脚必须加去耦电容：**

```
                     ┌──────────────┐
  VCC ───────────────┤ VDD          │
                     │              │
            C1 ──┤├──┤ 紧贴电源引脚  │  ← 100nF 陶瓷电容（高频去耦）
            100nF   │              │
                     │              │
            C2 ──┤├──┤ 可稍远       │  ← 10μF 电解或 MLCC（储能）
            10μF    └──────────────┘
                    GND
```

> ⚠️ **铁律**：每对 VDD/VSS 引脚旁边都要放一个 100nF 陶瓷电容，距离 < 5mm。这是 MCU 稳定工作的最低要求。

### 2.2 晶振时钟电路

MCU 需要时钟信号才能工作。时钟源有两种：

| 时钟源 | 精度 | 优点 | 缺点 |
|--------|------|------|------|
| **内部 RC 振荡器** | ±1%~±5% | 无需外围元件 | 精度低，温漂大 |
| **外部晶振** | ±20~±50ppm | 高精度 | 需要两个电容 |

**外部晶振电路（以 8MHz 为例）：**

```
                    ┌──────────────┐
        ┌──────────┤ OSC_IN       │
        │  C1      │              │
  X1 ──┴──┤├── 22pF               │
  8MHz    │        │              │
  ──┬──┤├── 22pF   │   MCU       │
        │  C2      │              │
        └──────────┤ OSC_OUT     │
                   └──────────────┘
                        GND
```

**晶振选型要点：**
- **负载电容**：CL = (C1 × C2) / (C1 + C2) + 寄生电容（约 3~5pF），结果要与晶振标称负载电容匹配
- **两个电容值相等**：通常 12pF~22pF
- **走线要短**：晶振到 MCU 引脚距离 < 10mm，两边等长
- **晶振下方不走其他信号线**：减少干扰

### 2.3 复位电路

MCU 上电时需要短暂保持复位状态，等电源稳定后再释放。

**RC 复位电路（低电平复位）：**

```
  VCC ──┬── R(10kΩ) ──┬── RST (MCU 复位引脚)
        │              │
        │         ┌────┤
        └─────────┤ C  │
                  │1μF│
                  └────┤
                       │
                      GND
```

**内部上拉 + 按键复位：**

```
  VCC ──┬── R(10kΩ) ──┬── RST (MCU 复位引脚)
        │              │
        │         ┌────┤
        └───┬─────┤ 按键 │
            │     └────┤
            │          │
           GND         GND
  （按键按下 → RST 接地 → 复位）
```

### 2.4 下载调试接口

MCU 没有程序就是一块"石头"。必须通过下载接口烧录程序。

| 接口类型 | 引脚数 | 适用 MCU |
|----------|--------|---------|
| **SWD** | 2 线（SWDIO + SWCLK） | STM32、CH32 系列 |
| **UART Bootloader** | 2 线（TX + RX） | ESP32、ATmega |
| **JTAG** | 4~5 线 | 高端 MCU/FPGA |

**SWD 接口电路：**

```
  ┌─── SWD 接口 ────┐
  │ 1: VCC          │
  │ 2: SWDIO ───────┤ MCU SWDIO
  │ 3: SWCLK ───────┤ MCU SWCLK
  │ 4: GND          │
  │ 5: NRST ────────┤ MCU NRST（可选）
  └─────────────────┘
  
  建议加 10kΩ 上拉至 VCC（SWDIO 和 SWCLK）
```

## 三、常见 MCU 引脚功能分类

| 引脚类型 | 功能 | 设计注意事项 |
|----------|------|-------------|
| **VDD/VSS（电源）** | 芯片供电 | 每对引脚都要加 100nF 去耦电容 |
| **NRST（复位）** | 复位芯片 | 不可悬空！10kΩ 上拉至 VCC + 可选按键 |
| **BOOT0/BOOT1** | 启动模式选择 | 一般通过 10kΩ 下拉到 GND（FLASH 启动） |
| **OSC_IN/OSC_OUT** | 外部晶振 | 走线短，负载电容匹配 |
| **GPIO** | 通用输入输出 | 默认状态：输入浮空（STM32），注意电平 |
| **UART_TX/RX** | 串口通信 | TX 接对方的 RX，RX 接对方的 TX（交叉） |
| **I²C_SCL/SDA** | I²C 总线 | 需要 4.7kΩ 上拉电阻至 VCC |
| **ADC_IN** | 模拟输入 | 走线远离数字信号，加 RC 滤波 |
| **SWDIO/SWCLK** | 调试接口 | 10kΩ 上拉，引出到排针 |

## 四、MCU 系统设计 checklist

- [ ] 每对 VDD/VSS 引脚都有 100nF 去耦电容吗？（距离 < 5mm）
- [ ] 整个系统至少有一个 10μF 储能电容？
- [ ] 复位引脚有上拉电阻吗？（不是悬空状态）
- [ ] 晶振负载电容值匹配吗？走线够短吗？
- [ ] BOOT 引脚设置了正确的启动模式？
- [ ] 下载调试接口（SWD/UART）引出来了吗？
- [ ] 所有未使用的 GPIO 怎么处理的？（悬空/接地/上拉？）
- [ ] 不同电压域之间有电平转换吗？（3.3V→5V？）
- [ ] I²C 总线上有上拉电阻吗？
