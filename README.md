# 电子电路基础教程

## 项目简介

本项目是**机动不已实验室**推出的电子电路基础教程，面向零基础学习者，通过一个个完整项目带领学员从入门到独立设计电路板。课程内容涵盖基础元件认知、电路原理分析、立创EDA设计、PCB打样下单以及焊接调试全流程，最终以多传感器巡线小车作为毕业项目，串联所有模块知识。

课程采用"项目驱动"的教学模式，每个项目围绕一个具体电路模块展开，学员在动手实践中逐步掌握电子电路的核心技能。

## 写在前面

其实实验室早在几年前就有了编写一套完善入门教程的想法，但碍于实验室常年比赛多、项目多，人手一直非常稀缺，这个计划只能一拖再拖。如今 AI 盛行，各类 AI 工具不断推出，加上本人也已大四，终于有了一小点空闲时间，于是趁此机会重启了教材计划。

本项目由**机动不已实验室**成员利用空闲时间、借助 AI 之力编写完成，难免有不完善之处，还请大家见谅。我自己也清楚教程中还有很多没有处理好的地方，欢迎大家提出批评，有空的时候学长会努力一点一点优化的。学习过程中如有不懂的地方，请及时向学长求助。

教程由学长列出整体大纲后，分人分章节协作完成，部分章节之间可能存在内容不够连贯的情况，欢迎大家提出建议。如有好的改进想法，可以通过 **Git Fork** 修改后 **Push** 并提交 **Pull Request** 请求合并，实验室学长看到后会第一时间审核处理——当然，如果学长太忙，也可以直接滴滴学长催更 😄。

此外，教程中部分图片可能存在缺失，大家在练习的过程中如果顺手帮我们截图补充完整，我们会在致谢中郑重加上你的名字！

我们非常欢迎大家参与教程的优化过程——你优化一点，我优化一点，明天就更美好了 🌟。
