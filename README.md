# 电子电路基础教程

## 项目简介

本项目是**机动不已实验室**推出的电子电路基础教程，面向零基础学习者，通过一个个完整项目带领学员从入门到独立设计电路板。课程内容涵盖基础元件认知、电路原理分析、立创EDA设计、PCB打样下单以及焊接调试全流程，最终以多传感器巡线小车作为毕业项目，串联所有模块知识。

课程采用"项目驱动"的教学模式，每个项目围绕一个具体电路模块展开，学员在动手实践中逐步掌握电子电路的核心技能。
