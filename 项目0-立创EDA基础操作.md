## 1. 什么是立创EDA？

立创EDA是立创商城（LCSC）推出的一款**国产免费PCB设计工具**。它有以下几个特点：

| 特点 | 说明 |
|------|------|
| **免费** | 个人使用完全免费，没有功能阉割 |
| **在线+离线** | 浏览器打开就能用，也提供桌面客户端 |
| **元件库丰富** | 直接对接立创商城——画原理图时就能看到元件的价格和库存 |
| **一键下单** | 画完PCB直接导出到嘉立创下单打样，不用手动填参数 |
| **中文界面** | 对英语不好的同学非常友好 |

对比业界其他工具：Altium Designer 功能最强但收费昂贵，KiCad 免费开源但上手曲线陡峭，EAGLE 被 Autodesk 收购后趋于边缘化。**立创EDA是国内大学生入门PCB设计的最佳选择。**

## 2. 环境安装

### 2.1 注册账号

1. 打开 [oshwhub.com](https://oshwhub.com)（立创开源硬件平台）
2. 点击右上角"注册"——用手机号或微信扫码
3. 注册完成后登录

> **一个账号就够了**——立创EDA、立创商城、嘉立创打样共享同一套登录体系。

### 2.2 桌面客户端 vs 网页版

| | 桌面客户端 | 网页版 |
|------|:--:|:--:|
| 需要安装 | ✅ 下载安装 | ❌ 浏览器打开即用 |
| 离线使用 | ✅ 可以 | ❌ 需要网络 |
| 性能 | 略好（大工程时更流畅） | 够用（我们的项目都很小） |
| 自动更新 | 手动 | 自动 |

**建议**：先用网页版快速上手（打开 https://lceda.cn/editor 就能画），如果觉得卡再装客户端。

### 2.3 安装桌面客户端（可选）

1. 打开 [lceda.cn](https://lceda.cn) → 点击"下载"
2. 选择对应系统版本（Windows/Mac/Linux）
3. 安装完成后用注册的账号登录

### 2.4 初始设置

打开立创EDA后，先做几项设置（只需做一次）：

| 设置项 | 路径 | 建议值 |
|--------|------|--------|
| 画布单位 | 设置 → 画布 → 单位 | **mm**（毫米） |
| 网格大小 | 设置 → 画布 → 网格 | **0.635mm**（25mil，DIP引脚间距的一半） |
| 光标样式 | 设置 → 画布 → 光标 | 大十字（方便对齐） |
| 自动保存 | 设置 → 通用 → 自动保存 | 开启，间隔5分钟 |

> **为什么网格设为0.635mm？** 插件元件的引脚间距通常是2.54mm（100mil），0.635mm是它的1/4——画原理图时连线刚好对齐网格。

## 4. 第一个工程：LED+电阻+电源

让我们从零开始画一个最简单的电路——一个LED串联一个限流电阻，两端接5V和GND。

### 4.1 新建工程

1. 菜单 → 文件 → 新建 → 工程
2. 工程名输入 `练习_LED电路`
3. 保存到默认位置
4. 工程树会出现一个空白原理图页 `Schematic_1`

### 4.2 放置第一个元件：电阻

1. 在左侧元件库的搜索框输入 `电阻`
2. 在结果列表中找到 "电阻 R" 或 "RES"
3. **双击**搜索结果中的元件——它会粘在光标上
4. 移动光标到画布中间，**单击**放置
5. 按 `Esc` 或右键退出放置模式

> 💡 如果放置了不想放的东西，按 `Esc` 取消。

### 4.3 修改电阻参数

1. **单击选中**画布上的电阻（它会高亮）
2. 右侧属性面板会显示它的属性
3. 找到 "阻值" 字段，改成 `1kΩ`（或 `1k`）
4. 找到 "名称" 字段，默认是 `R?`，不用改（自动编号）
5. 找到 "封装" 字段——现在是空的或默认值。点击旁边的按钮，在弹出的封装选择器中搜索 `AXIAL-0.3`（通用的插件电阻封装），选中它。

### 4.4 放置LED

1. 搜索 `LED`
2. 找到 "LED" 或 "发光二极管"
3. 双击放置到电阻右边
4. 修改LED的颜色为"红色"（属性面板中如果有的话），封装选 `LED-TH-5mm`（5mm插件LED）

### 4.5 放置电源和地

1. 在元件库搜索 `VCC` 或者在工具栏找到电源符号图标
2. 放置一个 **VCC（5V）** 符号在电阻左边
3. 放置一个 **GND** 符号在LED右边

> **VCC和GND不是真实的元件**——它们是"电源端口"符号，代表电路中所有的VCC是连在一起的、所有的GND是连在一起的。省去了拉很长的电源线。

### 4.6 连接导线

**方法一：用导线工具**
1. 按 **W** 键进入连线模式（光标变成十字）
2. 点击 VCC 的引脚端点，拖动到电阻的左引脚，点击完成
3. 依次连接：电阻右端→LED左端，LED右端→GND
4. 按 `Esc` 退出连线模式

**方法二：用网络标签（当前电路不需要，但要知道）**
- 按 **N** 键放置网络标签
- 给导线的端点命名——同名的网络标签在电路中是连通的
- 适合长距离连接或跨页连接

### 4.7 完成图

最终的原理图应该长这样：

```
VCC(5V)
  │
 1kΩ
  │
 LED(红色)
  │
 GND
```

### 4.8 保存

按 `Ctrl+S` 保存原理图。

## 6. 核心概念速通

### 6.1 原理图（Schematic）和PCB的区别

| | 原理图 | PCB |
|------|--------|-----|
| **你在画什么** | 电路的逻辑关系——"谁和谁连在一起" | 电路的物理实现——"元件摆在哪儿、线怎么走" |
| **元件长什么样** | 抽象符号（电阻=锯齿线） | 实际封装（电阻=两个孔+轮廓） |
| **连线是什么** | 逻辑网络（一条线=一个电气连接） | 实际铜走线（有宽度、有层、有间距） |
| **能生产吗** | ❌ 不能——只是图纸 | ✅ 导出Gerber后就能发厂生产 |

### 6.2 封装（Footprint）

封装是元件在PCB上的"脚印"——它定义了：
- 焊盘的位置、形状、大小
- 元件的外形轮廓（丝印框）
- 引脚编号和原理图符号的对应关系

**每个元件必须绑定封装才能转到PCB。** 在原理图中选中元件，右侧属性面板的"封装"栏就是做这个的。

常见的封装名：

| 封装名 | 元件类型 |
|--------|---------|
| AXIAL-0.3 | 插件电阻（1/4W） |
| CAP-DIP-5.08 | 插件电解电容（脚距5.08mm） |
| LED-TH-5mm | 5mm插件LED |
| DIP-8 | 8脚插件IC |
| SOP-8 | 8脚贴片IC（1.27mm间距） |
| 0603 | 贴片电阻/电容（1.6×0.8mm，最主流手工焊接尺寸） |
| 0805 | 贴片电阻/电容（2.0×1.25mm，比0603大一号） |
| TO-220 | 功率三极管/稳压器（带散热片） |

### 6.3 网络（Net）

网络就是原理图中电气连通的节点集合。VCC、GND、每根导线——都是网络。两个同名网络标签自动连通（即使没有导线连接）。

> 🧠 **关键理解**：原理图中的导线不代表实际走线路径——它只表示"这两点电气上连通"。到了PCB阶段你才决定走线具体走什么路径。

### 6.4 从原理图到PCB的流程

```
原理图完成 → 检查封装都绑定了 → 转为PCB
                                    ↓
                            所有元件出现在PCB画布上
                            元件之间有"飞线"（鼠线）
                            显示谁和谁需要连通
                                    ↓
                            手动布局（摆放元件位置）
                                    ↓
                            手动布线（画铜走线替代飞线）
                                    ↓
                            铺铜（大面积GND铜皮）
                                    ↓
                            DRC检查（设计规则检查）
                                    ↓
                            导出Gerber → 下单打样
```

### 6.5 PCB 的层叠结构——"三明治"里有什么

一块双层PCB不是简单的一块板子——它是由**多个功能层叠压**而成的"三明治"。理解每一层干什么用，是画好PCB的基础。

```
┌─────────────────────────┐
│   顶层丝印（白色字符）    │  ← 印位号(R1/C1)、Logo、版本号（无电气功能）
├─────────────────────────┤
│   顶层阻焊（绿色油墨）    │  ← 保护铜箔，焊盘处"开窗"露出铜
├─────────────────────────┤
│   顶层铜箔（Top Layer）   │  ← 放元器件 + 走线（1oz ≈ 35μm 厚）
│        ═══════            │
├─────────────────────────┤
│    FR-4 玻璃纤维基板      │  ← 绝缘层，1.6mm 厚（最常用）
│                          │
├─────────────────────────┤
│   底层铜箔（Bottom Layer）│  ← 走线（也可以放元器件）
│        ═══════            │
├─────────────────────────┤
│   底层阻焊（绿色油墨）    │  ← 保护底层铜箔
├─────────────────────────┤
│   底层丝印（白色字符）    │  ← 底部标识（一般不用）
└─────────────────────────┘
```

**各层在立创EDA中的颜色（记住这些，看PCB时就不会眼花）：**

| 层名 | EDA中颜色 | 作用 | 有铜吗 |
|------|:--------:|------|:------:|
| **Top Layer（顶层）** | 红色 | 放置元器件、走线 | ✅ 铜 |
| **Bottom Layer（底层）** | 蓝色 | 走线、铺GND、也可放元件 | ✅ 铜 |
| **Top Overlay（顶层丝印）** | 黄色 | 印位号、极性标记、版本号 | ❌ 油墨 |
| **Top Solder（顶层阻焊）** | 紫色 | 保护铜箔，焊盘处开窗 | ❌ 油墨 |
| **Board Outline（板框）** | 灰色 | 定义PCB外形，工厂据此切割 | ❌ |
| **Multi-Layer（多层）** | 白色 | 焊盘和过孔（贯穿全板） | ✅ |

**几个容易混淆的概念：**

| 概念 | 一句话解释 |
|------|-----------|
| **焊盘（Pad）** | PCB上用来焊接元件的金属接触点。贴片焊盘是表面的铜块，插件焊盘是孔+环形铜箔 |
| **过孔（Via）** | 从顶层贯穿到底层的导电孔，孔壁镀铜，用于连接不同层的走线 |
| **焊环（Annular Ring）** | 过孔或插件焊盘的铜环部分。焊环宽度 = (外径 - 内径) / 2，嘉立创要求 ≥ 0.125mm |
| **阻焊（Solder Mask）** | 绿色油墨层，覆盖不需要焊接的铜箔。三大作用：①防焊锡桥接短路 ②防铜氧化 ③绝缘 |
| **丝印（Overlay）** | PCB表面的白色字符，标注元件位号和极性——焊接时的"导航地图" |
| **FR-4** | 最常见的PCB基板材料——玻璃纤维布浸环氧树脂，绝缘、阻燃、强度高 |

> 💡 **关于铜厚**：1oz = 1平方英尺面积上铺1盎司铜 ≈ 35μm厚。1oz铜厚下，1mm宽走线约可通过1A电流（温升10°C）。大电流走线要么加宽、要么用2oz铜厚。

## 7. 练习任务

完成以下操作来巩固今天学的内容：

### 练习1：修改电路

在上面画的LED电路中：
- 把电阻从1kΩ改成 **470Ω**
- 把LED颜色从红色改成 **绿色**
- LED的限流电阻需要重新计算吗？→ 绿光Vf≈2.2V，R=(5-2.2)/0.01=**280Ω** → 选270Ω或330Ω

### 练习2：双LED交替闪烁电路（原理图预习）

画出两个LED并联、各串一个电阻、共用一个VCC和GND的原理图。这是项目1流水灯的简化版——熟悉了原理图操作，项目1画10个LED就不会手忙脚乱。

### 练习3：封装搜索练习

在元件库中分别搜索以下封装，看看它们长什么样：
- `DIP-8`（NE555的封装）
- `DIP-16`（CD4017的封装）
- `LED-TH-5mm`
- `AXIAL-0.3`
- `CAP-DIP-5.08`

> **方法**：在立创EDA中，元件库搜索框旁边有个"封装"标签——切换到封装搜索模式，输入封装名即可预览封装图形。

## 8. 常见问题

| 问题 | 解决 |
|------|------|
| 元件放歪了/没对齐网格 | 选中元件 → 属性面板 → 坐标改为整数（如X=50, Y=50） |
| 导线从元件引脚上脱开了 | 删掉重连——确保连线时光标出现一个**小圆圈**再点下去，小圆圈表示吸附到了引脚 |
| 搜索不到想要的元件 | 换关键词——试试英文型号、简写、封装名 |
| 不小心删了一个元件 | `Ctrl+Z` 撤销 |
| 画布太小看不清 | 鼠标滚轮缩放，或按 `Ctrl+0` 适合窗口 |
| 元件编号自动变成了R1/R2/LED1 | 这是自动编号，正常的——你也可以手动改成自己想要的名字 |
| 封装怎么选？ | 看实际买的元件是什么封装——直插电阻=AXIAL-0.3，直插LED=LED-TH-5mm，拿不准就问学长或看元件数据手册 |

## 9. 立创EDA和嘉立创下单的关系

画完PCB后最终要下单打样。立创EDA和嘉立创是兄弟公司，一键打通：

```
立创EDA → 导出Gerber → 嘉立创下单助手（自动识别参数）→ 付款 → 等一周收板
```

不需要手动填板厚、铜厚、颜色这些参数——立创EDA导出的Gerber里已经包含了。

> 💰 **学生福利**：嘉立创每个月给新注册用户送打样优惠券，双层板10×10cm以内5片经常只要5元还包邮。第一块板几乎等于白送，赶紧薅！
