# 项目3 · 灰度传感器模块

> **封装类型**：📐 全贴片（SMD）——0603阻容 + SOP-8芯片  
> **难度**：★★★☆☆（比项目2多一颗星，因为第一次焊贴片）  
> **画板时间**：1周（约5天，每天2-3小时）  
> **核心教学目标**：从插件过渡到贴片，理解模拟信号调理链，为巡线小车准备核心传感器。

---

## 1. 项目简介

前两个项目我们用插件元件实现了流水灯（纯数字信号）和电机驱动（功率电路）。这次我们进入一个新的领域——**模拟信号处理**。

灰度传感器（也叫巡线传感器）是巡线小车最重要的感知器官。它的原理很简单：

```
IR LED发送红外光 → 照到地面 → 反射回来 → 光电三极管接收
                                           ↓
                              黑色地面反射弱（输出电压低）
                              白色地面反射强（输出电压高）
                                           ↓
                              比较器把模拟电压变成"0/1"数字信号
```

我们要做的是一个**5路灰度传感器模块**——5个独立的传感器排成一排，可以一次检测5个位置的"黑还是白"，这就是巡线小车判断"我偏左了还是偏右了"的依据。

这次所有元件都用**贴片封装**——0603的电阻电容、SOP-8的比较器芯片。焊接难度比插件高，但学会之后你就是PCB世界的"成年人"了。

### 1.1 你将学到什么

- 贴片封装体系：0805/1206/0603的含义、SOP/SOIC的区别
- 贴片手工焊接技术（拖焊、点焊）
- 红外发射管和光电三极管的工作原理
- 比较器（LM393）的使用和滞回设计
- 模拟电路PCB布局——模拟地和数字地的处理

## 2. 贴片元件入门

### 2.1 贴片阻容的尺寸代码

贴片电阻和电容的尺寸用**英制代码**表示，比如 **0805** 意思是 0.08英寸 × 0.05英寸（约 2.0mm × 1.25mm）。

| 代码 | 尺寸（mm） | 手工焊接难度 | 适用场景 |
|------|-----------|:----------:|---------|
| 1206 | 3.2 × 1.6 | ★☆☆ 很容易 | 初学贴片，空间不敏感 |
| 0805 | 2.0 × 1.25 | ★★☆ 适中 | 项目3之前用这个，现在统一切0603 |
| **0603** | **1.6 × 0.8** | **★★★ 需要练习** | **本项目及之后全用这个——标准贴片入门尺寸** |
| 0402 | 1.0 × 0.5 | ★★★★★ 几乎看不见 | 手机等超密设计 |

> **为什么项目3开始统一用0603？** 0603比0805小一圈但仍可手工焊接（镊子+细尖烙铁头），是工业界和爱好者社区最主流的贴片阻容尺寸。0603以下（0402）确实需要热风枪或回流焊，但0603手工完全能驾驭——前面项目1-2已经把焊接基本功练好了。

### 2.2 贴片电阻的标识

贴片电阻不是色环，而是在顶上印数字或字母：

- **三位数标识**（±5%精度）：前两位是有效数字，第三位是乘数（加几个零）
  - "103" = 10 × 10³ = **10kΩ**
  - "221" = 22 × 10¹ = **220Ω**
  - "0" 或 "000" = **0Ω**（跳线/短接）

- **四位数标识**（±1%精度）：前三位有效数字，第四位乘数
  - "1002" = 100 × 10² = **10kΩ**
  - "4991" = 499 × 10¹ = **4.99kΩ**

> 贴片电阻的功率通常很小——0603的是 **1/10W（0.1W）**——但对本项目足够（电路电流都<20mA）。

### 2.3 贴片电容的标识

贴片陶瓷电容（MLCC，多层陶瓷电容）**通常不印字**——这是个大坑！拿到手后靠：
- 包装袋/料盘的标签
- 或者用电容表/万用表电容档测量

所以养成好习惯：**贴片电容从料带取下来后立刻用，用完封回去，不要混在一起。**

贴片陶瓷电容有极性吗？**没有**。两端一样，不用分正负。

但如果是贴片钽电容（黄色/黑色长方体，一端有横线标记），**有极性**——横线端是正极。

### 2.4 SOP封装

SOP（Small Outline Package，小外形封装）是DIP的贴片版本——两边各一排引脚，引脚向外弯（鸥翼脚）。

| 封装名 | 引脚间距 | 本体宽度 | 焊接方式 |
|--------|---------|---------|---------|
| SOP-8 | 1.27mm | ~4mm | 拖焊或逐脚点焊 |
| SOP-14 | 1.27mm | ~4mm | 同上 |
| SOP-16 | 1.27mm | ~4mm | 同上 |
| TSSOP-8 | 0.65mm | ~3mm | 需要更细的烙铁头 |

本项目LM393用 **SOP-8**，引脚间距1.27mm，手工焊接最友好的贴片IC。

**SOP引脚编号规则**：和DIP一样——正面朝上，缺口/圆点标记在左，左下角是1脚，逆时针数。

### 2.5 贴片LED

贴片LED一般是长方形或小方块，常见0603或0805封装。底部有一个**绿色小三角或缺口标记阴极（负极）**，或者看底部焊盘——大的那片通常是负极。

### 2.6 贴片电位器（可调电阻）

本项目需要一个可调电阻来设定灰度阈值。贴片的选择：
- **3296W 型**：小型立式可调电阻（插件式但很小），旋钮在侧面，方便调节
- **贴片微调电位器**：如 3×3mm 规格，需要在PCB上留出螺丝刀调节空间

建议用3296W——虽然是插件，但体积小、调节手感好、便宜。

## 3. 电路原理

### 3.1 单路传感器电路

```
VCC(5V)
  │
  ├──→ R1(220Ω) ──→ IR LED ──→ GND     ← 红外发射（恒亮）
  │
  ├──→ 光电三极管(集电极) ──→ R2(10kΩ) ──→ GND   ← 红外接收
  │         │
  │         └──→ 输出电压 V_sensor（接比较器输入）
  │
  └──→ LM393比较器
           ├─ 同相端(+) ← V_sensor
           ├─ 反相端(-) ← 电位器分压 V_ref（可调阈值）
           └─ 输出 → 指示灯LED + 排针引出
```

### 3.2 分步讲解

**发射部分**：IR LED + 220Ω限流电阻，一直发光。电流约 (5V-1.2V)/220Ω ≈ 17mA——红外LED压降较低，约1.2V。

**接收部分**：光电三极管（Phototransistor）——红外光越强，流过它的电流越大，发射极电阻R2上的电压越高。遇到白纸（反射强） → V_sensor高（接近VCC）；遇到黑线（反射弱） → V_sensor低（接近GND）。

**比较部分**（LM393 = 双比较器，一个芯片含两路）：

- 当 V_sensor > V_ref → 输出为高（≈VCC）→ 认为是"白" → 指示灯灭
- 当 V_sensor < V_ref → 输出为低（≈GND）→ 认为是"黑" → 指示灯亮

调节电位器改变V_ref，就能改变"黑白界线"的判定阈值。

> ⚠️ LM393的输出是**开漏（Open-Drain）**的——输出脚内部是一个接GND的开关，拉低时导通，不拉低时是**高阻态**（不是高电平）。所以输出脚必须接一个**上拉电阻（10kΩ）到VCC**，否则输出永远是浮空状态！

### 3.3 滞回（迟滞）——关键设计

没有滞回的比较器，当V_sensor在V_ref附近波动时（实际中一定会），输出会疯狂跳变（振荡）。

**加滞回**：在输出和同相输入端之间加一个正反馈电阻 Rh（例如100kΩ~1MΩ）。

```
LM393输出经过Rh回到同相端(+)
→ 当输出变高时，把同相端电压"托"高一点
→ 当输出变低时，把同相端电压"拉"低一点
→ 形成两个不同的阈值：V_th+（从低到高的跳变点）和 V_th-（从高到低的跳变点）
→ 两者之间的差值就是滞回宽度
```

滞回宽度 = (VCC × R2) / Rh（大约），Rh越小滞回越宽。

本项目滞回设为约0.1~0.2V就行——肉眼看不见输出抖动即可。

### 3.4 5路阵列化

把单路复制5份，排成一排：

```
[传感器1] [传感器2] [传感器3] [传感器4] [传感器5]
   ↓          ↓          ↓          ↓          ↓
  OUT1       OUT2       OUT3       OUT4       OUT5
```

5路输出通过排针引出，后续接MCU的5个GPIO。

每路一个独立电位器调节阈值（共5个电位器），因为不同位置距离地面高度可能略有不同。

### 3.5 BOM清单

| 序号 | 元件 | 参数/型号 | 封装 | 数量 |
|------|------|----------|------|------|
| 1 | U1~U3 | LM393 双比较器 | SOP-8 | 3（两个比较器/片，5路需要3片） |

## 4. 原理图绘制

### 4.1 放置元件

在立创EDA中搜索：
- `LM393` → 选 SOP-8 封装
- `红外发射管` → 选 3mm 插件 LED（注意不是普通LED，找IR/红外波长的）
- `光电三极管` → 选 PT333 或类似型号
- `3296W` → 可调电阻
- `排针 1×5` → 输出接口

### 4.2 画单路再阵列化

先画好一路（一个发射管+一个光电管+一个比较器通道+一个电位器），然后**复制粘贴4次**，改网络标签即可。

立创EDA支持多通道设计（Repeated Blocks），但对新手而言复制粘贴更直观。

### 4.3 绘制注意事项

- 别忘了每个LM393输出脚的上拉电阻（10kΩ到VCC）
- 别忘了滞回电阻（470kΩ，从输出到同相端+）
- 每个LM393的VCC脚（8脚）和GND脚（4脚）各接0.1μF去耦电容
- LM393不用的那个比较器：输入端分别接VCC和GND，输出悬空——防止振荡

## 5. PCB布局布线

### 5.1 结构设计

传感器要伸出板边，因为需要贴近地面。两种方案：

**方案A（推荐）：一体板**
- 板子主体是电路区，前面伸出5个"手指"，每个手指尖放一对IR LED+光电管
- 手指之间挖槽（机械层画槽孔），减少相邻传感器之间的光干扰

**方案B（模块化）**：5个独立小板，通过排线连到主板。好处是可以独立调节位置。

本项目用**方案A**，因为做一块板简单。

### 5.2 布局要点

```
┌─────────────────────────────────────────┐
│   传感器1   传感器2   传感器3   传感器4   传感器5  │← 伸出的手指
│   [IR][PT] [IR][PT] [IR][PT] [IR][PT] [IR][PT]  │
│                                                │
│   ┌── LM393 ──┐  ┌── LM393 ──┐  ┌── LM393 ──┐  │
│   │  比较器1+2 │  │  比较器3+4 │  │  比较器5   │  │
│   └───────────┘  └───────────┘  └───────────┘  │
│                                                │
│   VR1 VR2 VR3 VR4 VR5    [输出排针]  [电源排针]  │
└─────────────────────────────────────────┘
```

- 5个传感器均匀排布，间距12~15mm（标准巡线赛道黑线宽度15~20mm）
- 电位器一字排开在板子下方，旋钮朝上，方便调节
- 输出排针和电源排针在板子后侧
- 每个LM393的去耦电容就近放置

### 5.3 走线注意事项

- 信号线：0.3mm即可
- 模拟信号（传感器→比较器输入）尽量短，远离数字输出线
- 红外发射管的电流回路（约17mA）没必要用粗线——0.3mm足够
- 铺铜：整个底层铺GND铜，顶层也铺GND铜——最大程度减少干扰

### 5.4 传感器手指设计

每个传感器手指：
- 长度约15~20mm（伸出主体）
- 宽度约10mm
- IR LED和光电管之间间距约3~4mm（稍微倾斜，让光路呈V字形反射）
- 手指之间开槽（宽2~3mm）减少光串扰

## 6. 贴片焊接指南

### 6.1 工具要求

和插件焊接相比，额外需要：
- **尖头镊子**（弯头的更好）：夹取和定位贴片元件
- **细尖烙铁头**（如T12-K刀头或T12-BC2马蹄头）：方便拖焊
- **助焊剂**：贴片焊接的灵魂——涂一点在焊盘上，焊接成功率翻倍
- **吸锡带**：处理连锡

### 6.2 0603阻容焊接

**方法：先固定一端，再焊另一端。**

1. 在一个焊盘上熔一点锡（不要太多）
2. 镊子夹住元件放在焊盘上，烙铁熔化刚才那点锡 → 元件一端固定住
3. 焊另一个焊盘（加锡、烙铁停留1~2秒）
4. 回头补焊第一个焊盘（如果焊锡太少）

两个焊点都要饱满光滑，呈微凸的半月形——有放大镜或手机微距镜头可以检查。

### 6.3 SOP-8芯片焊接（拖焊法）

1. 先对准——芯片引脚和焊盘精确对齐（用镊子微调）
2. 焊对角两个引脚固定芯片
3. 在一边4个引脚上涂助焊剂
4. 烙铁头带一点锡，从引脚根部向外拖——**动作流畅，不要停顿**
5. 如果两个引脚之间连锡了，加助焊剂再拖一次，或者用吸锡带吸掉
6. 换个方向焊另一边

### 6.4 常见贴片焊接问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 元件立碑（一端翘起） | 两端受热不均/焊盘一边大一边小 | 两端同时加热，或先焊小焊盘一端 |
| 连锡（桥接） | 锡太多/引脚太密 | 加助焊剂+拖焊+吸锡带 |
| 虚焊（冷焊） | 烙铁温度不够/停留时间太短 | 温度设330°C，每个焊点保持2秒 |
| 元件移位 | 焊锡熔化后手抖/镊子松太早 | 焊完后等焊锡凝固（变暗）再松镊子 |

## 7. 测试

### 7.1 静态检查

上电前：
- 万用表通断档测VCC-GND不短路
- 目检每个SOP-8有没有引脚连锡（特别是相邻的4-5脚之间）
- 所有0603元件有没有立碑

### 7.2 上电测试

1. 接5V电源
2. 用手机摄像头看IR LED——手机能拍到红外光（紫色或白色光点），肉眼看不到
3. 万用表电压档测光电管的V_sensor——对着白纸和对着黑纸，电压应该明显不同
4. 调节电位器，看指示灯什么时候亮/灭
5. 在白纸上画一条黑线（约15mm宽），手持传感器扫描过去，看指示灯能否正常跳变

### 7.3 5路一致性校准

把所有5路放在同一白色表面上方同一高度，调节电位器使指示灯刚好灭（或刚好亮）——这样所有传感器阈值一致。

## 8. 常见问题

| 现象 | 可能原因 |
|------|---------|
| 对着白纸黑纸输出都一样 | IR LED没亮（坏/极性反/限流电阻焊错） |
| 输出一直跳变（闪烁） | 没加滞回电阻/滞回电阻太大/供电纹波大 |
| 输出一直高/一直低不变化 | 上拉电阻忘焊/电位器没调节/V_ref设置太极端 |
| 灵敏度很低 | 光电管和IR LED距离不对/角度不对/高度太远 |
| 相邻传感器之间有干扰 | 手指之间没有挖槽隔离/IR LED光线串过去了 |

## 10. 拓展思考

### 10.1 为什么不直接用ADC读模拟值？

用比较器输出的是0/1（黑/白），只能做二值判断。用ADC读V_sensor的模拟值可以得到0~4095的精细值——几十级灰度梯度！

这对高级巡线算法（如PID巡线）非常重要——不仅能知道"偏了"，还能知道"偏了多少"。

所以实际巡线小车中，传感器输出往往是**模拟量**，而不是比较器后的数字量。本项目用比较器是为了降低初学难度（后面的MCU项目可以直接GPIO读取），但建议在排针上同时引出V_sensor模拟信号。

### 10.2 为什么不能用普通可见光LED代替红外LED？

普通可见光LED也行，但：
- 红外LED的波长（850nm或940nm）光敏三极管响应最灵敏
- 红外光不受环境可见光（日光灯、阳光）的干扰——室内开着灯照样巡线
- 某些红外接收管内置了可见光滤光片

### 10.3 比较器 vs 运放

比较器（LM393）和运放（如LM358）看起来很像，但比较器是专门为"比较两个电压"优化的——输出是开关量，响应速度快。运放如果用做比较器会响应慢、可能振荡。所以做阈值判断就用比较器。
