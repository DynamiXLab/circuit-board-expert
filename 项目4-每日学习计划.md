### 下午（1.5h）：为什么要稳压

**（一）不稳压会怎样？**

看几个真实场景：

```
场景1：巡线小车用7.4V锂电池
  满电时：8.4V  → MCU还撑得住
  用到一半：7.0V → MCU开始不稳定
  快没电时：6.0V → MCU重启 / 舵机乱转
  ❌ 没有稳压，小车跑着跑着就"抽风"
```

```
场景2：用USB给传感器供电
  USB标称5V，实际在4.5V~5.5V之间波动
  灰度传感器对电压敏感——电压一变，读数就偏
  ❌ 没有稳压，传感器数据不可靠
```

**结论**：稳压电路 = 给电子系统一个"不管外面怎么变，我这边岿然不动"的稳定电压。

**（二）两种稳压方案的直觉选择**

```
你的需求：电池7.4V → 5V（给舵机/MCU，需要2A）
```

| 方案 | 效率 | 发热 | 选哪个？ |
|------|:----:|:----:|:---------|
| 方案A：用LDO直接降 | 5÷7.4=**68%** | 损耗(7.4-5)×2=**4.8W**🔥🔥🔥 | ❌ 热到能煎蛋 |
| 方案B：用Buck先降 | **~90%** | 损耗≈**0.5W** | ✅ 几乎不热 |

```
你的需求：5V → 3.3V（给传感器，只要100mA）
```

| 方案 | 效率 | 发热 | 选哪个？ |
|------|:----:|:----:|:---------|
| 方案A：用LDO降 | 3.3÷5=**66%** | 损耗(5-3.3)×0.1=**0.17W** | ✅ 几乎不发热 |
| 方案B：用Buck降 | ~85% | 损耗≈0.05W | ❌ 杀鸡用牛刀 |

**所以我们的架构是**：电池 → **Buck(MP1584)** → 5V → **LDO(662K)** → 3.3V

| 任务 | 操作 |

## 第4天 — 嘉立创EDA实践 · 布局与走线

> 🎯 **今天目标**：把原理图转成PCB，完成全部布局布线

### 上午（1h）：原理图转PCB + 布局

打开 **`项目4-嘉立创EDA实践.md`** 文件：

| 步骤 | 操作 | 完成标准 |
|------|------|---------|
| 1 | 原理图→PCB（设计→转换为PCB） | 元件都在PCB中出现了 |
| 2 | 设置板框：30mm×25mm | 板框画好 |
| 3 | 摆放**MP1584**：板子中央偏左 | 芯片放好 |
| 4 | 摆放**输入电容C1/C2**：紧贴MP1584的IN和GND脚（≤5mm） | **铁律一！** |
| 5 | 摆放**BST电容Cbst**：紧挨BST和SW脚 | 自举电容 |
| 6 | 摆放**电感L1**和**二极管D1**：紧挨SW脚，三者形成紧凑三角形 | **铁律二！** |
| 7 | 摆放**输出电容C3/C4**：紧挨电感和Vout侧 | |
| 8 | 摆放**反馈电阻R1/R2**：靠近FB脚，远离电感和SW区域 | **铁律三！** |
| 9 | 摆放**662K+其电容**：板子右侧，远离MP1584的SW区域 | LDO对噪声敏感 |
| 10 | 摆放**输入/输出端子**：板子左右两侧边缘 | 接插件靠边原则 |

> 💡 **布局口诀**：大芯片先放 → 电容贴芯片 → 电感二极管三角形 → 反馈远远躲 → LDO靠边站

### 下午（2h）：走线

| 步骤 | 走线 | 线宽 | 完成标准 |
|:----:|------|:----:|---------|
| 1 | **GND网络**：先铺GND铜（底层整片铺铜） | — | GND不用走线，靠铺铜 |
| 2 | **Vin → 输入电容 → MP1584的IN** | **1.5~2mm** | 短而粗 |
| 3 | **MP1584的SW → 电感L1** | **1.5mm** | 短而粗，但要面积小 |
| 4 | **电感L1 → 输出电容 → Vout端子** | **1.5~2mm** | |
| 5 | **FB走线**：Vout → R1 → FB，远离电感 | **0.25mm**（细线） | **铁律三！** 不打过孔 |
| 6 | **EN → R3 → Vin** | 0.5mm | |
| 7 | **662K走线**：Vin→C8→662K(③)→662K(②)→C9→Vout | 0.5mm | LDO电流小 |
| 8 | **GND过孔**：输入/输出电容的GND焊盘各打2~3个过孔到底层 | — | 降低回路阻抗 |

### 晚上（0.5h）：收尾检查

| 任务 | 操作 |

| □ 丝印调整 | 标注Vin/5V/GND/3.3V，位号不重叠 |
| □ 目检 | 对照布局口诀再扫一遍 |

### 上午（0.5h）：最终检查清单

```
□ 原理图

   □ 电源网络没有短路（Vin/GND之间没有直连）
   □ MP1584的BST接了0.1μF自举电容到SW
   □ 续流二极管SS34极性正确（阴极接SW）

   □ DRC零错误
   □ 输入电容离MP1584的IN和GND ≤5mm
   □ BST电容紧挨BST和SW
   □ FB走线远离电感和SW区域
   □ 底部GND铺铜完整连续
```
