# D9 — 原理图导入 PCB & 板框设计

> **学习目标**：把画好的原理图导入 PCB，学习画板框并设置原点，为布局布线做准备。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 从原理图到 PCB

点击顶部 **"设计 → 导入变化到 PCB"**（快捷键 `Alt+I`）：

1. 所有元件"飞"到 PCB 画布上，用**飞线（Ratlines）**显示连接关系——飞线是白色的临时虚拟连线，表示"这两个焊盘需要连接"，它不是实际铜箔，只起引导作用。你把元件摆好、走好线后，飞线就消失了
2. 元件最初堆在一起——这是正常的，下一步开始布局

### 导入前检查

- [ ] 所有元件都指定了封装？如果某元件封装是空的，导入会报错
- [ ] ERC 零错误
- [ ] 网络标号命名规范（VCC / GND / UART_TX 等）

## 📖 板框设计

板框（Board Outline）就是 PCB 的外形尺寸——板子最终长什么形状。

### 嘉立创 EDA 画板框步骤

1. 在 PCB 模式下，层选切换到 **"板框（Board Outline）"** 层
2. 用导线工具画一个矩形（也可画任意形状）
3. 选中画好的闭合边框 → 右键 → "设置为板框"
4. 板框变绿（不同版本颜色可能不同），四角可加圆角

### 板框设计要点

- **圆角比直角好**：直角在制造时应力集中，易开裂。建议 R2mm 圆角
- **定位孔（Mounting Hole）**：用于把 PCB 固定在外壳或工作台上的螺丝孔。**定位孔通常接地或不接任何网络，四周用铜皮/焊盘加强，防止螺丝拧紧时板子开裂**。四角放 3mm 直径的定位孔（孔径 3.2mm，比螺丝大 0.2mm 方便穿过）
- **尺寸策略**：尽量在 **10cm×10cm** 以内（嘉立创最优惠价格区间）
- **不要做太小**：小于 1cm×1cm 的板子不好拿、不好焊

### 设置原点

通常将板框左下角设为原点 (0,0)：

菜单"编辑 → 设定原点" → 点击板框左下角

这样所有坐标都是正的，导出 Gerber 时更清晰。

## 🔑 板框尺寸参考

| 用途 | 推荐尺寸 | 备注 |
|------|---------|------|
| 练习板 | 30×20mm | 元件少，够用 |
| STM32 最小系统 | 50×40mm | 够放芯片+电源+排针 |
| USB 转串口 | 25×15mm | 小型化 |
| Arduino 扩展板 | 68×53mm | 兼容 Arduino 尺寸 |

## ⚠️ 常见坑

- **板框没闭合** → 系统不认为是有效板框，导出报错
- **原点乱放** → 导出 Gerber 时坐标乱七八糟，建议固定在左下角
- **板子太挤** → 元件间距不够，布线走不通。板子省的钱不够折腾的
