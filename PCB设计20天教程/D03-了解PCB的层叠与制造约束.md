## 📖 了解 PCB 的各个"层"

PCB 不是一块简单的板子——它是由多个功能层叠压而成的"三明治"。每个层有不同的颜色和作用。下面按嘉立创 EDA 专业版 v3.2.131 的层管理器顺序逐一说明。

### 两层板的物理空间关系

```
┌─────────────────────────┐
│   顶层丝印（白色字符）    │  ← 印位号、Logo、版本号（无电气功能）
├─────────────────────────┤
│   顶层阻焊（绿色油墨）    │  ← 保护铜箔，焊盘处开窗
├─────────────────────────┤
│   顶层铜箔（Top Layer）   │  ← 放元器件 + 走线（1oz ≈ 35μm 厚）
├─────────────────────────┤
│    FR-4 玻璃纤维基板      │  ← 绝缘层，1.6mm 厚（典型值）
├─────────────────────────┤
│   底层铜箔（Bottom Layer）│  ← 走线（也可以放元器件）
├─────────────────────────┤
│   底层阻焊（绿色油墨）    │  ← 保护底层铜箔
├─────────────────────────┤
│   底层丝印（白色字符）    │  ← 底部标识（一般不用）
└─────────────────────────┘
```

### 一、电气层（实际有铜箔，能导电）

| 层名 | EDA 中颜色 | 作用 |
|------|-----------|------|
| **Top Layer（顶层信号层）** | 红色 | 主要放置元器件，也可以走线。双面板中是最重要的走线层 |
| **Bottom Layer（底层信号层）** | 蓝色 | 主要走线，也可以放元器件（称为"背面贴片"），常用于铺 GND 铜皮 |
| **Internal Planes（内电层）** | — | 仅在四层及以上 PCB 中使用，整片铜膜作为电源层或地层 |

> 💡 **PCB 常识：为什么四层板比双层板好？**
> 双层板的信号走线和电源/地线共用同一层铜箔，信号回路面大，容易受干扰。
> 四层板有专门的电源层和地层（内电层），信号层在外层，电源/地独立成层，信号完整性大幅提升。
> 但四层板成本是双面板的 2-3 倍，新手工匠精神先从双面板练起。

### 二、丝印层（白色字符，不导电）

| 层名 | EDA 中颜色 | 作用 |
|------|-----------|------|
| **Top Overlay（顶层丝印）** | 黄色 | 标注元器件投影轮廓、位号（R1/C1）、标称值、极性标记、版本号。**这是焊接和查板时的重要视觉参考** |
| **Bottom Overlay（底层丝印）** | 棕色 | 与顶层丝印作用相同。如果所有标注在顶层已包含，底层可以不印 |

> 丝印是白色油墨印刷在阻焊层之上的，**没有电气功能**，但位置错了可能会被焊盘挡住看不清楚。

### 三、阻焊层（绿色油墨，绝缘）

| 层名 | EDA 中颜色 | 作用 |
|------|-----------|------|
| **Top Solder（顶层阻焊）** | 紫色 | PCB 正面绿色油墨层，焊盘处开窗露出铜 |
| **Bottom Solder（底层阻焊）** | 绿色 | PCB 背面绿色油墨层 |

阻焊层使用**负片输出**：你在 EDA 中画阻焊层时，画了的地方反而会露出铜皮（称为"开窗"），没画的地方盖绿油。默认情况下走线和铜皮都是盖绿油的，只有焊盘会自动开窗。

**阻焊的三大作用**：①防止焊锡意外桥接短路 ②防止铜箔氧化 ③绝缘保护

### 四、助焊层（钢网层，辅助焊接）

| 层名 | 作用 |
|------|------|
| **Top Paste（顶层助焊）** | 也叫锡膏层/钢网层，对应 SMD 贴片焊盘的位置，用于制作焊接钢网 |
| **Bottom Paste（底层助焊）** | 底部贴片焊盘的钢网 |

钢网是一张薄钢板，在对应焊盘位置开孔。刷锡膏时锡膏透过钢网孔印到焊盘上，再把元件放上去过回流焊。**自己手工焊接不需要钢网，但工厂 SMT 贴片必须用**。

### 五、机械/禁布/多层（辅助层）

| 层名 | 作用 |
|------|------|
| **Board Outline（板框）** | 定义 PCB 的外形尺寸和形状，工厂据此切割板子 |
| **Keep Out Layer（禁布层）** | 定义布线边界，具有电气特性的走线不可超出此边界。**注意：不要同时用板框层和禁布层画外形，容易混淆** |
| **MultiLayer（多层）** | 抽象层，焊盘和过孔需要穿透整个板子，设置在此层才能在所有层显示 |
| **Drill Drawing（钻孔参考图）** | 标注钻孔的位置和尺寸信息 |

### 💡 PCB 常识：焊环（Annular Ring）

焊环是过孔或直插焊盘的铜环部分，计算公式：

```
焊环宽度 = (外径 - 内径) / 2
```

嘉立创要求最小焊环 ≥ 0.125mm。如果你的过孔内径 0.3mm、外径 0.45mm：
```
焊环 = (0.45 - 0.3) / 2 = 0.075mm → ❌ 不合格！
```
所以建议过孔外径至少 0.6mm（内径 0.3mm），焊环 = 0.15mm ✅。

## 🔑 关键制造约束（嘉立创标准）

这是 **100% 必须记住的参数**——违反它们做出来的板子可能无法生产。

| 参数 | 嘉立创最低标准 | 建议设计值 | 为什么 |
|------|---------------|-----------|--------|
| 最小线宽 | 5mil (0.127mm) | **≥6mil** | 太细了工厂良率下降 |
| 最小线距 | 5mil | **≥6mil** | 间距太小会短路 |
| 最小过孔内径 | 0.2mm (8mil) | **≥0.3mm** | 钻孔太小容易偏 |
| 最小过孔外径 | 0.45mm (18mil) | **≥0.6mm**  |焊环（Annular Ring）过小会降低钻孔偏移容忍度，增加断环风险和生产不良率。
| 最小字符线宽 | 0.15mm (6mil) | 6mil | 细了印不出来 |
| 钻孔到铜的距离 | 0.2mm | **≥0.25mm** | 近了会击穿 |
| 板厚 | 0.4–2.0mm | 1.6mm（默认） | 标准值最便宜 |
| 铜厚 | 1oz（默认） | 1oz | 大电流走线用 2oz |
| 板子尺寸 | 0.5cm×0.5cm 起 | ≤10cm×10cm | 最优惠价格区间 |

> 💡 **铁律**：给自己留余量。嘉立创能做 5/5mil，但新手一律用 **6/6mil** 甚至 **8/8mil**，良率更高、坏板更少。

### 什么是 1oz 铜？

1oz = 1 平方英尺面积上铺 1 盎司铜 = 约 35μm 厚。

在这个厚度下：
- 1mm 宽走线 ≈ 载流 1A（温升 10°C）
- 10mil（0.254mm）宽走线 ≈ 载流 0.5A
- 100mil（2.54mm）宽走线 ≈ 载流 4-5A

## 🔑 常见 PCB 术语

| 术语 | 含义 |
|------|------|
| **FR-4** | 最常见的 PCB 基板材料——玻璃纤维布浸渍环氧树脂压制而成。绝缘性好、强度高、阻燃，是绝大多数 PCB 的标准板材 |
| **HASL（热风整平）** | 焊盘表面处理工艺——板子浸入焊锡槽后，用热风刀吹掉多余焊锡，留下平整焊盘。HASL（有铅）最便宜，但表面不如沉金平整 |
| **沉金（ENIG）** | 更高级的表面处理——化学镀镍后浸金。表面极平整适合细引脚芯片，抗氧化好，但价格比 HASL 贵 2-3 倍 |
| **阻焊（Solder Mask）** | PCB 表面的绝缘油墨层（默认绿色），覆盖不需要焊接的铜箔。**作用：①防止焊锡意外桥接短路 ②防止铜箔氧化 ③绝缘保护**。焊盘处会"开窗"露出铜，方便焊接 |
| **1oz / 2oz** | 铜箔厚度单位。1oz = 1 平方英尺面积上铺 1 盎司铜 ≈ 35μm 厚。2oz = 70μm 厚，载流能力翻倍，适合大电流板子 |

PCB线宽与流经电流的关系：
I=K*T的0.44次方*A的0.75次方
（K为修正系数，一般覆铜线在内层时取0.024，在外层时取0.048
T为最大温升，单位为摄氏度（铜的熔点是1060°℃）
A为覆铜截面积，单位为平方MIL（不是毫米mm，注意是squaremil.）
I为容许的最大电流，单位为安培（amp）
一般10mi1=0.010inch=0.254可流1A
250MIL=6.35mm,为8.3A


## ✏️ 今日作业

1. 打开嘉立创 EDA 的 PCB 编辑模式（新建一个空白 PCB）
2. 按 `L` 键打开层管理器，记住每一层的颜色和名称：
   - Top Layer（红色）
   - Bottom Layer（蓝色）
   - Top Solder（紫色）
   - Bottom Solder（绿色）
   - Top Overlay（黄色）
   - Board Outline（灰色）
3. 在 PCB 上分别画一段 6mil 和一段 10mil 的走线，观察粗细差异
4. 放置一个过孔：内径 0.3mm、外径 0.6mm，再放一个内径 0.2mm、外径 0.45mm 的，对比视觉差异

> **完成标志**：你能不看笔记说出嘉立创的最低线宽、最小线距、最小过孔内径/外径这 4 个数字。
