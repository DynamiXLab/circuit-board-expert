## 📖 原理图绘制规范

一个好的原理图 = **让别人（和未来的自己）一眼看懂。**

### 核心原则

1. **信号流向从左到右** — 输入在左，输出在右，电源在上，地在下
2. **标注网络标号（Net Label）** — 网络标号相当于"有名字的导线"，相同名字的网络标号在电气上是连通的。长导线用网络标号代替，避免交叉混乱
3. **加注释（Text/Label）** — 标注电压值、功能说明、版本号
4. **分模块画** — 大的原理图拆成多张 Sheet（D6 会讲）

### 好 vs 坏的对比

```
❌ 差：导线乱拉，交叉成蜘蛛网，没有网络标号
✅ 好：模块清晰，信号从左到右，关键节点有网络标号
```

## ✏️ 今日实战：画一个 NE555 + CD4017 LED 流水灯

这是你的**第一张完整原理图**。

### 电路原理

用 NE555 产生时钟脉冲，送给 CD4017（十进制计数器/分配器），让 10 个 LED 依次点亮，形成"流水"效果。**滑动变阻器（电位器）调节流水速度。**

### 模块化电路连接

```
┌─────────────────────────────────────────────────────────────┐
│                        电源部分                              │
│  KF301-5.0-2P (输入端) ──┬── VCC                           │
│                          └── GND                           │
├─────────────────────────────────────────────────────────────┤
│                    NE555 时钟产生电路                        │
│  ┌─────────────────────────────────┐                       │
│  │  VCC ──────────┬─────── 8 (VCC)  │                       │
│  │                │                │                       │
│  │  RP(50k)─┬─────┤ 7 (DIS)       │  ← 滑动变阻器调频率    │
│  │  (滑动变阻) │    │                │                       │
│  │           │     │ NE555          │                       │
│  │  R1(10k)─┴─────┤ 6 (THR)       │                       │
│  │                │                │                       │
│  │  C1(100nF)─────┤ 2 (TRI)       │                       │
│  │                │                │                       │
│  │  C2(10μF)──────┤ 1 (GND)       │                       │
│  │                │       │        │                       │
│  │           ┌────┤ 4 (RST)│       │                       │
│  │           │    └────────┴────────┘                       │
│  │           └──── VCC（复位拉高）                          │
│  └─────────────────────────────────┘                       │
│                          │                                 │
│                  3 (OUT)─┤  ──→ CLK (CD4017 的时钟输入)     │
├─────────────────────────────────────────────────────────────┤
│                    CD4017 流水灯驱动                         │
│  ┌─────────────────────────────┐                           │
│  │  CLK ───────── 14 (CLK)      │                           │
│  │                      │       │                           │
│  │  VCC ───────── 16 (VDD)     │                           │
│  │  GND ───────── 8  (VSS)     │                           │
│  │                      │       │                           │
│  │  RST ──── GND (通过电阻接地) │                           │
│  │                      │       │                           │
│  │    Q0 ── R_10k ── LED1      │                           │
│  │    Q1 ── R_10k ── LED2      │                           │
│  │    Q2 ── R_10k ── LED3      │  ← 共 10 个输出           │
│  │    ...           ...        │    依次输出高电平          │
│  │    Q9 ── R_10k ── LED10     │    LED 逐个点亮形成流水    │
│  └─────────────────────────────┘                           │
└─────────────────────────────────────────────────────────────┘
```

> 💡 **PCB 常识：CD4017 是什么？**
> CD4017 是 CMOS 十进制计数器/分配器。每收到一个时钟脉冲，它的 10 个输出脚（Q0-Q9）会依次变为高电平。配合 NE555 产生的时钟脉冲，就能让 10 个 LED 依次点亮，看起来像"流水"一样。调节 NE555 的频率就能控制流水速度。

### 元件清单

| 位号 | 型号/值 | 封装 | 数量 |
|------|---------|------|------|
| U1 | NE555 | DIP-8_L9.2-W6.4-P2.54-LS8.7-BL | 1 |
| U2 | CD4017 | DIP-16_16P-L20.2-W6.6-P2.54-LS7.6-BL | 1 |
| RP | 滑动变阻器 50kΩ | RES-ADJ-TH_3P-L9.5-W4.9-P2.50-L_3296X | 1 |
| R1 | 电阻 10kΩ | MF1/4W-10K±1%-ST52 | 1 |
| R2-R11 | 电阻 10kΩ | MF1/4W-10K±1%-ST52 | 10 |
| C1 | 电容 100nF | KM105M050D11RR0VH2FP0 | 1 |
| C2 | 电容 10μF | KM105M050D11RR0VH2FP0 | 1 |
| LED1-10 | LED 黄色直插 5mm | 333-2UYD/S530-A3 LED-TH_BD5.0-P2.54-FD | 10 |
| J1 | 电源输入端 | KF301-5.0-2P | 1 |

### 操作步骤（按顺序来，不容易漏）

1. 新建原理图，图纸大小选 A3（元件多，A4 可能不够）
2. 从元件库搜索并放置 NE555、CD4017
3. 放置滑动变阻器 RP（50kΩ）——三个引脚，中间脚是滑动端
4. 放置电阻 R1(10kΩ)、R2-R11（10kΩ × 10 个）
5. 放置电容 C1(100nF)、C2(10μF)
6. 放置 LED1-LED10（注意正负极——正极接限流电阻，负极接 GND）
7. 放置电源输入端 KF301-5.0-2P
8. 用 ALT+W 画导线连接，先画 NE555 部分（时钟产生），再画 CD4017 部分（流水驱动）
9. 用 ALT+N 放置网络标号：VCC、GND、CLK（连接 NE555 输出和 CD4017 时钟输入）
10. 调整位号使其清晰不重叠
11. 加文字注释：输入电压、流水方向、版本号

> 💡 **PCB 常识：为什么用滑动变阻器（电位器）代替固定电阻？**
> 固定电阻只能产生固定频率。电位器可以实时调节 NE555 的充放电时间常数，从而改变时钟频率，实现"可调速流水灯"。这是 PCB 设计中"可调参数"电路的常见手法。

- [ ] 电源和地都标注了？
- [ ] LED 极性对了？（正极接限流电阻，负极接地）
- [ ] 所有元件都有位号（R1, C1, U1... 即 Reference Designator，焊接时用来对照物料清单的唯一标识）？
