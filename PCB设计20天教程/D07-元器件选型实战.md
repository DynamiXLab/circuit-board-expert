# D7 — 元器件选型实战

> **学习目标**：面对一个功能需求时，学会如何做元器件选型决策——电阻、电容、三极管、MOS 管、稳压芯片怎么选。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 选型决策树

拿到一个功能需求后，按以下顺序决策：

```
需求功能
  │
  ├─ 有现成集成芯片吗？
  │   （如 NE555 定时器 / STM32 单片机 / L9110S 电机驱动）
  │    └─→ 查数据手册 → 确认引脚兼容 → 选封装
  │
  └─ 用分立元件搭？
     （如三极管放大 / RC 滤波 / 稳压二极管）
      └─→ 计算参数 → 选标准值（E24/E96 系列）
```

**原则：优先用集成芯片。** 分立元件虽然便宜，但零件多、PCB 面积大、可靠性低。

## 📖 电阻选型要点

| 参数 | 怎么选 |
|------|--------|
| **阻值** | 通用电路优选 E24 系列（5% 精度）；反馈电路用 E96（1% 或更高） |
| **功率** | 0603 = 1/10W，0805 = 1/8W，1206 = 1/4W → 计算 `P = I²R` 确认没超 |
| **封装** | 一般信号线用 0603 或 0805；大电流用 1206 或直插 |
| **精度** | 分压/限流 5% 够用；反馈电路用 1% |
| **温度系数** | 普通 100ppm/°C 够用，精密电路用 25ppm/°C 或更低 |

### E24 标准值速记

```
10  11  12  13  15  16  18  20  22  24  27  30
33  36  39  43  47  51  56  62  68  75  82  91
```

乘以 10ⁿ 就是所有标准阻值：1.0k、1.1k、1.2k… 10k、11k…

> 记住：**计算值不是标准值，必须就近取标准值。**

## 📖 电容选型要点

| 类型 | 用途 | 优点 | 缺点 |
|------|------|------|------|
| **MLCC（多层陶瓷电容）** | **去耦（Decoupling）**——给 IC 提供瞬时电流，滤除电源上的高频噪声 / 滤波，高频特性好 | 小体积、**低 ESR**（等效串联电阻，ESR 越低滤波效果越好）、便宜 | 容量小、有 DC Bias 压降 |
| **铝电解电容** | 大容量储能，低频滤波 | 容量大（μF~mF）、便宜 | 体积大、ESR 高、寿命有限 |
| **钽电容** | 容量大体积小 | 稳定、比电解小 | 耐压低、**烧了会短路起火** |

### ⚠️ MLCC 的 DC Bias 效应（新手最容易忽略的坑）

同一颗 10μF/25V 的 MLCC，在 5V 下实际容量可能只剩 **6μF**！小封装（0603/0402）尤为严重。

**怎么办？** 关键场合（电源输出滤波）用更大封装或更高耐压值，或者查数据手册的 DC Bias 曲线。

## 📖 三极管 vs MOS 管选型

| | 三极管 (BJT) | MOS 管 |
|----|-------------|--------|
| 控制方式 | 电流控制（Ib → Ic） | 电压控制（Vgs → Id） |
| 导通压降 | Vce(sat) ≈ 0.2V | Rds(on) 可低至几 mΩ |
| 驱动功率 | 需要基极电流（mA 级） | 几乎不耗电流（只需电压） |
| 适合 | 小信号开关、模拟放大 | 大电流开关、电源、DC-DC |
| 典型 NPN | SS8050 (Ic=1.5A) | AO3400 (Id=5.7A) |
| 典型 PNP | SS8550 (Ic=1.5A) | AO3401 (Id=4.2A) |

**选型口诀**：小信号放大用 BJT，大电流开关用 MOS。

## ✏️ 今日作业：选型实战

**需求**：用 3.3V 单片机控制一个 **12V / 1A 的直流电机**。

1. 列出至少 2 种可行的驱动方案（如分立 MOS 管方案 / 集成电机驱动芯片方案）
2. 写出每种方案的选型理由
3. 推荐其中一种，并给出具体型号和在立创商城的物料编号
4. 计算：如果用分立 N-MOS 管，栅极能不能直接用 3.3V 驱动？需要加驱动电路吗？

**可选参考芯片**：L9110S / DRV8837 / TB6612FNG / 分立 AO3400 + AO3401 H 桥

> **完成标志**：对给定的任意功能需求（LED 驱动 / 电机驱动 / 电源转换），你能说出选型时优先考虑哪几个参数。
