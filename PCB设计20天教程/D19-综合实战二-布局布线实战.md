## 📖 布局策略（针对小尺寸板）

### 目标尺寸：25mm × 15mm

```
┌──────────────────────────┐
│  ○ 定位孔           ○    │
│                          │
│    ┌─────────┐           │
│  ──┤USB Type-C│          │
│  ──┤         │   ┌───┐  │
│    │         │   │CH │  │
│    │         │   │340│  │
│    └─────────┘   │ C │  │
│                  └───┘  │
│    ┌──┐ ┌──┐ ┌──┐      │
│    │R1│ │R2│ │R3│  ┌──┐│
│    └──┘ └──┘ └──┘  │JP││
│    ┌──┐ ┌──┐ ┌──┐  │1 ││
│    │L1│ │L2│ │L3│  └──┘│
│    └──┘ └──┘ └──┘      │
│  ○                 ○   │
└──────────────────────────┘
```

### 布局原则

1. **USB 座靠板边**—Type-C 放在板子一端
2. **CH340C 紧贴 USB 座**—UD+/UD- 走线最短
3. **排针在另一端**—4P 排针引出 TX/RX/VCC/GND
4. **去耦电容紧贴 CH340C**—C1(100nF) 在 16 脚旁边
5. **LED 和限流电阻放在侧面**—不挡走线通道

## 📖 布线策略

### 优先级

```
① USB 差分线 (UD+/UD-) → ② 电源 (VCC/GND) → ③ TX/RX 信号 → ④ LED 指示
```

### USB 差分线布线要求

UD+ 和 UD- 是一对**差分信号（Differential Signal）**——信息通过两根线上电压差来传递，抗干扰能力强。两根线必须平行走、等长度、靠得近，否则外部噪声对两根线的干扰不相等，信号会出错。需要特别注意：

- ✅ **平行走线**：两根线走在一起
- ✅ **等长**：长度差 ≤ 5mm（对于低速 USB 1.1 够用）
- ✅ **尽量短**：CH340C 到 USB 座不超过 10mm
- ✅ **包地**：两侧各有一条 GND 走线或靠近铺铜
- ❌ **不要加过孔**：UD+/UD- 最好在同一层，不要换层

### 电源布线

- VCC 走线 ≥ 15mil（USB 供电 5V/500mA）
- GND 用铺铜覆盖整板

## 📖 设计参考参数

| 参数 | 值 |
|------|-----|
| 板尺寸 | 25mm × 15mm |
| 最小线宽/线距 | 6mil / 6mil |
| 板厚 | 1.6mm |
| 铜厚 | 1oz |
| 过孔内径/外径 | 0.3mm / 0.6mm |

## ⚠️ 小尺寸板注意事项

- **焊盘之间保持距离**：排针的焊盘之间、IC 引脚之间，留够焊接空间
- **元件不要靠板边太近**：建议留 ≥ 0.5mm
- **不要用 0402 封装**：除非你焊接技术已经很好，0603 是新手友好极限
- **定位孔要放**：四角放 2.5mm 直径的定位孔（孔径 2.7mm）
