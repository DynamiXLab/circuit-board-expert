# D18 — 综合实战（一）：USB 转串口工具

> **学习目标**：独立完成一个实用项目的原理图设计——USB 转 TTL 串口工具（CH340G），从需求分析到原理图完成。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 项目简介

**项目名称**：USB 转 TTL 串口模块

**功能**：通过 USB 口与单片机的 UART 串口通信，用于调试和程序下载。

**难度**：⭐⭐（简单，单芯片方案）

**学习价值**：这是嵌入式开发中**最常用的调试工具**，做一块放桌上，以后所有单片机项目都能用。

## 📖 需求分析

### 功能需求

1. USB（Type-C 或 Micro USB）→ 串口 TTL（3.3V 电平）
2. 支持 3.3V 输出（常见单片机电平）
3. 带电源指示灯（通电即亮）
4. 带收发指示灯（TX/RX 时闪烁）

### 核心芯片选型

| 芯片 | 说明 | 选型理由 |
|------|------|---------|
| **CH340G** | USB 转串口芯片，SSOP-16 封装 | 便宜、稳定、国产 |
| **CH340C** | 同上，但内置晶振，更省事 | 少焊两个电容 |
| **CP2102** | 替代方案，更贵 | 兼容性更好 |

**推荐选 CH340C**——内置晶振，外围元件最少。

### 所需元件清单

| 位号 | 型号/值 | 封装 | 数量 |
|------|---------|------|------|
| U1 | CH340C | SOP-16 | 1 |
| C1 | 100nF | 0805 | 1 |
| C2 | 10μF | 0805 | 1 |
| LED_PWR | LED 绿色 | 0603 | 1 |
| LED_TX | LED 蓝色 | 0603 | 1 |
| LED_RX | LED 黄色 | 0603 | 1 |
| R1,R2,R3 | 1kΩ | 0603 | 3 |
| J1 | USB Type-C 母座 | 贴片 | 1 |
| JP1 | 排针 2.54mm 4P | — | 1 |

## 📖 今日任务

### 第一步：下载数据手册

1. 下载 CH340C 的数据手册
2. 找到引脚定义——特别注意：
   - VCC（电源）
   - GND（地）
   - UD+/UD-（USB 数据线）
   - TXD/RXD（串口）
   - RS232 模式相关引脚（不用，用 TTL 模式）

### 第二步：根据手册绘制原理图

**参考电路（来自数据手册的典型应用图）：**

```
         USB Type-C
        ┌──────────┐
        │ 1: VBUS  ├──────┬──── VCC (5V)
        │ 2: D-    ├──────┤ UD- (CH340C 6脚)
        │ 3: D+    ├──────┤ UD+ (CH340C 7脚)
        │ 4: GND   ├──────┤ GND
        └──────────┘
                          │
                     ┌────┴────┐
                     │ CH340C  │
                     │         │
                     │ 16 VCC  │──┬── C1(100nF) ── GND
                     │         │  └── C2(10μF)  ── GND
                     │ 6  UD-  │── USB D-
                     │ 7  UD+  │── USB D+
                     │ 3  TXD  │──→ RX (对端设备)
                     │ 4  RXD  │←── TX (对端设备)
                     │         │
                     │ 5  GND  │── GND
                     │ 9  V3   │── C3(100nF) ── GND（内部 3.3V 稳压输出）
                     └─────────┘

LED 指示灯：
VCC ── R1(1k) ── LED_PWR(绿) ── GND      （通电即亮）
TXD ── R2(1k) ── LED_TX(蓝) ── VCC      （发送数据时闪烁）
RXD ── R3(1k) ── LED_RX(黄) ── VCC      （接收数据时闪烁）
```

### 第三步：设计要点

1. **USB 差分线**（D+/D-）走线尽量短且等长
2. **去耦电容** C1(100nF) 必须紧贴 CH340C 的 VCC 引脚（16 脚）
3. V3 引脚（9 脚）的电容 C3 必须接，不能省
4. TXD/RXD 对外引出到 4P 排针
5. 板子尺寸建议 25×15mm（非常小，方便携带）
