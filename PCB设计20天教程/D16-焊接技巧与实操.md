# D16 — 焊接技巧与实操

> **学习目标**：掌握手工焊接贴片元件的基本技能，把这几天设计的板子焊出来。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 焊接前的准备

### 工具清单

| 工具 | 用途 | 建议 |
|------|------|------|
| 恒温电烙铁 | 焊接 | 建议 T12 或 936 焊台，温度 350°C |
| 焊锡丝 | 焊接材料 | 直径 0.5-0.8mm，含松香芯 |
| 助焊剂（松香/焊油） | 辅助焊接 | 不必须，焊锡丝自带松香通常够用 |
| 镊子 | 夹取贴片元件 | 尖头镊子，**一定要有**——0603 贴片只有 1.6×0.8mm 大小，手指根本拿不住 |
| 吸锡器/吸锡带 | 拆焊 | 焊错了要拆 |
| 洗板水/酒精 | 清洁板子 | 焊完擦掉松香残留 |
| 放大镜/手机微距 | 检查焊点 | 检查虚焊、连锡 |

### 焊接前的检查

1. **板子检查**：收到 PCB 后先看外观——有没有划伤、氧化、钻孔偏位
2. **元件核对**：BOM 单拿出来，逐个核对型号和封装
3. **静电防护**：如果焊接 MOS 管或 CMOS 芯片，最好戴静电手环

## 📖 贴片元件焊接步骤

### 第一步：焊接 IC（NE555 如果是 SOP-8 封装）

1. 在 PCB 的一个对角焊盘上镀一点点锡
2. 用镊子夹住 IC，对准方向（注意 1 脚标记的缺口/圆点）
3. 加热对角焊盘，把 IC 按住焊上去，固定
4. 焊接剩下的引脚：先镀锡，用拖焊法或逐个焊

**拖焊法（最适合多脚 IC）**：
1. 在 IC 所有引脚上涂助焊剂
2. 用烙铁尖带一大坨锡，沿着引脚排从头拖到尾
3. 多余的锡会被烙铁带走，引脚之间自动断开

### 第二步：焊接电阻电容（0603/0805）

**焊盘（Pad）** 是 PCB 上镀铜的焊接区域。贴片焊盘是两个平面的铜块，没有孔，焊接时用锡把元件引脚和焊盘熔在一起。而直插焊盘是贯穿板子的孔+环形铜箔，元件引脚穿过后焊接。

1. 先在 PCB 的一个焊盘上镀一点锡
2. 用镊子夹住元件放在焊盘上
3. 加热镀锡的焊盘，元件一头就被焊住了
4. 镊子松开，焊另一头

**关键技巧**：焊第一头时不用太多锡，够固定就好。确认位置对正再焊第二头。

### 第三步：焊接排针/插件

1. 把排针插进 PCB 孔中（从顶层插到底层露出引脚）
2. 焊其中一个引脚，然后检查排针是否垂直
3. 如果不垂直，加热该引脚并扶正
4. 再焊剩下的引脚

## 📖 焊点质量判断

| ✅ 好焊点 | ❌ 坏焊点 |
|-----------|-----------|
| 光滑、有光泽 | 粗糙、暗沉 |
| 焊锡铺满整个焊盘 | 焊锡只覆盖了部分焊盘 |
| 元件和焊盘之间形成"月牙" | 元件翘起（立碑） |
| 引脚轮廓隐约可见 | 太多锡成一坨（连锡） |

### 常见焊接缺陷

| 缺陷 | 原因 | 修复 |
|------|------|------|
| **虚焊** | 温度不够/焊盘氧化 | 加助焊剂重新焊 |
| **连锡（桥接）** | 锡太多 | 吸锡带吸掉多余的锡 |
| **立碑（曼哈顿现象）** | 两端受热不均 | 先焊一端，再加热+镊子扶正 |
| **焊盘脱落** | 加热时间过长/机械暴力 | 飞线补救（刮开附近走线飞线连接） |

## 💡 温度建议

| 焊料 | 烙铁温度 |
|------|---------|
| 有铅焊锡（Sn63Pb37） | 300-330°C |
| 无铅焊锡（SAC305） | 340-370°C |
| 修理拆焊 | 350-380°C |

> 每个焊点加热时间 ≤ 3 秒。超过 5 秒焊盘可能脱落。

## ⚠️ 安全提醒

- 烙铁温度 300°C+ —— 不要碰烙铁头
- 焊接时戴口罩（松香烟气有害）
- 焊完洗手（焊锡含铅）
- 剪掉的引脚不要乱丢（会扎脚）

## ✏️ 今日作业

1. 收到 PCB 板后，先检查外观
2. 按 小元件 → IC → 插件的顺序焊接（先低后高）
3. 焊接完用放大镜检查所有焊点
4. 用万用表二极管档测一下电源和地之间有没有短路
5. 用洗板水/酒精擦干净板子

> **完成标志**：所有元件焊好，万用表确认电源和地没有短路，准备上电测试。
