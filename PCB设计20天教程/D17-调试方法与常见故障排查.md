# D17 — 调试方法与常见故障排查

> **学习目标**：学会系统的调试方法——上电前检查、上电后测量、常见故障的定位与修复。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 调试前的准备

### 工具

| 工具 | 用途 |
|------|------|
| 万用表 | **必备**——测电压、通断、电阻 |
| 示波器 | 可选——看波形，排查复杂问题 |
| USB 可调电源 | 推荐——带电流显示，能看到板子是否短路 |
| 杜邦线/飞线 | 临时连接、信号测量 |

## 📖 上电前的必检项目

**绝对不要不检查就上电——烧芯片就在一瞬间。**

### 目视检查

- [ ] 元件方向都对了？（IC 1 脚、LED 正负、电容极性）
- [ ] 没有连锡？（用放大镜或手机微距）
- [ ] 没有漏焊？
- [ ] 锡珠、线头等杂物没有落在板子上？

### 万用表检查

- [ ] 电源和地之间**不短路**（万用表蜂鸣档 → 测 VCC 和 GND → 不应导通）
- [ ] 如果测到短路→找原因（最常见的是焊盘连锡）
- [ ] 电源正负极之间电阻正常（几百 Ω 以上，不是 0Ω）

## 📖 上电调试流程

### 第一步：先用可调电源限流

**操作方法**：可调电源设定为 5V，**限流 100mA**（或 50mA），再上电。

为什么？
- 如果板子有短路，限流 100mA 不会烧东西，电源电压会被拉低
- 正常的 NE555 电路，空载电流大约几个 mA

### 第二步：测电压

| 测量点 | 应该测到 | 如果不对 |
|--------|---------|---------|
| VCC 对 GND | 5V（或你设计的电压） | 检查电源输入 |
| NE555 的 8 脚 | 5V | 供电没过来 |
| NE555 的 4 脚（复位） | >2V（高电平） | 悬空？需要上拉 |
| NE555 的 3 脚（输出） | 方波（示波器）/ LED 闪烁 | 测直流电压约 2.5V（平均值） |
| LED 两端 | VF ≈ 1.8-2.2V | LED 不亮时检查 |

### 第三步：观察现象

LED 应该以大约 1Hz（每秒一次）的频率闪烁。

如果不闪：
1. 测量 3 脚电压——如果一直 0V 或一直 5V，说明 NE555 没起振
2. 检查定时电阻 R1、R2 和定时电容 C2（6 脚、7 脚）
3. 检查复位脚（4 脚）——悬空会导致 NE555 不工作

## 📖 常见故障排查表

| 故障现象 | 可能原因 | 排查方法 |
|---------|---------|---------|
| **上电没反应，LED 不亮** | 供电问题 / 虚焊 | 测 VCC 对 GND 电压 |
| **LED 常亮不闪** | NE555 没起振 / 输出一直高 | 测 3 脚电压是否变化 |
| **LED 极暗** | 限流电阻太大 / VF 不匹配 | 算一下 R 对不对 |
| **通电瞬间 IC 发烫** | 电源接反 / 内部短路 | 立刻断电，检查 VCC/GND |
| **电源短路（万用表量通）** | 焊盘连锡 / 元件焊错 | 放大镜检查，吸锡带清理 |
| **LED 闪一下就不亮了** | 芯片锁死 / 过热保护 | 断电冷却后重试 |
| **板子工作不稳定** | 去耦电容缺失 / 电源噪声 | 加 100nF 去耦电容 |

## 💡 排查方法论

**不要随机试，要系统排查：**

```
现象（LED不亮）
  │
  ├─ 测 VCC=5V？  ─── 否 → 查电源输入
  │                    是 ↓
  ├─ 测 NE555 8脚=5V？ ─── 否 → 查 8 脚到 VCC 的走线
  │                      是 ↓
  ├─ 测 NE555 4脚>2V？ ─── 否 → 4 脚需要接 VCC 或上拉
  │                      是 ↓
  ├─ 测 NE555 3脚有方波？ ─── 否 → 查 R1/R2/C2
  │                        是 ↓
  └─ 测 LED 和 R3 是否正常？ ─── 否 → 焊错/损坏
                               是 → 怀疑原理图有错 → 回到 D5
```

## ✏️ 今日作业
