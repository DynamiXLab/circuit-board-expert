# D11 — 手动布线基础

> **学习目标**：掌握手动布线的顺序、核心规则和快捷键，开始在 PCB 上画出真正的走线。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 布线 = 把飞线变成铜导线

布局完成后，PCB 上有很多**飞线（Ratlines）**——白色的临时虚拟连线，表示"这两个焊盘需要连接"。**焊盘（Pad）**是 PCB 上用来焊接元件的金属接触区域，贴片焊盘是平面铜箔，直插焊盘是孔+环形铜箔。布线的任务就是用铜箔走线代替这些虚拟飞线，把焊盘连通布线的任务就是用铜箔走线代替这些虚拟的飞线。

### 布线顺序（优先级）

```
① 电源网络（VCC, 3.3V, +5V） → ② 时钟/高速信号 → ③ 模拟信号 → ④ 普通信号
   ↑ 走宽线                       ↑ 尽量短         ↑ 远离数字      ↑ 最灵活
```

**原则：先把最苛刻的线走了，再走灵活的。**

## 🔑 核心布线规则

| 规则 | 说明 |
|------|------|
| **不要走直角** | 使用 45° 角或圆弧拐弯——直角会形成天线 + 蚀刻时过蚀 |
| **走线尽量短** | 直接决定信号质量和抗干扰能力 |
| **过孔不要太多** | 一个信号最好只换 1-2 次层，每个过孔增加阻抗不连续 |
| **走线间距均匀** | 避免一段很宽、一段很窄的"瓶颈" |
| **走线不要分叉** | 一条线从起点到终点是单一连续路径，不要分叉成 Y 型 |
| **外层走线比内层宽** | 双面板顶层底层都是外层 |

## 🔑 布线快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+W` | 开始走线 |
| `Space` | 布线中切换拐角方向（45° / 直角 / 圆弧） |
| `~`（Tab 上方） | 布线中放置过孔换层 |
| `Shift+S` | 单层显示——打开它专心走一层 |
| `Ctrl+Shift+W` | 结束走线 |
| `L` | 布线中切换层（自动加过孔） |
| `Backspace` | 布线中撤销上一段 |

## 💡 电流与线宽经验公式

铜厚 1oz，温升 10°C：

| 电流 | 所需线宽 | 建议取整 |
|------|---------|---------|
| 0.5A | 约 10mil | ≥ 10mil |
| 1A | 约 12-15mil | ≥ 15mil |
| 2A | 约 25-30mil | ≥ 30mil |
| 3A | 约 50-60mil | ≥ 60mil |
| 5A | 约 100-120mil | ≥ 120mil |

> 嘉立创 EDA 内置工具：**"工具 → 导线/过孔载流"**，可以输入电流自动计算线宽。

## 📖 新手布线流程（针对 NE555 项目）

1. **先走 VCC 和 GND**：
   - VCC 走 ≥ 15mil 宽的线
   - GND 先不着急走线——后面用铺铜（D12）处理
2. **走输出信号**：
   - NE555 的 3 脚（输出）→ LED1
   - 走线宽度 8mil 即可（信号电流很小）
3. **走定时电路**：
   - R1、R2、C2 到 NE555 的 2/6/7 脚的连接
   - 走线尽量短，减少杂散电容
4. **走完所有信号后**，DRC 检查

### 小技巧

- 布线时打开网格吸附（`G` 切换），走线更整齐
- 复杂区域先用 `Shift+S` 单层显示，只看一层
- 如果走不通，**不要硬挤**——回到布局阶段微调元件位置

## ⚠️ 常见布线错误

- ❌ **直角走线** — 高频下会辐射干扰
- ❌ **过孔太多** — 一个信号穿 3-4 次层，阻抗变化大
- ❌ **走线太靠近板边** — 板边 < 0.3mm 的走线可能在分板时被切坏
- ❌ **焊盘上直接拐弯** — 走线应从焊盘的长边引出，再拐弯
