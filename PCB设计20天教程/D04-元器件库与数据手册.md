# D4 — 元器件库与数据手册

> **学习目标**：学会在嘉立创 EDA 中找到正确元器件，以及——更重要的是——学会读元器件数据手册（Datasheet）。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**，元件库数据以立创商城为准。

---

## 📖 如何找到正确的元器件

嘉立创 EDA 内置了**立创商城**的元件库，既有原理图符号，也有 PCB 封装，还关联了购买链接。

**搜索元器件的步骤：**

1. 找到顶部菜单栏-放置-器件/复用模块 → 搜索框输入型号（如 `STM32F103C8T6`）
2. 结果列表会显示：符号预览 + 封装名称 + 是否立创商城有库存
3. 双击放置到原理图

**搜索结果里很多同名元件，选哪个？**

| 优先级 | 选择标准 |
|--------|---------|
| ⭐ 首选 | 带"立创商城"标识的，封装已验证，可购买 |
| ✅ 可用 | 社区贡献的符号，核对引脚后可用 |
| ❌ 不要用 | 封装明显不对、引脚数不匹配的 |

> 💡 **自己建库 vs 用现成的**：初学阶段 90% 的元件直接用立创 EDA 库里的。只有特别冷门的芯片才需要自己画符号和封装。D14 会教你怎么建库。

## 📖 如何读数据手册（Datasheet）

数据手册是元器件的"使用说明书"。第一次看会觉得很厚（几百页），但 **不需要全读**。只需定位以下几个章节：

| # | 章节 | 找什么、含义是什么 | 重要性 |
|---|------|-------------------|--------|
| ① | **Pin Description（引脚定义）** | 每个引脚的名字、功能、类型（输入/输出/电源）。**拿到一个芯片首先要看的——知道哪个脚接电源、哪个脚接地、哪个脚是信号** | ⭐⭐⭐ |
| ② | **Typical Application（典型应用电路）** | 芯片厂商给出的参考电路图。**直接照着画——比你自己琢磨的靠谱 100 倍** | ⭐⭐⭐ |
| ③ | **Recommended Operating Conditions（推荐工作条件）** | 芯片正常工作时需要的电压、温度、时钟范围。**设计依据——超过这个范围芯片不保证正常工作** | ⭐⭐⭐ |
| ④ | **Electrical Characteristics（电气特性）** | Voh（输出高电平的最低值）、Vol（输出低电平的最高值）、Ioh/Iol（输出驱动电流）、功耗等。**判断芯片能不能驱动下级电路** | ⭐⭐ |
| ⑤ | **Package Information（封装尺寸）** | 芯片的物理尺寸图——焊盘间距（Pitch）、本体长宽、引脚宽度。**画封装时必须对照此图核对每个尺寸，差 0.1mm 元件可能焊不上** | ⭐⭐ |
| ⑥ | **Absolute Maximum Ratings（绝对最大额定值）** | 芯片能承受的极限值——**超过任何一个数字芯片立刻永久损坏**。设计时需留余量，一般用到 80% 以下 | ⭐⭐⭐ |


### 💡 实战：读一个 LED 的数据手册

打开某普通贴片 LED 的数据手册，找到这些参数：

| 参数 | 典型值 | 为什么重要 |
|------|--------|-----------|
| 正向压降 Vf（红色） | ≈ 1.8–2.2V | 算限流电阻要用 |
| 正向压降 Vf（蓝色/白色） | ≈ 3.0–3.4V | 比红色高很多 |
| 最大正向电流 If | 20mA（直插）/ 5mA（0603） | 超过就烧 |
| 亮度（mcd） | 几十到几百 | 亮度够不够 |

**限流电阻计算（必须会）：**

```
R = (Vcc - Vf) / If
```

例：5V 供电，红色 LED（Vf=2V），目标电流 2mA 时：
```
R = (5 - 2) / 0.002 = 1500Ω → 选 1.5kΩ（E24 标准值）
```

## ⚠️ 常见坑

- **Vf 不是固定值**：不同颜色 LED 的 Vf 差很多，不看手册直接硬算会翻车
- **电阻选 E24 系列**：计算值 1.5kΩ 是标准值，但 1.37kΩ 不是——选最接近的标准值
- **功率要算**：`P = I²R`，如果电流大，0805（1/8W）可能不够

## ✏️ 今日作业

1. 下载 `LM358`（通用运放）的数据手册（PDF），按上面的表格定位 7 个关键章节，截图或做笔记
2. 在嘉立创 EDA 中搜索 `LM358`，放置到原理图，对照数据手册的引脚图逐个核对引脚编号和功能
3. 计算：5V 供电，蓝色 LED（Vf=3.2V），目标亮度电流 3mA，限流电阻用多大？

> **完成标志**：你能打开一个没见过的芯片的数据手册，10 分钟内找到引脚定义、工作电压、参考电路。
