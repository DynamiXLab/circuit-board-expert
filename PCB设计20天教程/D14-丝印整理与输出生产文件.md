# D14 — 丝印整理与输出生产文件

> **学习目标**：整理丝印层使其清晰规范，导出 Gerber 和钻孔文件并用在线预览验证。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 丝印（Silkscreen）规范

**丝印（Silkscreen）** 是印刷在 PCB 表面的字符标识（通常为白色），用来标注元件位号（R1、C1）、极性标记（二极管 `▶`、电解电容 `+`）、版本号、测试点名称等。丝印层不是铜箔，**没有电气功能**，但它是焊接和维修时的重要视觉参考。

丝印层（Top Overlay / Bottom Overlay）是 PCB 上的白色字符层。

### 丝印规范

| 要求 | 说明 |
|------|------|
| 字号 ≥ 0.8mm（32mil） | 太小工厂印不清楚 |
| 线宽 ≥ 0.15mm（6mil） | 建议 6-8mil |
| **不要压在焊盘上** | 丝印上焊盘 = 文字被阻焊覆盖 = 看不见 |
| 标明极性 | 二极管用 `|▶|` 或 `+`，电容用 `+` |
| 标注版本号 | 板子一角写上 `V1.0`, `2025-04` |
| 标注测试点 | 如 `TP1`, `GND`, `3V3` |
| 不要重叠 | 两个元件的位号不要压在一起 |

### 如何调整丝印

1. 在 PCB 模式下，切换到 **Top Overlay** 层
2. 选中位号文字（如 `R1`、`C1`）
3. 拖拽到元件旁边的空白区域
4. 选中文字 → 右侧属性面板可调字号、线宽、旋转

> 💡 **技巧**：在"设计管理器"中全选所有位号，一次性缩小字号，再逐个微调位置。

## 📖 导出 Gerber 文件

Gerber 是 PCB 工厂的标准格式——EDA 软件画出来的设计，要转成 Gerber 工厂才能生产。

### 嘉立创 EDA 导出步骤

1. **文件 → 制造 → Gerber Files**
2. 在弹出的对话框中，选择需要导出的层（一般全选）：
   - Top Layer（顶层铜）
   - Bottom Layer（底层铜）
   - Top Solder（顶层阻焊）
   - Bottom Solder（底层阻焊）
   - Top Overlay（顶层丝印）
   - Board Outline（板框）
3. 设置输出目录
4. **文件 → 制造 → 钻孔文件**（NC Drill）
5. 把 Gerber + Drill 文件打成一个 ZIP 包

### 在线预览 Gerber

嘉立创 EDA 内置了 Gerber 预览功能：

**文件 → 制造 → 在线预览 Gerber**

检查每一层：
- [ ] 顶层铜：走线都在吗？电源线够宽吗？
- [ ] 底层铜：铺铜完整吗？
- [ ] 阻焊层：焊盘开窗对吗？有没有意外的开窗？
- [ ] 丝印层：位号清晰吗？有没有压在焊盘上？
- [ ] 钻孔文件：孔位和数量对吗？

## 📖 建立自己的元件库（扩展知识）

如果你发现某个元件在嘉立创库里找不到，或者对现有封装不放心，可以：

1. **原理图符号**：菜单"放置 → 绘制 → 符号向导"
2. **PCB 封装**：菜单"放置 → 封装向导"
3. 根据数据手册的 Package Information 部分填入尺寸参数
4. 保存到个人库，下次直接用

**关键参数要核对**：
- 焊盘间距（Pitch）
- 焊盘宽度和长度
- 本体尺寸
- 热焊盘尺寸（如果有）
