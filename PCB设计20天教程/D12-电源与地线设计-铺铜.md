# D12 — 电源与地线设计、铺铜

> **学习目标**：掌握 PCB 上电源和地线的设计方法，学会铺铜（Copper Pour）来形成完整的地平面。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 电源走线设计

电源是 PCB 的"血管"——电源走线不好，整个电路工作不稳定。

### 树形电源拓扑

```
电源输入 (5V DC)
   │
   ├──→ 走宽线到达 NE555 的 8 脚 (VCC)
   │        │
   │        ├──→ C1 (100nF) 去耦电容 (紧贴 8 脚)
   │        ├──→ C2 (10μF)  储能电容
   │        └──→ R1 上拉电阻
   │
   └──→ 连接到 LED 输出？不需要——LED 通过 NE555 内部供电
```

### 电源走线规范

| 信号类型 | 线宽 | 说明 |
|---------|------|------|
| 主干电源（VCC） | ≥ 15mil | 载流足够，IR Drop 小 |
| 支路电源（到单个 IC） | ≥ 10mil | 够用 |
| 地线（GND） | **尽量宽** | 后面用铺铜解决 |

### ⚠️ 电源设计的常见陷阱

- **IR Drop**：细长走线的远端电压会降低。5V 的 VCC，经过 10cm 长的 10mil 走线带 500mA 负载，远端可能只剩 4.7V
- **回路面积**：电源和地回路面积太大 → 电磁干扰（EMI）严重
- **环路**：避免形成大电源环路（天线效应）

## 📖 铺铜（Copper Pour）

**铺铜（Copper Pour）** = 用大面积的铜皮覆盖 PCB 上的空白区域，通常连接 GND 网络。铺铜不是"画出铜皮"而是"声明一块区域属于 GND"——EDA 会自动在这块区域内填充铜箔网格。

### 铺铜的好处

1. **低阻抗地回路** — 信号的回流路径最小
2. **散热** — 铜皮帮助热量扩散
3. **减少 EMI** — 完整的地平面屏蔽电磁干扰
4. **节约蚀刻药水** — 工厂角度（大面积铜比细线好生产）

### 嘉立创 EDA 铺铜操作

1. 菜单：**放置 → 铺铜**
2. 点击鼠标画出铺铜边界（一般沿着板框内缩 0.5mm）
3. 在右侧属性中：
   - **网络（Net）**：选择 `GND`
   - **删除孤立铜皮**：✅ 开启
   - **连接方式**：十字连接（Thermal Relief）

### 十字连接（Thermal Relief） vs 直连

| 连接方式 | 适用场景 |
|---------|---------|
| **十字连接** | 普通焊盘、通孔——焊接时热量不会快速散失 |
| **全连接（直连）** | 大电流焊盘、散热焊盘 |

> 如果是手工焊接，十字连接会让你好焊很多——直连的话烙铁温度被大片铜皮吸走，焊半天化不了锡。

## 📖 地线设计：先走线再铺铜

**新手容易犯的错误**：先铺铜，然后发现 GND 引脚没连上铺铜就跑去手动连线。

**正确顺序**：

1. 先走完所有信号线和电源线
2. GND 网络暂时不处理
3. **最后** 铺 GND 铜皮
4. 所有 GND 焊盘会自动通过铜皮连接

## ⚠️ 铺铜注意事项

- **板边缩进 0.5mm**：铜皮不要铺到板子边缘，否则分板时会露出铜边
- **孤立铜皮（孤岛）必须删**：被走线包围形成的"孤岛"铜皮没有任何电气连接，它没有实际功能反而会成为天线辐射噪声。EDA 中的"删除孤立铜皮"选项会自动处理
- **铺铜后运行 DRC**：检查有没有意外的短路

## ✏️ 今日作业

1. 在 D11 的 PCB 上，先走完 VCC 电源主干（≥15mil 宽）
2. 顶层铺 GND 铜皮：
   - 沿着板框内缩 0.5mm 画铺铜边界
   - 网络选 GND
   - 十字连接
3. 底层也铺 GND 铜皮
4. 检查有没有孤立铜皮，有的话删除或通过过孔连接
5. 截图保存铺铜前后的对比
