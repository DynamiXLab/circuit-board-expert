# D13 — 布线收尾与设计规则检查（DRC）

> **学习目标**：完成所有布线，运行 DRC 修复全部错误，拿到一张零错误的 PCB。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 完成所有布线

先对照检查清单：

- [ ] 所有飞线（Ratlines）都消失了（或只剩 GND——GND 靠铺铜连接）
- [ ] 电源走线宽度符合载流要求
- [ ] 底层也铺了 GND 铜皮
- [ ] 每个 IC 旁都有去耦电容（已经在原理图中画了的）
- [ ] 走线和焊盘之间没有意外的间距过小

### 如果还剩飞线走不通怎么办？

1. **微调布局**：把两个元件稍微拉开一点，创造走线通道
2. **换层走**：顶层走不通就通过过孔换到底层
3. **加粗**：确保电源线够宽
4. **补过孔**：GND 过孔多打几个（并联降低阻抗）

## 📖 DRC（设计规则检查）

点击 **"工具 → DRC/ERC 检查"**（在 PCB 模式下，v3.2.131 的 DRC 入口在工具菜单下）。

### DRC 能查出的问题

| 问题 | 含义 | 怎么修复 |
|------|------|---------|
| **Clearance Violation** | 两个导体间距不足 | 拉开或者改规则 |
| **Width Violation** | 走线宽度小于最小线宽 | 加粗走线 |
| **Short Circuit** | 不同网络意外短接 | 找到短接点，删除/重画 |
| **Unconnected Pin** | 元件引脚没接任何网络 | 检查是不是漏接了 |
| **Un-Routed Net** | 还有没走完的飞线 | 完成布线 |
| **Solder Mask Bridge** | 两个焊盘之间阻焊桥不够宽 | 增大焊盘间距 |

### 自定义设计规则

**设计 → 设计规则**（v3.2.131 的设计规则在"设计"菜单下）：

| 参数 | 建议值 | 备注 |
|------|--------|------|
| 最小线宽 | 6mil | 留余量 |
| 最小线距 | 6mil | 留余量 |
| 最小过孔内径 | 0.3mm | 嘉立创要求 ≥ 0.2mm |
| 最小过孔外径 | 0.6mm | 嘉立创要求 ≥ 0.45mm |
| 铜皮到走线距离 | 0.2mm | 太近会短路 |
| 铜皮到板边距离 | 0.5mm | 制造工艺需要 |

## 💡 什么是"假报错"？

有时候 DRC 会报一个你**故意**这样做的错误。

例如：两个 GND 网络（虽然都叫 GND，但可能用了不同的符号）通过铺铜连在一起了。DRC 会报 "Short Circuit"。

**处理方式**：
- 先确认这确实是你想要的效果
- 右键 → 忽略此规则（DRC Exclude）
- **但一定要在旁边加注释**，不然下次你自己看都忘了
