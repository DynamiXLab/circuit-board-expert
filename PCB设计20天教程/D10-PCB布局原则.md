# D10 — PCB 布局原则

> **学习目标**：掌握 PCB 布局的核心原则——布局决定了布线的难度和板子的最终性能。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**。

---

## 📖 布局——最体现功力的环节

"布局 70%，布线 30%"。好的布局让布线变得轻松，差的布局让你怎么走都走不通。

### 先做一件重要的事：冻结原理图

**开始布局前，确保原理图不会再改。** 如果布局了一半又改原理图，之前的布局就白费了。

## 🔑 十大布局原则

| # | 原则 | 说白了就是 |
|---|------|-----------|
| 1 | **先大后小** | 先放连接器、大芯片、电源模块，再塞小电阻电容 |
| 2 | **模块化布局** | 同一个功能模块的元件聚在一起，不要打散 |
| 3 | **电源优先** | 电源电路先占好位置（靠近板边或输入连接器） |
| 4 | **去耦电容就近** | **去耦电容（Decoupling Capacitor）**：每个 IC 在工作时会有瞬态电流需求，去耦电容充当"蓄水池"就近提供电流，防止电压跌落。典型值 100nF，必须贴在 IC 电源引脚旁边 ≤ 5mm，否则失去作用 |
| 5 | **信号流向** | 布局遵循信号从左到右/从上到下的方向 |
| 6 | **模拟/数字分离** | 高速数字电路远离模拟信号（音频、ADC、传感器） |
| 7 | **发热元件分散** | 大功率管、稳压 IC 不要堆在一起 |
| 8 | **接插件靠边** | USB、排针、DC 座放在板边 |
| 9 | **晶振紧贴 MCU** | 晶振走线要短、两边等长、不加过孔 |
| 10 | **预留调整空间** | 元件之间不要太挤，留出走线通道（建议 ≥ 0.3mm） |

## 📖 实战示范：NE555 PCB 布局

```
┌──────────────────────────────────────────────────┐
│                                          ○ 定位孔 │
│    ┌──────┐                                     │
│    │  C1  │  ┌──────────┐      ┌──┐              │
│    │100nF│  │          │      │R3│              │
│    └──────┘  │  NE555   │      │1k│              │
│              │          │      └──┘              │
│    ┌──────┐  │  DIP-8   │           ┌──┐        │
│    │  C2  │  │          │           │LED│        │
│    │10μF │  └──────────┘           └──┘        │
│    └──────┘                                     │
│              ┌──┐  ┌──┐                         │
│              │R1│  │R2│                         │
│              │10k│  │10k│                        │
│              └──┘  └──┘                         │
│                                          ○ 定位孔 │
└──────────────────────────────────────────────────┘
```

### 操作步骤

1. **先放 IC** — 把 NE555 放在板子中央偏上位置
2. **再放连接器** — 电源输入排针放在板边
3. **放去耦电容** — C1(100nF) 紧贴 NE555 的电源脚（8 脚）
4. **放定时元件** — R1、R2、C2 靠近 NE555 的 6/7 脚
5. **放输出元件** — R3 和 LED 靠近 NE555 的 3 脚（输出）
6. **调位号** — 把 R1、R2 等丝印文字移到不遮挡走线的位置

## ⚠️ 常见布局错误

- ❌ 去耦电容离 IC 太远（> 5mm 效果大打折扣）
- ❌ 所有元件贴在一起没留走线通道
- ❌ 晶振远离 MCU 或用过孔绕远路
- ❌ 电源模块放在板子正中央（维修麻烦，散热差）
