# D1 — PCB 是什么？从原理图到电路板的全流程

> **学习目标**：理解 PCB 的完整制造流程，建立从"电路想法"到"实物板子"的全景认知。
>
> 📌 本教程基于 **嘉立创 EDA 专业版 v3.2.131**，界面以实际版本为准。

---

## 📖 阅读与理解

### 什么是 PCB？

**PCB（Printed Circuit Board，印制电路板）** 是电子元器件的物理载体。它通过铜箔走线代替导线，把元器件焊接在焊盘上，构成完整的电路。

简单说：**原理图画的是"逻辑怎么连"，PCB 画的是"实物怎么摆"。**

### 💡 PCB 常识：单片机与 PCB 的关系

你可能听说过 Arduino、STM32、ESP32 这些词——它们叫**单片机（Microcontroller Unit, MCU）**，是一颗集成了 CPU、内存、IO 口的芯片，是电子产品的"大脑"。

单片机不能单独工作——它需要：
- **电源电路**（把电池/USB 的电压转为芯片需要的 3.3V 或 5V）
- **晶振电路**（提供时钟节拍）
- **去耦电容**（滤除电源噪声，让芯片稳定运行）
- **外围电路**（传感器、电机驱动、LED 显示……）

**这些电路全部画在 PCB 上**。所以学 PCB 设计 = 学会怎么给单片机搭建一个"家"。后续 D18-D20 的实战项目就是做一块 USB 转串口工具，用来给单片机下载程序和调试。

### PCB 的完整制造流程

一块 PCB 从设计到成品，要经过以下环节：

```
需求分析 → 原理图设计 → PCB 布局布线 → 导出 Gerber → 工厂制板 → 焊接 → 调试
                                                         ↑
                                                  （嘉立创打样）
```

| 环节 | 你在做什么 | 用什么工具 |
|------|-----------|-----------|
| 需求分析 | 明确功能、选型芯片 | 脑子 + 数据手册 |
| 原理图设计 | 画电路连接图 | 嘉立创 EDA 专业版 |
| PCB 布局布线 | 摆放元件 + 画铜线 | 嘉立创 EDA 专业版 |
| 导出 Gerber | 生成工厂需要的生产文件 | 嘉立创 EDA 专业版 |
| 工厂制板 | 嘉立创帮你做出来 | 嘉立创 PCB 打样服务 |
| 焊接 | 把元器件焊上去 | 烙铁 + 焊锡 |
| 调试 | 上电测试，排查问题 | 万用表 + 示波器 |

## 🔑 关键概念

| 术语 | 说明 |
|------|------|
| **原理图（Schematic）** | 逻辑连接图，表示"谁和谁连在一起"，不关心物理位置 |
| **网表（Netlist）** | 原理图到 PCB 的"桥梁"，描述所有连接关系 |
| **封装（Footprint）** | 元器件的物理占位——包含了焊盘位置、引脚间距、外形尺寸、丝印轮廓。**封装错了焊盘对不上，元件就焊不上去，整块板子报废** |
| **焊盘（Pad）** | PCB 上用来焊接元件的金属接触点。贴片焊盘是表面的一块铜，直插焊盘是孔+环。焊盘尺寸必须和元件引脚匹配 |
| **丝印层（Silkscreen / Overlay）** | PCB 表面的白色字符层，用来印元件位号（R1、C1）、极性标记（`+`、`▶`）、版本号。**丝印是生产时的印刷标识，不是铜箔，没有电气连接功能** |
| **阻焊层（Solder Mask）** | PCB 表面的绿色（或其它颜色）油墨层，覆盖不需要焊接的铜箔。焊盘处开窗露出铜，便于焊接。**阻焊的作用是防止焊锡意外桥接、防止铜氧化** |
| **过孔（Via）** | 从顶层贯穿到底层的导电孔，孔壁镀铜，用于连接不同层的走线。分为通孔（贯穿全板）和盲孔/埋孔（只通部分层，双面板不用） |
| **层（Layer）** | PCB 的不同功能面——顶层铜（Top Layer）、底层铜（Bottom Layer）、丝印层（白色字符）、阻焊层（绿色油墨）、板框层（外形）等 |
| **Gerber 文件** | PCB 工厂生产用的标准格式文件。每个层（铜、阻焊、丝印、钻孔……）对应一个 Gerber 文件，工厂根据这些文件制造板子 |
| **DRC** | 设计规则检查——自动检查 PCB 上的走线/过孔/间距有没有违反制造约束 |
| **ERC** | 电气规则检查——自动检查原理图有没有逻辑错误（悬空引脚、短路等） |

## ⚠️ 常见坑

- **不要把原理图和 PCB layout 混在一起画**：先完成原理图，再导入 PCB，不要跳步。
- **不要跳过数据手册**：选芯片必须看数据手册，不能只看型号就画。
- **封装错了板子就废了**：焊盘间距差 0.1mm 就可能焊不上。
