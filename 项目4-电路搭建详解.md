# 项目4 · 电路搭建详解 — 为什么要这样连线

> **目标**：不是机械地照着画，而是理解每一条连线"为什么这样接"。以后你画任何电源电路都能举一反三。

---

## 一、662K（LDO部分）—— 为什么只需要两个电容？

### 完整电路

```
               ┌──────────┐
   5V (来自Buck) ───┤③ Vin    │
                    │         │
   C1 ──┤├──┬──────┤① GND    │
   1μF     │      │         │
           │      │② Vout ──┼──┬── C2 ──┤├── GND
          GND     └──────────┘  │   1μF
                                │
                           Vout (3.3V)
```

### 连线解释

**① 为什么Vin要接一个1μF电容到GND（C1）？**

```
没有C1会怎样？
                 ┌─ 负载突然需要更多电流
5V ─────────── 662K ── Vout
                │         ↓
                │     Vout瞬间跌落 ⚠️
               GND
```

C1像是一个**小水库**——当负载（传感器）突然需要更多电流时，C1里的电荷可以**瞬间补充**，防止Vin抖动 → Vout跟着抖。

> 为什么不用更大的电容？662K内部稳压能力好，1μF就够。而且传感器总共不到100mA，不会突然抽走大电流。

**② 为什么Vout也要接一个1μF电容到GND（C2）？**

C2的作用是**滤波**——把662K输出中的微小噪声滤掉，让3.3V更干净。

> 你可以把C2想象成一个**小水池**——水流（电流）里的波纹到了水池里就被抚平了。

**③ 为什么GND要大面积铺铜？**

任何电路都需要一个**稳定的0V参考点**。如果GND走线太细或太长，不同位置之间的GND电位会有差异（这叫"地弹"）——参考点不稳定，输出电压也会不稳。

**一句话总结662K的连线逻辑**：
> 输入电容防抖 → 芯片稳压 → 输出电容滤波 → 大面积GND保证参考点稳定

## 二、MP1584EN（Buck部分）—— 每一条线都是精心设计的

### 完整电路

```
                    ┌──────────────────────────────────┐
                    │          MP1584EN                │
                    │                                  │
  Vin (7~12V) ──┬──┤⑦ IN                   ⑥ SW├───┬── L1(10~15μH) ──┬── Vout (5V)
                │   │                                  │                │
           C1 ──┤├──┤                                  │                │
         22μF   │   │                                  │           C3 ──┤├──
                │   │                        ┌─── D1 ──┤          22μF   │
           C2 ──┤├──┤                        │  (SS34) │                │
         22μF   │   │                        │         │                ├── C4 ──┤├──
                │   │                       GND       GND                │  22μF   │
                │   │                                                    │         │
                │   │② EN ── R3(100k) ── Vin                            │        GND
                │   │                                                    │
                │   │③ COMP ─┬── R4(10k) ─┬── C7(3.3nF) ── GND         │
                │   │         │             │                             │
                │   │         └─────────────┘                             │
                │   │                                                    │
                │   │④ FB ──┬── R1(52.3kΩ) ── Vout                      │
                │   │       │                                            │
                │   │       └── R2(10kΩ) ── GND                          │
                │   │                                                    │
                │   │⑧ SS ── C5(0.1μF) ── GND                           │
                │   │                                                    │
                │   │① BST ── Cbst(0.1μF) ──┬──⑥ SW（自举，必须接！）    │
                │   │                       │                            │
                │   │                       └── 去 L1 + D1               │
                │   │                                                    │
                │   │⑤ GND ── 大面积GND铺铜                              │
                │   └────────────────────────────────────────────────────┘
                │
               GND
```

### 连线分组解释

#### 组1：输入回路（⑦IN + ①BST → Cbst → ⑥SW + C1/C2）

**为什么C1、C2要两个22μF并联？**

```
一个22μF
  ┌──────┐
  │ 22μF │ ← ESR = 5mΩ
  │ ESR  │
  └──────┘
  
两个22μF并联 → 容量=44μF，ESR=2.5mΩ
```

两个并联的好处：
1. **容量翻倍** = 存更多电，输入更稳定
2. **ESR减半** = 电容本身的内阻变小，高频响应更好

> 你把电容想象成**水桶+水管**：容量是水桶大小（存多少电），ESR是水管粗细（电流进出快慢）。两个并联 = 两个水桶同时工作，装水多、进出快。

**为什么BST要接0.1μF到SW？**

这是MP1584最容易被忽略但最关键的一条连线。

MP1584内部的高端MOS管（负责导通Vin→SW的那个开关）需要**比Vin更高的电压**才能完全导通。BST引脚和Cbst电容的作用就是"把SW上的电压抬起来"：

```
① 当MOS导通时：Cbst充电，BST端电压 = Vin - VSW ≈ 5V
② 当MOS关断时：SW电压降到~0V，Cbst"踩"在SW上
③ 再导通时：BST电压 = SW电压 + Cbst电压，比Vin高几伏
④ 高出的电压用来驱动MOS→MOS完全导通→效率高
```

> **通俗理解**：你推一扇很重的门（MOS管），直接推推不动。你先后退两步（MOS关断），再冲两步跳起来（BST电容蓄力），用下落的势能去撞门——劲儿就更大了。

**如果不接Cbst**：MOS管半导通，内阻很大→发热量巨大→要么效率极低、要么烧芯片。

#### 组2：功率回路（⑥SW → L1 + D1 → C3/C4）

**为什么SW要接电感到输出？**

电感是Buck的**核心储能元件**。开关管导通时电感储能，断开时电感释放能量——这样即使输入是脉冲，输出也是连续的。

```
SW上的电压波形（示波器看）：
Vin ──┬──┬──┬──┬──┬──  ← 开关管导通时=Vin，关断时≈0V
      │  │  │  │  │
0V ──┴──┴──┴──┴──┴──
          ↓ 经过电感+电容滤波后 ↓
5V ─────────────────────  ← Vout平稳的5V
```

**为什么续流二极管SS34的阴极接SW、阳极接GND？**

这涉及到电流的"回路"：
- **开关管导通时**：电流从Vin → 开关管 → 电感 → Vout。此时SW电压≈Vin，二极管反偏（不导通）
- **开关管关断时**：电感的"惯性"让电流继续流，但需要一条回路。电流路径：GND → 二极管（正向导通）→ 电感 → Vout

```
关断时的电流路径：
                  ┌──────────┐
Vin ── [开关断开] ──┤          │
                  │  电感 L  │──→ Vout
         ↑        │          │
     从GND来      │          │
      ↑          └──────────┘
      │
  续流二极管 ← 电流从这里走！
  (正向导通)
```

**为什么SS34选肖特基二极管？**

普通二极管（如1N4007）导通时压降约0.7~1V，而肖特基二极管（SS34）只有0.3~0.5V。别小看这0.4V——在3A大电流下，损耗差了 **0.4V×3A=1.2W**！

#### 组3：反馈回路（④FB → R1/R2 → Vout）

**为什么FB要接两个电阻分压？**

MP1584内部有一个**0.8V的基准电压**。FB引脚的任务就是"把Vout的分压值拉到0.8V"。

```
Vout ───┬── R1(52.3kΩ) ──┬── FB → 内部比较器 → 调整占空比
        │                │
        └── R2(10kΩ) ───┤
                        │
                       GND
```

分压公式：
```
FB电压 = Vout × R2 / (R1 + R2) = 0.8V
→ Vout = 0.8 × (1 + R1/R2)
→ Vout = 0.8 × (1 + 52.3/10) = 0.8 × 6.23 ≈ 5V
```

**反馈的工作过程（闭环控制）：**
```
  ① FB测到电压 > 0.8V → Vout太高了！→ 减小占空比 → Vout降下来
  ② FB测到电压 < 0.8V → Vout太低了！→ 增大占空比 → Vout升上去
  ③ FB测到电压 = 0.8V → 刚刚好！→ 保持当前占空比
  
  ↗ 再次测量 → 回到① → 永远在调整，永远在稳定
```

> 🔄 这就是为什么我们说**Buck也是一个闭环系统**——它通过FB引脚"睁着眼睛"工作。

#### 组4：辅助引脚（②EN + ③COMP + ⑧SS）

**为什么EN要接100kΩ到Vin？**

EN引脚有一个内部弱下拉，如果不接外部上拉电阻，EN默认是低电平→芯片不工作。R3(100kΩ)把EN拉高到Vin→芯片一直处于使能状态。

**为什么COMP要接RC补偿网络？**

电源反馈环路如果不加补偿，可能会**振荡**——输出电压像波浪一样忽高忽低。R4+C7组成的补偿网络让环路"冷静"下来，稳定工作。

> 你不用深究补偿原理，知道"不加可能会振荡"就行。具体值照着数据手册参考电路选。

**为什么SS要接0.1μF到GND？**

SS = Soft Start（软启动）。如果一上电就让芯片满负荷工作，输入电容会瞬间被抽走大量电流——可能把输入电压拉低或产生尖峰。

C5让输出电压**慢慢升起来**，避免冲击：
```
无软启动：  5V ────────┤  ← 瞬间冲击
有软启动：  5V ────╱    │  ← 缓慢爬升，约1ms
              0 ─╱─────
```

## 三、两级电路的连接关系

### 3.1 整体电路图

```
 锂电池(7.4V)
      │
      ▼
 ┌──────────────┐
 │  MP1584 Buck │ → 输出5V / 2A → 给舵机+MCU供电
 │  输入电容    │
 │  Cbst自举    │
 │  电感L1      │
 │  续流D1      │
 │  输出电容    │
 └──────────────┘
      │
      │ 5V
      ▼
 ┌──────────────┐
 │  662K LDO    │ → 输出3.3V / 250mA → 给传感器供电
 │  输入电容C1  │
 │  输出电容C2  │
 └──────────────┘
      │
      │ 3.3V
      ▼
  灰度传感器、蓝牙等
```

### 3.2 为什么这样连接？

| 连接方式 | 为什么这样接 | 可以反过来吗？ |
|---------|-------------|--------------|
| 电池 → Buck → 5V → LDO → 3.3V | Buck高效降大压差，LDO干净出小电压 | ❌ 反过来(LDO先降→Buck再降)效率低、LDO会烧 |
| Buck输出电容 → 662K输入 | 662K输入最高6V，Buck输出5V正好 | ❌ 662K不能直接接电池(7.4V>6V) |
| 两个芯片共地(GND连一起) | 电路需要一个共同的0V参考点 | ❌ 不共地，信号无法传递 |

## 四、常见连线疑问QA

| 问题 | 回答 |
|------|------|
| **电感值大一点小一点有什么区别？** | 电感大→纹波小→响应慢；电感小→纹波大→响应快。10~15μH是5V输出的"甜点区" |
| **电容可以用电解电容代替MLCC吗？** | 输入电容可以用电解（便宜），但输出电容最好用MLCC（低ESR）。不要混用不同类型 |
| **为什么R1/R2必须用1%精度？** | 反馈精度直接决定输出电压精度。5%精度的电阻可能导致输出变成4.75V或5.25V |
| **Cbst的0.1μF可以用1μF代替吗？** | **不行！** 数据手册指定0.1μF。太大或太小都会影响自举效果 |
| **SS34可以用普通二极管代替吗？** | 可以，但效率会降低。普通二极管压降0.7V vs 肖特基0.3V——3A时差1.2W散热 |
| **COMP的10kΩ+3.3nF能随便改吗？** | 不建议。补偿网络影响环路稳定性，改错了输出可能振荡 |

---

> **总结**：电源电路的每一条连线都不是随便接的——它们背后是"能量怎么存储""噪声怎么滤除""反馈怎么闭环"等物理原理。理解了这个，以后你看到任何电源IC的数据手册，自己就能画出正确的电路。
