# 电路原理图专题 — 闭环控制部分

> **学习目标**：理解闭环控制的核心原理（反馈），掌握常见闭环控制电路的设计方法，能区分开环与闭环的适用场景。

---

## 一、什么是闭环控制？

**闭环控制（Closed-Loop Control）**：系统的输出被测量并反馈到输入端，与目标值比较，根据误差调整控制信号。

**核心三要素：测量 → 比较 → 修正**

```
        ┌─── 目标值 ──→ [比较器] ←── 反馈 ─────┐
        │                   │                    │
        │              [控制器]                   │
        │                   │                    │
        │              [执行器]                    │
        │                   │                    │
        │              [被控对象] ──→ 输出       │
        │                                          │
        └──────────────────────────────────────────┘
                         ↑
                    [传感器]
```

### 生活中的例子

| 场景 | 传感器 | 控制器 | 执行器 |
|------|--------|--------|--------|
| 空调恒温 | 温度传感器 | 比较实际与设定温度 | 压缩机启停 |
| 电动车定速巡航 | 速度传感器 | PID 计算 | 电机驱动 |
| 平衡车 | 陀螺仪+加速度计 | 姿态算法 | 电机补偿 |
| 恒流 LED 驱动 | 采样电阻测电流 | 运放/比较器 | MOS 管调节 |

## 二、常见闭环控制电路

### 2.1 电压反馈稳压（最简单的闭环）

LDO 和 DC-DC 本身就是闭环系统。

```
  ┌────────────────────────────────────────────────┐
  │                 LDO 内部结构                     │
  │                                                 │
  │  Vin ──┬── [调整管] ──┬── Vout                 │
  │        │              │                        │
  │        │         [分压电阻] ← 反馈采样          │
  │        │              │                        │
  │        │  [比较器] ←──┘                        │
  │        │      ↑                                │
  │        │  [基准电压]   ← 目标值（1.25V）        │
  │        │                                       │
  │        └── GND ─────────── GND                 │
  └────────────────────────────────────────────────┘
  
  比较器比较 Vout 的采样值和基准电压：
  Vout > 目标 → 调大调整管阻抗 → Vout 下降
  Vout < 目标 → 调小调整管阻抗 → Vout 上升
  → 最终 Vout 稳定在目标值
```

### 2.2 比较器闭环控制（最简单的硬件反馈）

LM393/LM358 等比较器可以直接搭建闭环。

**恒温控制电路：**

```
  ┌─────────────────────────────────────────────┐
  │  LM393 比较器                               │
  │                                             │
  │  VCC ──┬── R1(10k) ──┬── 3 (同相输入)        │
  │        │             │                      │
  │        │        NTC 热敏电阻                 │
  │        │         ↓ 温度 ↑→电阻↓              │
  │        └─────────────┴── GND                │
  │                                             │
  │  VCC ──┬── R2(10k) ──┬── 2 (反相输入)        │
  │        │             │          设定温度      │
  │        │      电位器(10k)                    │
  │        │             │                      │
  │        └─────────────┴── GND                │
  │                                             │
  │             1 (输出) ──┬── R3(1k) ── 加热器  │
  │                       │        继电器/MOS    │
  │                       └── GND               │
  │                                             │
  │  工作原理：                                  │
  │  温度 < 设定 → NTC 电阻大 → V3 > V2 → 输出高→加热     │
  │  温度 > 设定 → NTC 电阻小 → V3 < V2 → 输出低→停止加热  │
  └─────────────────────────────────────────────┘
```

**这是典型的** **滞回比较器（施密特触发器）**——需要加正反馈电阻防止输出在设定点附近抖动：

```
  输出（1脚）── R4(1MΩ) ──┬── 同相输入（3脚）
                          │（正反馈，产生滞回区间）
```

### 2.3 运放闭环控制（PID 雏形）

运放可以做比例（P）控制：

```
  ┌─────────────────────────────────────────┐
  │  比例控制运放电路                        │
  │                                         │
  │  Rin                    Rf               │
  │  ┌──[10k]──┬──[100k]──┐                │
  │  │         │           │                │
  │  输入──────┤ 2         │                │
  │            │    LM358  │               │
  │  基准──────┤ 3         │──→ 输出        │
  │            │     1     │                │
  │            └───────────┘               │
  │                                         │
  │  Vout = (Rf/Rin) × (基准 - 输入)        │
  │        比例增益 Kp = Rf/Rin = 10         │
  └─────────────────────────────────────────┘
```

### 2.4 恒流 LED 驱动（采样电阻闭环）

```
  ┌─────────────────────────────────────────┐
  │  恒流驱动 LED                            │
  │                                         │
  │  VCC ──┬── LED ──┬── D (MOS 管漏极)     │
  │        │         │                      │
  │        │         │  N-MOS (AO3400)      │
  │        │         │                      │
  │        │         S ──┬── Rsense(10Ω) ── GND
  │        │              │                  │
  │        │         ┌────┘                  │
  │        │         │                       │
  │        │   LM358 │                       │
  │        │         │                       │
  │  基准 ──┤ 3 (+)  │  ← 设定电流：Vref     │
  │        │         │                      │
  │  Rsense─┤ 2 (-)  │  ← 实际电流：I×Rsense │
  │        │         │                      │
  │        │   1(输出)┴── G ──（驱动 MOS）     │
  │        └──────────────────────────────── │
  │                                         │
  │  闭环过程：                              │
  │  LED 电流偏大 → Rsense 电压升高          │
  │  → 运放输出下降 → MOS 关小 → 电流回落     │
  │  → 稳定在 Vref / Rsense                 │
  │  恒流值 I = Vref / Rsense                │
  └─────────────────────────────────────────┘
```

### 2.5 MCU 实现闭环（ADC + PID）

现代闭环大多由 MCU 通过 ADC 采样并用 PID 算法实现。

```
  ┌─────────────────────────────────────────────┐
  │  MCU 闭环控制系统（以恒温为例）               │
  │                                             │
  │  设定温度 25°C                               │
  │      │                                       │
  │      ▼                                       │
  │  [PID 计算] ←── 当前温度 ←── NTC ←── 被加热物 │
  │      │                                       │
  │      ▼                                       │
  │  [PWM 输出] ──→ MOS管 ──→ 加热丝              │
  │                                             │
  │  PID 公式（位置式）：                          │
  │  Output = Kp×e + Ki×∫e·dt + Kd×de/dt        │
  │          e = 设定值 - 测量值                  │
  └─────────────────────────────────────────────┘
```

## 三、传感器选型速查

| 被测量 | 传感器 | 输出信号 | 精度 |
|--------|--------|---------|------|
| 温度 | NTC 热敏电阻 | 电阻变化（需分压） | ±1~5°C |
| 温度 | DS18B20 | 数字 1-Wire | ±0.5°C |
| 温度 | LM35 | 模拟 10mV/°C | ±0.5°C |
| 光强 | 光敏电阻 LDR | 电阻变化 | ±20% |
| 距离 | HC-SR04 超声波 | 脉冲宽度 | ±1cm |
| 距离 | VL53L0X 激光 | I²C 数字 | ±1mm |
| 电流 | ACS712 | 模拟 2.5V±185mV/A | ±5% |
| 电流 | 采样电阻 + 运放 | 模拟 差分电压 | ±1% |
| 角度 | MPU6050 陀螺+加速 | I²C 数字 | ±1° |

## 四、闭环控制设计 checklist

- [ ] 传感器量程是否覆盖了整个工作范围？
- [ ] 传感器响应速度是否足够快？（温度用 NTC 反应慢，电机编码器需要快）
- [ ] 信号调理电路（放大/滤波）是否跟得上？
- [ ] 反馈极性对吗？（负反馈 → 稳定；正反馈 → 振荡/自锁）
- [ ] 是否需要滞回（施密特）来防止输出抖动？
- [ ] PID 参数是否需要整定？（P 太大→振荡，I 太大→过冲）
- [ ] 采样频率 ≥ 2 × 系统带宽（奈奎斯特）
- [ ] 是否有安全保护？（失控时自动切断输出）
- [ ] MCU 的 ADC 分辨率够吗？（0~3.3V / 12bit = 0.8mV 分辨率）

## 五、开环 vs 闭环：选型决策树

```
需要控制一个输出量吗？
  │
  ├── 否 → 开环（纯时序控制）
  │
  └── 是
       │
       ├── 精度要求 < ±20%？且负载稳定？
       │    └── 是 → 开环就够了
       │    └── 否 → 需要闭环
       │
       ├── 响应速度要求（ms 级）？
       │    └── 是 → 硬件闭环（比较器/运放）
       │
       └── 需要复杂控制算法？
            └── 是 → MCU + ADC + PID 软件闭环
```
