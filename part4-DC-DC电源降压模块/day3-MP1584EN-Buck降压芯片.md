# 元件档案 · MP1584EN — 降压开关稳压器（Buck）

> **一句话认识它**：一颗8只脚的电源芯片，用"高速开关+电感"的高效方式把高电压变成低电压。
> **封装**：SOP-8 EP（8个引脚 + 底部大散热焊盘）
> **售价**：≈14元/颗（立创商城，MPS原装）
> **核心数据**：输入4.5~28V → 可调输出（本项目设5V），最大3A，效率~90%

---

## 1. 长什么样？

### 外观与引脚定义

```
           俯视图（看芯片正面）
    ┌───────────────────────────────┐
    │    ① BST      ⑧ SS           │
    │                               │
    │    ② EN       ⑦ IN           │
    │                               │
    │    ③ COMP     ⑥ SW           │
    │                               │
    │    ④ FB       ⑤ GND          │
    └───────────────────────────────┘
           ┌─────────────────┐
           │  底部散热焊盘     │  ← 必须焊接到GND！
           │  (Exposed Pad)   │
           └─────────────────┘
```

**引脚速记口诀**：
```
① BST → ② EN → ③ COMP → ④ FB
 ⑤ GND → ⑥ SW → ⑦ IN  → ⑧ SS
 自举   使能   补偿   反馈
        地    开关   输入   软启动
```

### 各引脚详细说明

| 引脚 | 名称 | 全称 | 功能 | 不接会怎样？ |
|:----:|:----:|------|------|------------|
| **①** | **BST** | Bootstrap（自举） | 给内部高端MOS管驱动供电。通过0.1μF电容接到SW | **整个芯片不工作！** |
| **②** | **EN** | Enable（使能） | 高电平开启芯片(>1.35V)，低电平关闭(<1.2V)。**最高只能承受6V** | 拉低就没输出 |
| **③** | **COMP** | Compensation（补偿） | 接RC网络到GND，稳定反馈环路 | 输出可能振荡 |
| **④** | **FB** | Feedback（反馈） | 检测输出电压（内部基准0.8V） | 输出会失控 |
| **⑤** | **GND** | Ground（地） | 连接GND和底部散热焊盘 | |
| **⑥** | **SW** | Switch（开关节点） | 接电感和续流二极管。**电压在0V和Vin之间高频跳变** | 能量传不出去 |
| **⑦** | **IN** | Input（输入） | 主电源输入（4.5~28V） | 没电 |
| **⑧** | **SS** | Soft Start（软启动） | 接0.1μF到GND，设定启动时间 | 启动电流冲击大 |

## 2. 内部原理——"高速水龙头 + 飞轮"

### 2.1 四要素

MP1584的Buck降压电路由四个核心元件组成：

```
Vin ──→ [① 开关管] ──→ [② 电感 L] ──→ Vout
              │              │          │
              ├──→ [③ 续流二极管] ←─────┤
              │   (D)                    │
             GND               [④ 输出电容 Cout]
                                      │
                                     GND
```

| 元件 | 在水龙头比喻中 | 在飞轮比喻中 | 实际作用 |
|:----:|---------------|-------------|---------|
| **① 开关管** | 水龙头的开关 | 脚踩油门 | 以1.5MHz频率通断，把直流切成脉冲 |
| **② 电感L** | 水轮机（有惯性） | **飞轮**（储存旋转动能） | 通电时储能，断电时释放，维持电流 |
| **③ 续流二极管** | 回水管道 | 单向阀 | 开关断开时给电感电流提供回路 |
| **④ 输出电容Cout** | 大水桶 | 缓冲器 | 把脉冲电流"抚平"成平稳直流 |

### 2.2 分阶段工作过程（看懂这个就懂Buck了）

#### 阶段一：开关管导通（Ton）

```
Vin → [开关管·导通] → [电感 L] → Vout
                         ↓
                     电流从0开始上升
                     电感储存磁场能量
                         ↓
                     续流二极管·截止（反向偏置）
```

- 电流从Vin流过开关管→电感→输出
- 电感中的电流**逐渐上升**（因为电感"不愿意"电流突变）
- 电感以磁场形式储存能量
- 此时二极管不工作（SW节点电压≈Vin，二极管负极电压高、正极接GND，反偏）

**Buck两阶段波形图（示波器上看到的真实样子）：**

```\
              Ton（导通）│  Toff（关断） │ Ton │  Toff
SW节点电压：    ┌──────┐              ┌──────┐
Vin ───────────┤      │     Vin      │      │
               │      │              │      │
0V ────────────┘      └──────────────┘      └────
               
电感电流：     ╱╲     ╱╲             ╱╲     ╱╲
Ipeak ────────╱  ╲───╱  ╲───────────╱  ╲───╱  ╲
              ╱    ╲ ╱    ╲         ╱    ╲ ╱    ╲
Ivalley ─────╱      ╲      ╲───────╱      ╲      ╲
             ╱        ╲      ╲     ╱        ╲      ╲
            ╱          ╲      ╲   ╱          ╲      ╲
           ──────────────────────────────────────────
            ▲          ▲
      电感储能    电感释放能量
     （电流上升）  （电流下降，但不为0）
     
输出端：经过电容滤波后，Vout ≈ 直线
5V ──────────────────────────────────────────────
```

#### 阶段二：开关管关断（Toff）

```
Vin → [开关管·关断] ─ ─ [电感 L] → Vout
                          ↑    ↓
                          │  电流从峰值开始下降
                          │  电感释放磁场能量
                          │  ↓
                       [续流二极管·导通]
                           ↓
                      GND（电流从地流过二极管再到电感）
```

- 开关管断开后，电感的"惯性"让它继续推电流
- 电流路径变成：GND → 续流二极管（正向导通）→ 电感 → 输出
- 电感释放之前储存的磁场能量
- 电流逐渐下降

#### 阶段三：下一个周期……

以 **1.5MHz**（每秒150万次）重复导通→关断→导通→关断……

```
电流波形（理想化）：
I_L
↑
│    ┌───┐    ┌───┐    ┌───┐
│    │   │    │   │    │   │    ← 电流在I_peak和I_valley之间波动
│   ┌┘   └┐  ┌┘   └┐  ┌┘   └┐
│   │     │  │     │  │     │
└───┴─────┴──┴─────┴──┴─────┴──→ t
    ↑     ↑
  导通   关断
```

### 2.3 关键公式

**输出电压 = 输入电压 × 占空比（理想）**

```
Vout = Vin × D   其中 D = Ton ÷ (Ton + Toff)
```

**实际**：因为二极管压降和线路损耗，实际公式为：

```
Vout = Vin × D - V_二极管
```

**更重要：实际工作中占空比由反馈环路动态调节**

FB引脚检测输出电压，与内部基准0.8V比较：
- Vout太高 → 减小占空比 → SW导通时间缩短 → Vout降回来
- Vout太低 → 增大占空比 → SW导通时间加长 → Vout升上去

**反馈闭环的图示：**

```
                         ┌── 目标：Vout = 5V
                         │      （由R1/R2设定）
                         ▼
                ┌─────────────────┐
                │  误差放大器      │ ← 比较"实际"和"目标"的差距
                │  Vref = 0.8V    │
                └────────┬────────┘
                         │ 误差信号
                         ▼
                ┌─────────────────┐
                │  PWM控制器      │ ← 根据误差调整占空比
                │  占空比 = ?%    │
                └────────┬────────┘
                         │ 控制开关管
                         ▼
                ┌─────────────────┐
                │  开关管 + 电感   │ ← 执行能量转换
                │  + 续流二极管   │
                └────────┬────────┘
                         │ Vout
                         ▼
                ┌─────────────────┐
                │  分压电阻       │ ← 反馈：把Vout缩小后
                │  R1/R2 = 5.23倍 │    送回误差放大器
                └────────┬────────┘
                         │
                         └──→ 回到误差放大器，再来一次
                         
         这就是一个完整的闭环控制系统！
```

> 🔄 **这就是闭环控制！** MP1584的反馈环路就是"睁眼走路"——不断看输出、调占空比。

## 6. 容易犯的错误

| ❌ 错误 | 后果 | ✅ 正确做法 |
|---------|------|------------|
| **忘记焊BST自举电容** | **芯片不工作！** 输出=0V | 0.1μF电容焊在BST和SW之间 |
| **BST电容远离芯片** | 驱动不稳定 | 电容紧贴BST和SW引脚(≤5mm) |
| **输入电容离得远** | IN脚产生电压尖峰，可能击穿芯片 | 输入电容紧贴IN和GND |
| **EN直接接>6V电压** | 烧坏EN引脚 | Vin>6V时加分压电阻 |
| **FB走线经过电感和SW下方** | 输出纹波巨大、不稳定 | FB走线远离电感和SW区域 |
| **电感饱和电流不够** | 重载时输出电压掉下去 | 电感饱和电流 ≥ 3.9A（3A×1.3） |
| **二极管SS34极性焊反** | 短路！上电就烧 | 阴极（有横线端）接SW，阳极接GND |
| **忘记焊散热焊盘** | 芯片过热烧毁 | 底部散热焊盘必须焊接到GND铺铜 |

## 🧩 拓展延伸 — 小故事

### 🚀 "Buck"这个名字怎么来的？

Buck电路为什么不叫"Step-Down"或者"降压电路"，而要叫一个这么怪的名字？

这个名称其实来自**变压器的"Bucking"接法**——在早期电力系统中，两个变压器可以串联连接，如果次级绕组的极性相反，就会产生一个"抵消"电压（Bucking Voltage），用来降低输出电压。

1950~60年代，工程师把这种"相消"的思想用到了开关晶体管电路上——通过高速开关让输入电压被"抵消"掉一部分，剩下的就是想要的低压输出。于是这种电路就被叫做了 **Buck Converter**。

> 你每次说"Buck降压"的时候，说的其实是100年前变压器工程师的"互相抵消"方案——技术和名字一起流传到了今天。

### 🔥 开关电源改变了世界——IBM 360的故事

1964年，IBM发布了一款划时代的产品——**System/360大型计算机**。但它有一个严重的问题：电源系统体积巨大、发热极高。IBM的工程师**Bob Mammano**被派去解决这个问题。

他发现传统线性稳压电源的效率只有30~40%，360计算机里面几十个电路板需要的总功率高达几千瓦——按这个效率，**光电源散热就得装一台空调**。

Mammano把目光投向了当时还处于实验室阶段的"开关模式电源"。他设计了世界上第一款商用的开关电源控制芯片 **SG1524**（后来成为行业标准 μA78540 的前身），把电源效率从30%提升到了75%以上。

> IBM 360最终卖出了超过3万台，奠定了现代计算机的架构。你桌上每一台电脑、手机里的电源，都是Bob Mammano思想的后代。

### 🏭 MPS公司——从车库到上市

MP1584的生产商 **MPS（Monolithic Power Systems）**，成立于1997年，创始人是三位华裔工程师。公司起步时在一个只有几百平方英尺的小办公室，启动资金20万美元。

当时的电源芯片市场被Linear Tech、Maxim、TI等大厂垄断，MPS的策略很简单：**比大厂做得更便宜、更集成。** 他们把同步整流管也集成进芯片里，外面只需要一个电感和几个电容——对手需要十几个元件，MPS只要三五个。

2023年，MPS的年收入已经超过18亿美元，成为全球最大的模拟IC公司之一。

> MP1584是MPS的经典量产芯片，虽然已经"不推荐新设计使用"（NRFND），但它在全球的累计出货量超过数亿颗。你手里这块电源板上用的，是一家年营收百亿人民币的公司最经典的芯片。
