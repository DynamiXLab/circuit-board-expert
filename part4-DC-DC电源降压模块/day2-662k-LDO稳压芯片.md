# 元件档案 · 662K — 3.3V低压差线性稳压器（LDO）

> **一句话认识它**：一颗只有三只脚、比米粒还小、花8分钱就能买到的3.3V稳压芯片。
> **封装**：SOT-23-3（三个引脚的小贴片，长得像三极管）
> **售价**：≈0.08元/颗（立创商城）

---

## 1. 长什么样？

### 外观与引脚定义

```
         俯视图（看芯片正面，有字朝上）
              __
             /  \
         ┌──┤662├──┐   ← 正面丝印写 "662K"
         │  \__/    │
         │          │
       ①脚       ②脚       ③脚
       GND      Vout      Vin
       (地)     (3.3V输出) (5V输入)
```

**引脚说明（记住这个顺序！）**

| 引脚 | 名称 | 功能 | 在电路中接什么 |
|:----:|:----:|------|----------------|
| **①** | **GND** | 地——0V参考点 | 接GND（大面积铺铜） |
| **②** | **Vout** | **3.3V稳压输出** | 接负载（传感器/MCU等） |
| **③** | **Vin** | 电源输入（**最高6.0V**） | 接5V（来自MP1584的输出） |

> ⚠️ **重要**：引脚顺序是 **①GND ②Vout ③Vin**，不是③①①的顺序！焊之前一定要用万用表通断档确认。

### 实物大小对比

    │SOT│    │  SOT-223 │    ← 662K只有AMS1117的1/4大

## 2. 内部原理——"智能可变电阻"

### 2.1 LDO内部结构（实际电路）

```
                      ┌──────────────────┐
                      │   662K 内部       │
                      │                  │
 Vin ───┬─────────────┤ ③ 调整管(MOS)    │
        │             │     ↑            │
        │             │     │ 控制信号   │
        │             │  ┌──┴───┐       │
        │             │  │误差放大器│     │
        │             │  └──┬───┘       │
        │             │     ↑           │
        │             │  ┌──┴───┐       │
        │             │  │基准电压│0.8V  │  ← 内部的\"标准尺子\"
        │             │  └──────┘       │
        │             └────────┬────────┘
        │                      │② Vout
        │     ┌──────────┐     │
        └─────┤ ① GND    ├─────┘
              └──────────┘
                    │
                   GND
```

**工作流程（闭环控制）：**
```
① 误差放大器 \"看\" Vout 的分压值
② 和内部基准 0.8V 比较 → 差值 = 误差信号
③ 误差信号控制调整管的导通程度
④ Vout 偏高了 → 关小调整管 → 压降增大 → Vout 回落
⑤ Vout 偏低了 → 开大调整管 → 压降减小 → Vout 回升
⑥ 回到①，一秒钟重复成千上万次
```

### 2.2 用比喻理解

**662K的内部就像一个自动调节的可变电阻：**

```
   Vin (5V) ──┬── [智能可变电阻] ──┬── Vout (3.3V)
               │                     │
               │      ↑              │
               │   [控制电路] ← 基准电压 0.8V
               │                     │
               └── GND ──────────── ─┴── GND
```

**工作过程**（闭环控制）：（复习：睁眼走路）

```
① 控制电路"看"输出电压（Vout）
② 和内部基准（0.8V）比较：
   - Vout太高？→ 增大电阻 → 压降变大 → Vout降回来
   - Vout太低？→ 减小电阻 → 压降变小 → Vout升回去
③ 回到①，不断循环
```

> 💡 整个过程每秒调整**成千上万次**，所以输出非常稳定。

### 2.2 核心参数

| 参数 | 值 | 什么意思 |
|------|:---:|---------|
| **输入电压范围** | ≤ **6.0V** | ⚠️ **不能接电池！** 必须接5V |
| **输出电压** | **3.3V（固定）** | 买回来就是3.3V，不需要调 |
| **最大输出电流** | **250mA** | 给传感器够用了（总共不到100mA） |
| **压差（Dropout）** | ~0.2V @ 90mA | 输入只比输出高0.2V就能工作 |
| **静态电流（Iq）** | **25μA** | 自己几乎不耗电，适合电池供电 |
| **精度** | ±3% | 输出3.3V时，实际在3.2~3.4V之间 |
| **输出噪声** | 极低（μV级） | 比Buck干净得多，适合模拟电路 |
| **工作温度** | -40°C ~ +85°C | 四季都能用 |

## 3. 典型应用电路

### 3.1 完整电路（就这几个元件）

```
               ┌──────────┐
   5V (来自Buck) ───┤③ Vin    │
                    │         │
   C1 ──┤├──┬──────┤① GND    │
   1μF     │      │         │
           │      │② Vout ──┼──┬── C2 ──┤├── GND
          GND     └──────────┘  │   1μF
                                │
                           Vout (3.3V)
```

**就这么多——输入一个电容、输出一个电容。**

### 3.2 元件说明

| 元件 | 规格 | 作用 | 为什么是这个值 |
|------|------|------|--------------|
| **C1 (输入电容)** | 1μF MLCC 0603 | 防止输入电压抖动 | 662K内部已稳压，外部1μF足够 |
| **C2 (输出电容)** | 1μF MLCC 0603 | 让3.3V输出更平滑 | 传感器电流小(<100mA)，1μF够用 |

> 对比：如果你用AMS1117，需要10μF电解+100nF各两个（4个电容）。662K只用**2个0603小电容**——省地方、省成本。

## 4. 为什么选662K而不是AMS1117？

### 4.1 对比表

| 对比项 | 662K（本模块选用） | AMS1117-3.3（传统方案） |
|-------|-------------------|----------------------|
| **封装** | **SOT-23-3**（极小） | SOT-223（较大） |
| **价格** | **~0.08元** | ~0.3元 |
| **静态电流** | **25μA**（几乎不耗电） | ~5mA（一直耗电） |
| **所需外围** | **2个1μF** | 4个电容（10μF+100nF×2） |
| **输出电流** | 250mA（够用） | 1A（大材小用） |
| **输入电压** | ≤6.0V（限制！） | ≤15V（更宽） |
| **PCB占板面积** | **~3mm²** | ~20mm² |

### 4.2 选择理由

| 理由 | 说明 |
|------|------|
| **① 够用≠浪费** | 传感器总共不到100mA，250mA绰绰有余 |
| **② 省电是关键** | 662K静态电流25μA vs AMS1117的5mA——差了**200倍**！巡线小车用电池，省电很重要 |
| **③ 越小越好** | SOT-23-3比SOT-223小得多，给巡线小车主板省下宝贵的板面空间 |
| **④ 外围简单** | 两个1μF电容搞定，焊接也简单 |
| **⑤ 便宜** | 8分钱一片，跟白送差不多 |

### 4.3 什么场景应该用AMS1117？

如果项目要求 **输出 > 250mA**（比如带大功率传感器、多个舵机），或者输入电压 > 6V，那就得用AMS1117。但在我们的巡线小车上，传感器电流总和 < 100mA——662K正好。

## 5. 容易犯的错误

| ❌ 错误 | 后果 | ✅ 正确做法 |
|---------|------|------------|
| 把662K的Vin直接接7.4V电池 | 芯片烧毁（最高6V！） | 必须接Buck的5V输出 |
| 引脚搞错（把Vin当GND焊） | 不工作或烧芯片 | 焊前用万用表确认①=GND ②=Vout ③=Vin |
| 省掉输入电容C1 | 输出电压抖动 | 必须焊C1（1μF），紧贴芯片 |
| 输出电流超过250mA | 芯片过热保护/输出电压下降 | 确认负载电流 ≤ 250mA |
| 把662K靠近电感放 | 输出噪声增大 | 662K尽量远离MP1584的电感和SW区域 |

## 🧩 拓展延伸 — 小故事

### 👨‍🔬 线性稳压器的"教父"——Bob Widlar

如果你打开任何一个老式电源芯片的数据手册，很大概率会在设计者一栏看到同一个名字：**鲍勃·威德拉**（Bob Widlar）。

Widlar是模拟IC设计领域的传奇人物。1960年代，他在仙童半导体（Fairchild）设计了世界上第一款集成运放μA702，后来跳槽到国家半导体（National Semiconductor）。他有一个著名习性：**如果同事太吵影响他思考，他就把办公室的门拆下来当桌子用。**

1970年，Widlar设计了一款划时代的产品——**LM109**，五端稳压器。但更出名的是几年后的 **LM317**（可调三端稳压器），它的外围只需要两个电阻就能设定任意输出电压。LM317从1977年一直生产到今天——**47年不断供**，堪称芯片界的常青树。

> 662K的原理和LM317一样，只是封装更小、电流更小。你今天学的"可调电阻分压"思想，源头就来自Widlar在1970年代的一张草图。

### 💰 压差的故事——从1.5V到0.1V

早期的线性稳压器（如7805）有一个坑：**输入电压必须比输出高2~3V才能工作**。你要输出5V，至少得输入7V——多出来的2V就是压在芯片上的额外功耗。

这限制了电池供电设备的发展。一个4.2V的锂电池，根本没法用7805输出3.3V（输入不够高）。

1980年代，工程师们想了一个办法：把调整管从普通PNP三极管换成**PNP达林顿管**，再后来换成**PMOS管**——它们的导通压降可以做到0.1~0.4V。这种新型稳压器被称为 **LDO（Low Dropout，低压差）**。

> 662K的压差只有0.2V——这意味着你用5.0V输入就能输出稳定的3.3V，中间只浪费了0.2V×0.1A=0.02W。没有LDO的发明，今天你的手机根本撑不了半天。

### 🔢 为什么叫"662K"？

芯片上的丝印"662K"其实是一种**简化的标记代码**，因为SOT-23-3封装太小了，印不下完整的型号。

662K真正的芯片型号通常是 **XC6206P332MR**（或类似），由日本Torex公司设计。其中：
- **"332"** 表示输出电压3.3V（如果是"182"就是1.8V，"502"就是5.0V）
- 前面的数字是产品系列编号

立创商城里卖8分钱一个的"662K"，和原装Torex XC6206P332的核心设计基本一致，来源是国内晶圆厂流片生产的兼容版本。

> 这就是中国制造的魅力：8分钱一个，性能够用，让每个电子爱好者都能轻松做出一块自己的电源板。
