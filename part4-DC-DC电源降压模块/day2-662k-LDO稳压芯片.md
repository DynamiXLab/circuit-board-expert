# 元件档案 · 662K — 3.3V低压差线性稳压器（LDO）

> **一句话认识它**：一颗只有三只脚、比米粒还小、花8分钱就能买到的3.3V稳压芯片。
> **封装**：SOT-23-3（三个引脚的小贴片，长得像三极管）
> **售价**：≈0.08元/颗（立创商城）

### 外观与引脚定义

```
         俯视图（看芯片正面，有字朝上）
              __
             /  \
         ┌──┤662├──┐   ← 正面丝印写 "662K"
         │  \__/    │

       ①脚       ②脚       ③脚
       GND      Vout      Vin
       (地)     (3.3V输出) (5V输入)
```

**引脚说明（记住这个顺序！）**

| 引脚 | 名称 | 功能 | 在电路中接什么 |
|:----:|:----:|------|----------------|
| **①** | **GND** | 地——0V参考点 | 接GND（大面积铺铜） |

| **③** | **Vin** | 电源输入（**最高6.0V**） | 接5V（来自MP1584的输出） |

> ⚠️ **重要**：引脚顺序是 **①GND ②Vout ③Vin**，不是③①①的顺序！焊之前一定要用万用表通断档确认。

## 2. 内部原理——"智能可变电阻"

### 2.1 芯片内部长什么样？（四个核心模块）

662K的内部不是"一块东西"，而是由四个功能模块精密配合构成的：

```
                     ┌─────────────────────────────────────┐
                     │            662K 内部                  │
                     │                                     │
Vin ───┬─────────────┤  ┌─────────────────────────────┐    │
       │             │  │ ③ 调整管（Pass Element）      │    │
       │             │  │  ┌───────────────────────┐   │    │
       │             │  │  │    PMOS 功率管          │   │    │
       │             │  │  │    (低导通电阻)         │   │    │
       │             │  │  └───────────┬───────────┘   │    │
       │             │  │              │ 栅极控制电压    │    │
       │             │  └──────────────┼──────────────┘    │
       │             │                 │                    │
       │             │     ┌───────────┴───────────┐       │
       │             │     │ ① 误差放大器            │       │
       │             │     │  (Error Amplifier)     │       │
       │             │     │  把\"实际电压\"和\"目标\"比较  │       │
       │             │     └───────────┬───────────┘       │
       │             │                 ↑                   │
       │             │     ┌───────────┴───────────┐       │
       │             │     │ ② 基准电压源            │       │
       │             │     │  (Bandgap Reference)   │       │
       │             │     │  提供稳定的 1.2V 参考    │       │
       │             │     └───────────┬───────────┘       │
       │             │                 │                    │
       │             │     ┌───────────┴───────────┐       │
       │             │     │ ④ 反馈分压电阻          │       │
       │             │     │  (内置于芯片,用户看不到)  │       │
       │             │     │  把 Vout 分压后送入 EA   │       │
       │             │     └───────────┬───────────┘       │
       │             │                 │② Vout             │
       │             │                 ▼                    │
       │             │           ┌──────────┐              │
       │             │           │ 保护电路  │              │
       │             │           │ (过流+过热)│              │
       │             │           └──────────┘              │
       │             └─────────────────────────────────────┘
       │                                    │
       │      ┌───────────┐                 │
       └──────┤ ① GND     ├─────────────────┘
              └───────────┘
                    │
                   GND
```

#### 模块①：误差放大器（Error Amplifier）—— 芯片的"眼睛和大脑"

误差放大器是一个**运算放大器**，它的工作就是不停地做一件事：

```
       FB端（实际电压的分压值）────┬── 运放同相输入
                                   │
                                   ├──→ 输出 = (V_FB - V_ref) × 增益
                                   │
       基准电压（1.2V）────────────┴── 运放反相输入
```

- 它把**实际输出电压的分压值**（FB电压）和**内部基准电压**（1.2V）做比较
- 如果FB > 1.2V → 输出变高 → 告诉调整管"关小一点"
- 如果FB < 1.2V → 输出变低 → 告诉调整管"开大一点"
- 如果FB = 1.2V → 保持不动 → 输出刚刚好

> 这个放大器的增益非常高（典型值 > 60dB，也就是放大1000倍以上），所以只要FB和基准有微小的差距，它就能产生足够强的控制信号。

#### 模块②：基准电压源（Bandgap Reference）—— 芯片的"标准尺子"

基准电压源是一个**不受温度、电压影响的稳定电压源**。662K内部的基准电压是 **1.2V**（带隙基准，Bandgap）。

为什么它这么重要？
```
如果基准电压不稳：
   基准  当前值   →  输出电压
   1.20V  1.20V   →  3.30V（正确）
   1.18V  1.20V   →  3.28V（偏低了！）
   1.22V  1.20V   →  3.32V（偏高了！）
   
→ 基准漂移多少，输出就漂多少
```

662K内部的带隙基准电路能做到在**全温度范围（-40°C~85°C）内漂移小于±1%**——也就是说，无论冬天东北的户外还是夏天海南的车内，基准电压基本不变。

#### 模块③：调整管（Pass Element）—— 芯片的"执行器"

调整管是一个**PMOS功率管**（P沟道MOSFET），它像一个可以连续调节的电子阀门：

```
PMOS管的三个工作区：

① 截止区（完全关断）：
   栅极电压 = 高 → 管子完全关闭 → 电流 = 0 → Vout = 0

② 线性区（部分导通——正常工作区）：
   栅极电压 = 中等 → 管子打开一部分 → 电流受控 → Vout = 设定值
   误差放大器不断微调栅极电压，让管子停留在"刚刚好"的位置

③ 完全导通（饱和区）：
   栅极电压 = 低 → 管子完全打开 → 电流最大 → Vout ≈ Vin（无法稳压）
```

**为什么662K用PMOS而不是NMOS？**
- PMOS管的**栅极阈值电压是负的**（栅极比源极低时导通）
- 这意味着PMOS的栅极可以用较低电压控制——对于低压差应用很关键
- 早期LDO用PNP三极管，压差大（~1V）；现代LDO用PMOS，压差可做到0.1~0.2V

#### 模块④：反馈分压电阻 — 芯片的"测量工具"

662K是**固定输出3.3V**的版本，所以内部的反馈分压电阻已经做好了：

```
        Vout（目标3.3V）
            │
         ┌──┴──┐
         │ R1  │ ← 内部电阻（用户看不见）
         └──┬──┘
            │
            ├────→ FB（送到误差放大器，目标1.2V）
            │
         ┌──┴──┐
         │ R2  │ ← 内部电阻（用户看不见）
         └──┬──┘
            │
           GND
```

分压公式（内部已经算好了）：
```
V_FB = Vout × R2 / (R1 + R2)

要让 V_FB = 1.2V（基准），Vout = 3.3V：
R1 / R2 = (3.3 / 1.2) - 1 = 1.75
```

> 如果是"可调版本"的LDO（比如LM317），这些电阻在芯片外面，用户可以自己选。662K是固定版本，已经焊死在芯片内部了——这也是它只需要两个外接电容的原因。

### 2.2 完整工作流程（多图串联）

把四个模块串起来，看一次完整的稳压过程：

```
步骤①：负载突然需要更多电流
  传感器突然启动 → 电流从10mA跳到80mA
  Vout瞬间跌落（从3.300V降到3.295V）

步骤②：误差放大器发现偏差
  FB电压 = 3.295V × R2/(R1+R2) = 1.198V（原来是1.200V）
  偏差 = 1.200V - 1.198V = 0.002V = 2mV
  误差放大器把这个2mV放大1000倍 → 输出变化2V

步骤③：调整管响应
  误差放大器的输出控制PMOS的栅极
  栅极电压变化 → PMOS导通程度增大 → 电阻变小
  更多的电流可以通过 → Vout开始回升

步骤④：新的平衡
  Vout回到3.300V → FB回到1.200V
  误差放大器输出稳定 → PMOS停在新的导通位置
  系统重新达到平衡

整个过程的耗时：约 10~50 微秒（μs）
也就是 0.00005 秒——你完全感觉不到
```

```
时间轴（用眼睛看电压变化）：
Vout
│
3.300V ────┐              ┌──────── ④ 恢复稳定
           │              │
3.298V ────┴ ①负载变化    │
           │              │
3.295V ────┴────────── ②误差放大
           │    │ ③调整管响应
           └────┴────────────────→ t
           ← 10~50μs →
           从跌落到恢复，十万分之五秒
```

> 这整个过程每秒发生**成千上万次**——负载在变、温度在变、输入电压在变，误差放大器在一刻不停地"监视"和"修正"。

### 2.3 保护电路（藏在芯片里的"安全卫士"）

662K内部还集成了两个保护功能，虽然平时用不到，但关键时刻能救芯片一命：

**① 过流保护（Overcurrent Protection — OCP）**

```
正常情况下：输出电流 < 250mA → 一切正常
异常情况：输出短路 → 电流突然变得非常大
保护动作：内部电路检测到电流超过阈值 → 限制最大输出电流
效果：芯片不会烧，短路解除后自动恢复
```

**② 过热关断（Thermal Shutdown — TSD）**

```
正常工作：芯片温度 < 125°C → 正常工作
异常情况：散热不良/环境温度高 → 芯片温度超过150°C
保护动作：芯片内部检测到过高温度 → 自动关闭输出
效果：芯片冷却到约120°C后自动重启
```

> **这两个保护让662K非常"皮实"**——新手焊接时不小心短路一下，或者接错线了，芯片大概率不会坏，它会自己保护自己。

### 2.4 用比喻理解（通俗版）

**662K的内部就像一个自动调节的可变电阻：**

```
   Vin (5V) ──┬── [智能可变电阻] ──┬── Vout (3.3V)
               │                     │
               │      ↑              │
               │   [控制电路] ← 基准电压 0.8V
               │                     │
               └── GND ──────────── ─┴── GND
```

**工作过程**（闭环控制）：（复习：睁眼走路）

```
① 控制电路"看"输出电压（Vout）
② 和内部基准（0.8V）比较：
   - Vout太高？→ 增大电阻 → 压降变大 → Vout降回来
   - Vout太低？→ 减小电阻 → 压降变小 → Vout升回去
③ 回到①，不断循环
```

> 💡 整个过程每秒调整**成千上万次**，所以输出非常稳定。

### 2.5 核心参数

| 参数 | 值 | 什么意思 |
|------|:---:|---------|
| **输入电压范围** | ≤ **6.0V** | ⚠️ **不能接电池！** 必须接5V |
| **输出电压** | **3.3V（固定）** | 买回来就是3.3V，不需要调 |
| **最大输出电流** | **250mA** | 给传感器够用了（总共不到100mA） |
| **压差（Dropout）** | ~0.2V @ 90mA | 输入只比输出高0.2V就能工作 |
| **静态电流（Iq）** | **25μA** | 自己几乎不耗电，适合电池供电 |
| **精度** | ±3% | 输出3.3V时，实际在3.2~3.4V之间 |
| **输出噪声** | 极低（μV级） | 比Buck干净得多，适合模拟电路 |
| **工作温度** | -40°C ~ +85°C | 四季都能用 |

## 3. 典型应用电路

### 3.1 完整电路（就这几个元件）

```
               ┌──────────┐
   5V (来自Buck) ───┤③ Vin    │
                    │         │
   C1 ──┤├──┬──────┤① GND    │
   1μF     │      │         │
           │      │② Vout ──┼──┬── C2 ──┤├── GND
          GND     └──────────┘  │   1μF
                                │
                           Vout (3.3V)
```

**就这么多——输入一个电容、输出一个电容。**

### 3.2 元件说明

| 元件 | 规格 | 作用 | 为什么是这个值 |
|------|------|------|--------------|
| **C1 (输入电容)** | 1μF MLCC 0603 | 防止输入电压抖动 | 662K内部已稳压，外部1μF足够 |
| **C2 (输出电容)** | 1μF MLCC 0603 | 让3.3V输出更平滑 | 传感器电流小(<100mA)，1μF够用 |

> 对比：如果你用AMS1117，需要10μF电解+100nF各两个（4个电容）。662K只用**2个0603小电容**——省地方、省成本。

## 4. 为什么选662K而不是AMS1117？

### 4.1 对比表

| 对比项 | 662K（本模块选用） | AMS1117-3.3（传统方案） |
|-------|-------------------|----------------------|
| **封装** | **SOT-23-3**（极小） | SOT-223（较大） |
| **价格** | **~0.08元** | ~0.3元 |
| **静态电流** | **25μA**（几乎不耗电） | ~5mA（一直耗电） |
| **所需外围** | **2个1μF** | 4个电容（10μF+100nF×2） |
| **输出电流** | 250mA（够用） | 1A（大材小用） |
| **输入电压** | ≤6.0V（限制！） | ≤15V（更宽） |
| **PCB占板面积** | **~3mm²** | ~20mm² |

### 4.2 选择理由

| 理由 | 说明 |
|------|------|
| **① 够用≠浪费** | 传感器总共不到100mA，250mA绰绰有余 |
| **② 省电是关键** | 662K静态电流25μA vs AMS1117的5mA——差了**200倍**！巡线小车用电池，省电很重要 |
| **③ 越小越好** | SOT-23-3比SOT-223小得多，给巡线小车主板省下宝贵的板面空间 |
| **④ 外围简单** | 两个1μF电容搞定，焊接也简单 |
| **⑤ 便宜** | 8分钱一片，跟白送差不多 |

### 4.3 什么场景应该用AMS1117？

如果项目要求 **输出 > 250mA**（比如带大功率传感器、多个舵机），或者输入电压 > 6V，那就得用AMS1117。但在我们的巡线小车上，传感器电流总和 < 100mA——662K正好。

## 🧠 小巧思 — 想一想

### 思考①：662K的输入端能不能接一个LED指示灯？

```
问题：我想在662K的Vin和GND之间接一个LED，显示\"电源已接通\"，可以吗？
```

**可以，但要算一下电流：**

```
Vin = 5V，LED压降≈2V，限流电阻1kΩ
LED支路电流 = (5-2) / 1000 = 3mA

662K总输入电流 = 传感器电流(80mA) + LED电流(3mA) = 83mA
→ 远小于250mA，完全没问题 ✅
```

但注意：**LED的GND要和662K的GND连在一起**，不能分开接——否则两个GND之间有电位差，3.3V输出就不准了。

### 思考②：662K的输出能不能给两个传感器同时供电？

```
问题：我有一个3.3V的温湿度传感器（工作电流2mA）
       和一个3.3V的蓝牙模块（工作电流40mA，峰值100mA）
       能同时接在662K的输出上吗？
```

**算总账：**

```
总电流 = 2mA（温湿度）+ 100mA（蓝牙峰值，通讯时）
       = 102mA

662K最大输出 = 250mA
余量 = 250 - 102 = 148mA（还有60%的余量）✅
```

**但有坑！** 蓝牙模块在发射信号时电流会**从40mA瞬间跳变到100mA**。662K的响应时间约10~50μs，在这个瞬间输出电压会有一个短暂的小跌落（约几mV）。如果温湿度传感器对电压很敏感，可能会读数跳一下。

**解决办法**：在蓝牙模块的电源脚旁边多加一个**10μF电容**，提供瞬时电流缓冲。

> **结论**：能用，但建议在蓝牙模块处额外加电容。

---

### 思考③：输入电压刚好5.0V，662K能正常工作吗？

```
问题：662K的最低输入电压是多少？我的Buck输出正好5.00V，够不够？
```

**算：**

```
662K输出3.3V，压差是0.2V @ 90mA
→ 最低输入电压 = 3.3 + 0.2 = 3.5V

所以5.0V输入绰绰有余 ✅
```

甚至Buck输出精度有±5%——也就是4.75V~5.25V——也完全够用。

**延伸思考：** 如果输入电压从5V降到4V，662K还能输出3.3V吗？

```
→ 压差 = 4.0 - 3.3 = 0.7V，大于0.2V → 能正常工作 ✅
→ 但此时调整管上的功耗 = (4-3.3) × 0.1A = 0.07W，比原来多了一倍多
→ 芯片会稍微热一点，但仍在安全范围内
```

### 思考④：如果把662K的输入和输出接反了会怎样？

```
Vin ←────── 662K ──────→ Vout
  正常：5V接③Vin，3.3V从②Vout出
  接反：3.3V接③Vin，5V从②Vout出
```

**结果：** 662K内部有一个**从Vout到Vin的寄生二极管**（PMOS的体二极管）。如果Vout比Vin高，电流会通过这个二极管从Vout倒灌回Vin。

```
5V → ②Vout ──→ [体二极管导通] ──→ ③Vin → 3.3V
                                     ↑
                                相当于把5V接到了3.3V上！
```

**可能造成：**
1. 662K本身不会立即烧（有保护）
2. 但是**前级的Buck（输出5V）会受到反向电压冲击**
3. 如果Buck没有防反灌保护，Buck的输出电容会被662K的输入端充电到异常电压

**正确做法**：焊接前用万用表通断档确认①=GND ②=Vout ③=Vin。

### 思考⑤：为什么662K的数据手册推荐输入输出电容不小于1μF，用10μF是不是更好？

```
问题：1μF换成10μF，滤波效果不是更好吗？
```

**答案是：不一定。**

```

用1μF时：
  ┌──────┐
  │ 1μF  │ ← 响应快，高频噪声滤除好
  └──────┘

用10μF时：
  ┌──────┐
  │ 10μF │ ← 储能量大，但高频特性差
  └──────┘
```

| | 1μF MLCC | 10μF MLCC |
|:---|:---------:|:----------:|
| 储能容量 | 小 | 大 ✅ |
| 高频响应 | **快** ✅ | 慢 |
| 高频阻抗 | **低** ✅ | 高（内部电感大） |
| 价格 | 0.02元 ✅ | 0.1元 |

**结论：** 1μF MLCC在高频段的阻抗比10μF MLCC更低——滤除Buck残留的高频开关噪声效果更好。如果在1μF的基础上再**并联**一个10μF，倒是可以兼顾高频和储能——但662K电流小、不需要那么大储能，1μF就是性价比最优解。

> 在电子设计中，"越大越好"不总是对的——频率特性、响应速度往往比容量更重要。

| ❌ 错误 | 后果 | ✅ 正确做法 |
|---------|------|------------|

## 🧩 拓展延伸 — 小故事

### 👨‍🔬 线性稳压器的"教父"——Bob Widlar

如果你打开任何一个老式电源芯片的数据手册，很大概率会在设计者一栏看到同一个名字：**鲍勃·威德拉**（Bob Widlar）。

Widlar是模拟IC设计领域的传奇人物。1960年代，他在仙童半导体（Fairchild）设计了世界上第一款集成运放μA702，后来跳槽到国家半导体（National Semiconductor）。他有一个著名习性：**如果同事太吵影响他思考，他就把办公室的门拆下来当桌子用。**

1970年，Widlar设计了一款划时代的产品——**LM109**，五端稳压器。但更出名的是几年后的 **LM317**（可调三端稳压器），它的外围只需要两个电阻就能设定任意输出电压。LM317从1977年一直生产到今天——**47年不断供**，堪称芯片界的常青树。

> 662K的原理和LM317一样，只是封装更小、电流更小。你今天学的"可调电阻分压"思想，源头就来自Widlar在1970年代的一张草图。

### 💰 压差的故事——从1.5V到0.1V

早期的线性稳压器（如7805）有一个坑：**输入电压必须比输出高2~3V才能工作**。你要输出5V，至少得输入7V——多出来的2V就是压在芯片上的额外功耗。

这限制了电池供电设备的发展。一个4.2V的锂电池，根本没法用7805输出3.3V（输入不够高）。

1980年代，工程师们想了一个办法：把调整管从普通PNP三极管换成**PNP达林顿管**，再后来换成**PMOS管**——它们的导通压降可以做到0.1~0.4V。这种新型稳压器被称为 **LDO（Low Dropout，低压差）**。

> 662K的压差只有0.2V——这意味着你用5.0V输入就能输出稳定的3.3V，中间只浪费了0.2V×0.1A=0.02W。没有LDO的发明，今天你的手机根本撑不了半天。

### 🔢 为什么叫"662K"？

芯片上的丝印"662K"其实是一种**简化的标记代码**，因为SOT-23-3封装太小了，印不下完整的型号。

662K真正的芯片型号通常是 **XC6206P332MR**（或类似），由日本Torex公司设计。其中：
- **"332"** 表示输出电压3.3V（如果是"182"就是1.8V，"502"就是5.0V）
- 前面的数字是产品系列编号

立创商城里卖8分钱一个的"662K"，和原装Torex XC6206P332的核心设计基本一致，来源是国内晶圆厂流片生产的兼容版本。

> 这就是中国制造的魅力：8分钱一个，性能够用，让每个电子爱好者都能轻松做出一块自己的电源板。
