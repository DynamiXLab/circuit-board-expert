# Day 4 — 电源管理模块 · 综合实践

> **目标**：把Day 2（662K LDO）和Day 3（RT8289GSP Buck）的知识整合起来，完成完整的电源模块设计——从原理图到PCB到焊接测试。
> **在巡线小车中的角色**：电池(7.4V) → **RT8289GSP Buck** → 5V（给舵机+MCU）→ **662K LDO** → 3.3V（给传感器）

---

## 1. 整体供电架构

```
锂电池组（7.4V / 2S）
     │
     ▼
 ┌──────────────┐
 │ RT8289GSP    │ → 5V / 3A
 │ Buck降压      │    ├── 舵机供电（5V）
 └──────────────┘    ├── MCU供电（5V→板载LDO→3.3V）
     │               └── Buck输出→662K输入
     │ 5V
     ▼
 ┌──────────────┐
 │ 662K LDO     │ → 3.3V / 250mA → 灰度传感器、蓝牙等
 └──────────────┘
```

**为什么不用一级降压？**

```
一级7.4V→3.3V（只用LDO）:
  效率 = 3.3 ÷ 7.4 = 44.6% → 超过一半的电变成热量 ❌

两级7.4V→5V（Buck）→3.3V（LDO）:
  Buck效率≈90%，LDO效率=3.3÷5=66%
  综合效率 ≈ 90% × 66% = 59.4% → 好得多 ✅
```

> 每个芯片的详细原理见 `day1-基础知识的补充.md`（基础知识）和 `day2/day3`（芯片档案）。

## 4. PCB布局——四大铁律

> ⚠️ 这是整个模块最关键的部分。布局不对，电路可能完全不工作。

### 铁律一：输入电容紧贴芯片

```
✅ 正确：                      ❌ 错误：
┌──────┐                    C1  C2（放远了）
│C1 C2 │ ← 距离≤5mm          │    │
└──┬───┘                     │    │ ← 长走线
   ▼ ▼                       ▼    ▼
┌──────────┐              ┌──────────┐
│IN    GND │              │IN    GND │ ← 电压尖峰会击穿芯片
│RT8289GSP│              │RT8289GSP│
└──────────┘              └──────────┘
```

### 铁律二：SW节点面积最小化

SW引脚 + 电感L1 + 二极管D1 → 三者摆成**紧凑三角形**，走线短而粗。

### 铁律三：FB走线远离SW和电感

FB走线用0.25mm细线，不打过孔，远离电感和SW区域。

### 铁律四：底部完整地平面

MP1584下方一整片GND铺铜，不打断。输入/输出电容的GND焊盘各打2~3个过孔到底层。

> 详细布局步骤见 `day5-嘉立创EDA实践.md`。

| 走线 | 电流 | 线宽 |
|:-----|:----:|:----:|
| Vin → C1/C2 → RT8289GSP的IN | ~3A | **2~2.5mm** |

| FB分压走线 | <1mA | 0.25mm |
| 662K输入/输出 | ~100mA | 0.5mm |
| GND | 回路电流 | **大面积铺铜** |

## 6. 焊接注意事项

| 芯片 | 封装 | 焊接要点 |
|:-----|:-----|:---------|
| **RT8289GSP** | SOP-8 EP | 底部散热焊盘先预锡，热风枪或背面灌锡 |
| **SS54二极管** | SMA | **阴极（有横线端）接SW**，焊反了短路！ |
| **电感L1** | CD75/CD105 | 焊接牢固（电流大，虚焊会发热） |
| **662K** | SOT-23-3 | 引脚间距小，注意连锡。①GND ②Vout ③Vin |

## 7. 测试（⚠ 安全第一）

### 上电前检查

- [ ] 万用表通断档测输入/输出正负极间**不短路**
- [ ] 测RT8289GSP的IN和GND之间**不短路**
- [ ] 测662K的Vin和GND之间**不短路**
- [ ] 目检SS54极性是否正确

### 首次上电

**强烈建议**：用稳压电源限流100mA，或输入串10Ω/2W电阻。

1. 输入端接7.4V（或12V）
2. 测Buck输出→应≈5V（±0.1V）
3. 测LDO输出→应≈3.3V（±0.1V）
4. 用示波器看输出纹波（AC耦合）→Buck应<50mV

### 带载测试

接5Ω/5W电阻做1A负载→输出电压应保持5V，纹波<100mV。

## 8. 常见问题排查

| 现象 | 可能原因 | 先查什么 |
|:-----|:---------|:---------|
| Buck输出=0V | EN没拉高(<0.7V)/输入<5.5V | 测EN脚电压（应>1.5V） |
| Buck输出=输入电压 | FB开路→满占空比 | 测FB脚电压（应=1.222V） |
| 输出纹波巨大 | 电容太小/电感饱和/布局差 | 检查电感是否饱和 |
| 芯片上电就烧 | 输入电容离太远/电压超32V | 检查布局 |
| LDO输出不对 | 662K型号买错/电容焊错 | 看丝印"662K" |
| LDO发热(>80°C) | 电流超250mA/输入电压>6V | 检查负载电流 |

## 9. 布局小技巧——回流路径

> 画PCB时，走线是"去程"，铺铜是"返程"——回路面积越小，EMI越低。

```
回路1（导通时）：Vin→C1/C2→IN→SW→L1→C3/C4→负载→GND→回C1/C2
回路2（关断时）：L1续流→C3/C4→负载→GND→D1→回L1
```

三条实操建议：
1. 输入电容GND打多个过孔到底层
2. SW周围GND铺铜不打断
3. FB走线下方有完整GND平面
