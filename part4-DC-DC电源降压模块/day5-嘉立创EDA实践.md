## 第2步：元件布局 — 四大铁律实操

### 🥇 铁律一：输入电容紧贴芯片

**摆放顺序**：RT8289GSP → 输入电容C1/C2

```
❌ 错误（电容离得远）：
     C1/C2
       │
       │ ← 5mm以上的走线
       │
  ┌────┴────┐
  │ IN  GND │  ← 寄生电感大 → 电压尖峰
  │ RT8289GSP  │
  └─────────┘

✅ 正确（电容紧贴）：
  ┌────┐
  │C1/C2│ ← 距离≤5mm
  └─┬──┘
    ││
  ┌─┴───────┐
  │IN    GND│ ← 电容正极→IN, 负极→GND
  │ RT8289GSP  │
  └─────────┘
```

### 🥇 铁律二：BOOT自举电容紧贴BST和SW

```
    Cbst(0.1μF)
   ┌──────────┐
   │          │
   │  ① BST  │ ← 左端接BST
   │          │
   │  ⑥ SW   │ ← 右端接SW
   └──────────┘
```

**这个电容非常小（0603）**，可以直接放在RT8289GSP的①脚和⑥脚之间，走线越短越好。

### 🥇 铁律三：SW节点面积最小化

SW引脚 + 电感L1 + 二极管D1 → 三者摆成**紧凑三角形**：

```
         RT8289GSP
         ┌────┐
         │ SW │ ← ⑥
         └─┬──┘
           │ ← 极短走线
      ┌────┴────┐
      │  L1电感  │
      └────┬────┘
           │
      ┌────┴────┐
    ┌─┤ D1二极管 │
    │ └─────────┘
    │
   GND
```

**注意**：SW节点的铜箔既要**够粗走大电流**，又要**面积尽量小**减少EMI——这是一个矛盾，目标是在"够粗"的前提下做到最小。

### 🥇 铁律四：FB走线远离SW和电感区域

反馈电阻R1和R2要放在远离电感和SW的地方：

```
✅ 正确：
                          ┌── FB脚
                          │
      [远离电感的区域]     │
  Vout ───── R1 ──────┬──┤
                       │  │
                       │ R2
                       │  │
                      GND GND
   ↑ FB走线用细线(0.25mm)，不打过孔

❌ 错误：
  Vout ──── R1 ──── FB
                │
              R2 ── GND
                ↑ 走线经过电感或SW下方 → 噪声耦合
```

### 其他元件摆放

| 元件 | 摆放位置 | 原因 |
|------|---------|------|
| **C3/C4 输出电容** | 紧挨电感L1的输出端和Vout走线 | 输出滤波回路越短越好 |
| **R4/C7 补偿网络** | 靠近COMP引脚 | 走线短，噪声干扰小 |
| **C5 软启动电容** | 靠近SS引脚 | |
| **R3 上拉电阻** | 靠近EN引脚 | |
| **662K + C8/C9** | 板子右侧，**远离**RT8289GSP的电感区域 | LDO对噪声敏感，不要靠近SW和电感 |
| **输入端子J1** | 板子左侧边缘 | 接插件靠边 |
| **输出端子J2** | 板子右侧边缘（662K旁边） | 接插件靠边 |

### 完整布局参考图

```
  ┌──────────────────────────────────────┐
  │                                      │
  │ [J1] ── C1/C2 ── RT8289GSP ── L1 ──┐   │
  │ 输入          │       │    │    │   │
  │               D1(SS34) ┘    │    │   │
  │               │            C3/C4 │   │
  │               GND           │    │   │
  │                              │    │   │
  │  ┌──── 反馈电阻R1/R2 ───────┘    │   │
  │  │  (远离电感！)                 │   │
  │  │                              │   │
  │  │  C5 SS  R4/C7 COMP  R3 EN    │   │
  │  │                              │   │
  │  │         [662K + C8/C9] ── [J2]   │ ← 右侧LDO区
  │  └────────────────────────────         │
  │                                       │
  │       ▼底层：整片GND铺铜▼             │
  └──────────────────────────────────────┘
```

## 第3步：走线

### 走线宽度设置

在嘉立创EDA中设置走线宽度规则：

```
设计 → 设计规则 → 走线宽度
  ├── 最小：0.25mm
  ├── 推荐：0.5mm  
  └── 最大：2mm
```

实际走线时：

| 走线段 | 电流 | 线宽 | 在EDA中按哪个快捷键 |
|--------|:----:|:----:|-------------------|
| Vin → C1/C2 → RT8289GSP的IN | ~2A | **2~2.5mm** | 按W进入走线，Tab改宽度 |
| RT8289GSP的SW → L1 → D1 | ~2A | **2mm（短走线）** | 同上 |
| L1 → C3/C4 → Vout端子 | ~2A | **2~2.5mm** | 同上 |
| FB走线 | <1mA | **0.25mm** | 细线，远离电感 |
| EN/COMP/SS走线 | 极小 | **0.3mm** | |
| 662K的输入输出 | ~100mA | **0.5mm** | LDO电流小 |
| **GND** | 回路电流 | **大面积铺铜** | 不单独走线 |

### 走线技巧

**① 先走GND铺铜，再走信号线**

操作：`放置 → 铺铜区域` → 选择网络"GND" → 画矩形覆盖整个板子底层

**② 大电流走线用"泪滴"**

```
操作：选中焊盘 → 工具 → 泪滴
效果：焊盘和走线之间平滑过渡，电流更顺畅
```

**③ FB走线打"45度角"，不打过孔**

```
✅ 45度角走线：  ────╲───     
❌ 90度直角走线： ────┴───  ← 容易产生EMI问题
❌ FB走线打过孔： ╴╴╴●╴╴╴  ← 过孔增加噪声
```

**④ 电源走线尽量短、尽量宽**

操作提示：按 `Shift + W` 可以快速切换线宽。

## 第4步：铺铜（GND平面）

### 为什么需要铺铜？

铺铜 = 用一大片铜皮代替无数条GND走线。好处：

| 好处 | 解释 |
|------|------|
| **低阻抗** | 大面积铜皮电阻几乎为0，GND各点电位一致 |
| **散热** | 铜皮把芯片的热量传导开 |
| **屏蔽** | GND铜皮阻挡电磁干扰 |

### 铺铜操作

```
放置 → 铺铜区域
  ├── 网络：选择 "GND"
  ├── 层：顶层（Top Layer）和底层（Bottom Layer）各铺一次
  ├── 移除孤岛铜：开启（否则会有小块悬浮铜皮）
  └── 铺铜样式：网格（散热好）或 实心（阻抗低，本项目选实心）
```

### 铺铜注意事项

```
✅ 正确：一整块连续GND铜皮
  ┌──────────────────────┐
  │ GND  GND  GND  GND   │ ← 所有GND焊盘通过铜皮连在一起
  │  GND  GND  GND  GND  │
  └──────────────────────┘

❌ 错误：GND被走线切断
  ┌──────┬───────────────┐
  │ GND  │ 走线穿过       │ ← GND被切断成两块
  │      │               │
  └──────┴───────────────┘
```

> 铺铜后跑一下DRC——如果有"孤岛铜"（小块悬浮的铜皮），嘉立创EDA会自动报错，选中删除即可。

## 第5步：过孔

### 什么时候需要打过孔？

| 场景 | 打几个过孔 | 作用 |
|------|:---------:|------|
| 输入电容C1/C2的GND焊盘 | 各2~3个 | 降低GND回路阻抗 |
| RT8289GSP底部散热焊盘 | 3×3 或 4×4 矩阵 | 散热到背面GND |
| 输出电容C3/C4的GND焊盘 | 各2~3个 | 降低GND回路阻抗 |
| 662K的GND引脚 | 1~2个 | |
| FB走线 | **不打！** | 过孔增加噪声 |

### 过孔参数设置

```
嘉立创EDA默认过孔：
  内径：0.3mm
  外径：0.6mm
  
  不建议改小——立创最小支持0.3mm内径，再小要加钱
```

## 第7步：DRC检查

### 运行DRC

```
设计 → 设计规则检查（DRC） → 运行
```

### 常见DRC错误及修正

| DRC错误 | 原因 | 修正方法 |
|---------|------|---------|
| 线宽不足 | 某条走线宽度小于规则设置 | 加粗走线，或改小最小线宽规则 |
| 间距不足 | 两个网络靠太近 | 拉开间距，或加粗隔离区域 |
| 未连接的引脚 | 有引脚没走线 | 检查原理图，确认该引脚是否确实悬空 |
| 孤岛铜 | 铺铜后有小块悬浮铜皮 | 选中删掉，或开启"移除孤岛铜" |
| 丝印重叠 | 位号和文本叠在一起 | 拖动分开 |

### DRC通过标准

```
✅ 所有错误（Error）= 0
✅ 所有警告（Warning）≤ 3（且不影响功能）
✅ 板框闭合
✅ 所有网络有至少一条走线连接（除了故意悬空的）
```

## 常见布局问题排查

| 问题 | 原因 | 改板方案 |
|------|------|---------|
| 上电后输出0V | BST自举电容没焊或离太远 | 检查Cbst → 紧贴BST和SW |
| 输出纹波很大(>100mV) | FB走线靠近电感/输入电容不够 | 下次布局注意FB远离电感 |
| 输出不稳、跳动 | COMP补偿不对/电感饱和 | 换大电感/检查补偿RC值 |
| 芯片发烫 | SW节点铜皮面积太大/太小 | 散热焊盘打矩阵过孔到底层 |
| LDO输出有噪声 | 662K太靠近电感和SW | 下次板子把662K放远些 |
| 带不动1A负载 | 电感饱和电流不够 | 换≥3A饱和电流的电感 |
