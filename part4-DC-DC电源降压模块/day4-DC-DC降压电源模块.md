# 项目4 · 电源管理模块（LDO & DC-DC）

> **封装类型**：📐 全贴片（SMD）——0603阻容 + SOP-8芯片 + SMD电感 + SOT-223（LDO）  
> **难度**：★★★☆☆（难点不在焊，而在理解原理和PCB布局——画不好可能完全不工作）  
> **画板时间**：1周（约5天，每天2-3小时）  
> **核心教学目标**：理解两种稳压方案（LDO和DC-DC）的原理与区别，掌握开关电源PCB布局的铁律。  
> **在巡线小车中的角色**：电池（7.4V/11.1V）→ DC-DC降到5V（给舵机和MCU）→ LDO降到3.3V（给传感器）

---

## 1. 项目简介

前面三个项目我们都用5V电源——要么USB供电，要么稳压电源直接给。但巡线小车用的是**电池**（锂电池7.4V或11.1V），电池电压会随着电量消耗而下降（从4.2V/节逐渐降到3.0V/节），而且电压比5V高——我们需要一个**电源管理模块**，把电池电压稳定地变成我们需要的电压。

这个模块由**两大部分**组成：

| 部分 | 功能 | 用的芯片 | 在车上的作用 |
|------|------|----------|-------------|
| **DC-DC降压** | 电池(7.4~12V) → 5V | MP1584 | 给舵机、MCU、电机驱动供电（大电流） |
| **LDO降压** | 5V → 3.3V | 662K（3.3V固定输出） | 给灰度传感器等模拟电路供电（低噪声、低功耗） |

为什么要分成两级？往下看你就明白了。

## 2. 前置知识补全

> 在正式开始之前，有几个物理概念需要先搞清楚。如果你已经熟悉了，可以跳过直接看第3节。

### 2.1 欧姆定律——电学界的"F=ma"

最基本的电学公式，整个电源设计都离不开它：

**U = I × R**（电压 = 电流 × 电阻）

或者写成：
- **I = U ÷ R**（电流 = 电压 ÷ 电阻）
- **R = U ÷ I**（电阻 = 电压 ÷ 电流）

**打个比方**：电压相当于水压（推动水流的力量），电流相当于水流大小，电阻相当于水管粗细。水管越细（电阻越大），水流越小。

### 2.2 功率——"做了多少功"

**P = U × I**（功率 = 电压 × 电流），单位是瓦特（W）。

**通俗理解**：功率就是"单位时间内消耗了多少能量"。一个100W的灯泡比10W的亮，因为它每秒钟消耗的能量更多。

在电源设计中，功率就是**发热量**——一个元件上消耗的功率越大，它就越烫。

### 2.3 电容——"储电的水桶"

电容就是一个**储存电荷的元件**，可以理解成一个小水桶：

- **充电**：往水桶里装水（装电荷）
- **放电**：把水桶里的水倒出来（释放电荷）
- **容量**：水桶能装多少水，单位是**法拉（F）**，常用μF（微法，百万分之一法拉）

电容在电源电路中的两个作用：
1. **滤波**——把脉动的电压"抚平"（就像在水龙头出口放一个水桶，水流就不会忽大忽小）
2. **储能**——当电路突然需要大电流时，电容先顶上（就像跑步时突然加速，先猛吸一口气）

> ⚡ **重要性质**：电容两端的电压**不能突变**。这就是它能"抚平"电压的原因。

### 2.4 电感——"有惯性的水流"

电感是一个**绕成线圈的导线**，它在电源中扮演着关键角色。

**电感的本质**：电流流过线圈时会产生磁场，磁场会"抵抗"电流的变化。

**类比理解**：电感就像水流中的一个大**水轮机（飞轮）**——它有惯性，水流突然变大时它会阻止，水流突然变小时它会释放储存的能量维持水流。

- 开关电源就是利用电感的这个"惯性"性质来做电压转换的

### 2.5 PWM——"快速开关"

PWM = Pulse Width Modulation（脉冲宽度调制），中文叫**脉宽调制**。

```
高电平 ──┐    ┌────┐    ┌────┐    ┌────
         │    │    │    │    │    │
低电平 ──┴────┘    └────┘    └────┘    └──
         ← 周期T → ← 周期T → ← 周期T →
         ← 高电平时间 →
```

- **占空比（Duty Cycle）** = 高电平时间 ÷ 总周期 × 100%
- 占空比50%：一半时间开、一半时间关
- 占空比75%：四分之三时间开、四分之一时间关

**通俗理解**：就像水龙头的开关——你快速开关水龙头，开的时间比例越大，平均出水量就越大。DC-DC就是靠调整这个"开关比例"来控制输出电压的。

### 2.6 开环 vs 闭环——"蒙眼走路 vs 睁眼走路"

这两个概念来自**控制理论**，但理解起来很简单：

| | 开环控制 | 闭环控制 |
|------|----------|----------|
| **比喻** | 蒙着眼睛走路 | 睁着眼睛走路 |
| **怎么做** | 按预设指令执行，不管实际结果 | 不断检查实际结果，根据偏差调整 |
| **优点** | 简单、响应快 | 精确、抗干扰 |
| **缺点** | 走歪了不知道 | 复杂、可能振荡 |
| **电源的例子** | 固定占空比的开关（不稳压） | LDO和DC-DC（有反馈，稳压） |

**关键点**：LDO和DC-DC都是**闭环控制系统**——它们内部有一个"眼睛"（反馈电路）不断检查输出电压，高了就调低、低了就调高。这是下面理解稳压原理的核心。

## 3. LDO线性稳压部分

### 3.1 原理——"智能可变电阻"

LDO = Low Dropout Regulator（低压差线性稳压器）。

**工作原理用一个比喻讲清楚：**

想象你站在一个**大瀑布**下面（高电压输入），但你只需要**一杯水**（低电压输出）。

- **不聪明的做法**：直接用杯子接——会被巨大的水压冲飞
- **LDO的做法**：你用一个开关，**只开一点点缝**，让刚好一杯水量的水流下来——多余的水压全部由你的身体承受，变成热量散发掉

换个更准确的比喻——LDO内部相当于一个**自动调节的可变电阻**：

```
Vin (比如5V) ──┬── [可变电阻] ──┬── Vout (3.3V)
               │                │
               │      ↑         │
               │   [控制电路]    │
               │      ↑         │
               │   [基准电压]    │  ← 内部的"标准尺子"
               │                │
               └── GND ────── ──┴── GND
```

控制电路不断检查输出电压：
- Vout太高了 → 增大可变电阻 → 压降变大 → Vout降回来
- Vout太低了 → 减小可变电阻 → 压降变小 → Vout升回去

> 🔄 **这就是闭环控制**：测量输出 → 和基准比较 → 调整电阻 → 再测量……永远在修正，永远在稳定。

### 3.2 LDO的短板——效率问题

**核心公式**：LDO的效率 ≈ Vout ÷ Vin

**什么意思？** 如果输入5V输出3.3V，效率 = 3.3/5 = **66%**。剩下的34%去哪了？变成**热量**了。

看一个具体计算——如果电路需要500mA电流：

```
P损耗 = (Vin - Vout) × I
      = (5 - 3.3) × 0.5
      = 1.7 × 0.5 = 0.85W
```

0.85W是什么概念？手机充电的时候也就发热几瓦——这0.85W已经能让一个小芯片烫手了。

所以LDO适合**小电流、低压差**的场景。如果压差大、电流大，用LDO就像用一个大功率电暖器——大部分电都变热了，浪费。

### 3.3 LDO的关键参数

| 参数 | 含义 | 白话解释 |
|------|------|----------|
| **Vdropout（压差）** | 输入电压至少比输出高多少才能工作 | 普通LDO要1~2V，低压差LDO只要0.1~0.3V |
| **Iout（输出电流）** | 最大能输出多大电流 | 至少留30%余量 |
| **PSRR** | 对输入噪声的抑制能力（单位dB） | 数值越大，输出越"干净" |
| **Iq（静态电流）** | LDO自己消耗的电流 | 电池供电要选小的（<10μA） |

### 3.4 典型电路——662K（3.3V LDO）

662K 是一颗非常小巧的 3.3V 固定输出 LDO，SOT-23-3 封装（只有三个小小的引脚，像三极管一样）。

**为什么选它而不是 AMS1117？**

| 对比 | AMS1117-3.3 | 662K |
|------|-------------|------|
| 封装 | SOT-223（较大） | **SOT-23-3**（极小，像三极管） |
| 输出电流 | 1A | 250mA（给传感器够用了） |
| 静态电流 | ~5mA | **25μA**（省电200倍！） |
| 最低压差 | ~1V | **~0.2V**（低压差性能好） |
| 输入电压范围 | ≤15V | **≤6.0V**（只能接5V，不能直接接电池！） |
| 所需电容 | 10μF + 100nF 各两个 | **两个1μF就够了** |
| 价格 | ~0.3元 | **~0.08元** |

> 💡 **关键限制**：662K的**最高输入电压只有6.0V**，所以不能直接从电池（7.4V）供电，必须接在DC-DC的5V输出后面。但这正好就是我们设计的两级架构！

**引脚定义（SOT-23-3）：**

```
    俯视图（看芯片正面，有字朝上）
         __
        /  \
    ┌──┤662├──┐
    │  \__/   │
    │         │
  ① GND  ② Vout  ③ Vin
```

- **① GND** —— 地（接GND）
- **② Vout** —— 3.3V输出（接负载）
- **③ Vin** —— 5V输入（接DC-DC的5V输出）

**完整电路图：**

```
               ┌──────────┐
 5V (来自Buck) ───┤③ Vin    │
                   │         │
  C1 ──┤├──┬──────┤① GND    │
  1μF     │      │         │
          │      │② Vout ──┼──┬── C2 ──┤├── GND
         GND     └──────────┘  │   1μF
                               │
                          Vout (3.3V)
```

**就是这么简单！**——只需要输入一个1μF、输出一个1μF，两个陶瓷电容就搞定了。

| 元件 | 作用 | 为什么选1μF |
|------|------|------------|
| C1 (1μF 输入) | **输入稳压**——防止输入电压抖动 | 662K内部已经有稳压，外部1μF就够了 |
| C2 (1μF 输出) | **输出滤波**——让3.3V更平滑 | 传感器电流小，1μF足够滤除噪声 |

> 💡 **为什么662K只需要两个小电容？** 因为它内部有较好的电源抑制能力，且工作电流小（传感器总共不到100mA），不需要像AMS1117那样用大电解电容储能。这也是小封装LDO的优势——省空间、省元件。

### 3.5 LDO的PCB布局要点

662K（SOT-23-3封装）的布局**比DC-DC简单太多**了——毕竟没有高频开关，不用担心EMI。但有几个小要点：

- **电容要紧贴芯片引脚**——C1（输入1μF）放在Vin和GND之间，距离<5mm；C2（输出1μF）放在Vout和GND之间
- **走线不用太宽**——传感器电流小（<100mA），0.3mm线宽就够
- **不需要特殊散热处理**——功耗极小（5V→3.3V，100mA时只有0.17W），SOT-23-3封装自己能搞定

## 4. DC-DC降压（Buck）部分

### 4.1 原理——"高速水龙头"

DC-DC降压的核心拓扑叫 **Buck**（降压），也叫**降压开关电源**。

**用两个比喻讲清楚：**

**比喻一（水龙头版）**：
想象你有一个高压水龙头（电池），想要稳定的低压水流（5V）。不聪明的做法（LDO）是把水龙头只开一点点，多余的压力变成热量。聪明的做法（Buck）是：**快速开关水龙头**——每秒开关150万次！每次只开一小下，然后让水流过一个"水轮机"（电感），水轮机的惯性把脉冲水流"抚平"成平稳的低压水流。

**比喻二（搬运工版）**：
搬运工要把一个重物从一楼搬到三楼。LDO的做法是：一次搬上去，累得半死（发热）。Buck的做法是：每次只搬一小段路，但来回跑很多次，每次歇在中间的平台上（电感储能），虽然跑得勤快，但一点都不累（效率高）。

**Buck电路的四个核心元件：**

```
Vin ──→ [开关管 SW] ──→ [电感 L] ──→ Vout
              │              │        │
              ├──→ [续流二极管] ←──── ┤
              │   (D)                  │
             GND              [输出电容 Cout]
                                     │
                                    GND
```

这四个元件各司其职，我们来逐一理解：

| 元件 | 在水龙头比喻中 | 在电路中的实际作用 |
|------|---------------|-------------------|
| **开关管S** | 水龙头的开关 | 以极快速度（1.5MHz）接通和断开，把直流切成脉冲 |
| **电感L** | 水轮机（有惯性） | 开关接通时储存磁能，开关断开时释放磁能，维持电流 |
| **续流二极管D** | 回水管道 | 开关断开时，给电感电流提供一条回路"继续流" |
| **输出电容C** | 大水桶 | 把脉冲电流"抚平"成平稳的直流 |

### 4.2 Buck的工作过程（看懂这一节就懂了Buck）

**阶段一：开关管导通（T_on）**

```
Vin → [开关管·开] → [电感 L] → Vout（电流从Vin流向Vout）
                      ↓
                  电流增大，电感储存磁场能量
                      ↓
                  续流二极管·截止（反偏）
```

- 输入电流流过电感
- 电感中的电流**逐渐增大**（因为电感会抵抗电流的变化，所以不会瞬间变大）
- 电感在储存能量（以磁场的形式）
- 续流二极管此时是反向偏置（阳极接GND，阴极接高电位），所以不导通

**阶段二：开关管关断（T_off）**

```
Vin → [开关管·断] ─ ─ [电感 L] → Vout（电感还在"推"电流）
                         ↑  ↓
                         │ 电流继续流，逐渐减小
                         │  电感释放磁场能量
                         │  ↓
                      [续流二极管·导通]
                           ↓
                         GND（电流从GND流过二极管再到电感）
```

- 开关管断开，但**电感的惯性**让它不想让电流停止
- 电感说："不行！电流不能断！"——它开始释放之前储存的磁场能量
- 电流继续从**GND → 续流二极管 → 电感 → Vout**的方向流动
- 这个阶段电流逐渐减小

**阶段三：下一个周期开始……**

Buck就这样以极高的速度（1.5MHz，即150万次/秒）重复"导通→关断→导通→关断……"

**输出电压由什么决定？**

输出电压 = 输入电压 × 占空比

**Vout = Vin × D**

其中 D = T_on ÷ (T_on + T_off) —— 即"开的时间占整个周期的比例"

> **举个例子**：输入电压12V，如果占空比=50%，理论输出=12×0.5=6V。但实际因为二极管压降和电感损耗等，会低一些，所以实际公式是 Vout = Vin × D - V_diode。

在MP1584中，实际是**通过反馈闭环控制**来精确调节占空比的：
- 检测输出电压 → 与内部基准（0.8V）比较 → 调整占空比 → 精确输出目标电压

> 🔄 **这也是闭环控制！** 和LDO一样，Buck也有反馈环路。只是LDO通过"调整电阻"来稳压，Buck通过"调整开关时间比例（占空比）"来稳压。

### 4.3 LDO vs Buck——"一静一动"

| 对比项 | LDO（线性稳压） | Buck（开关稳压） |
|-------|----------------|-----------------|
| **核心原理** | 可变电阻分压 | 高速开关+电感储能 |
| **效率** | 低（Vout/Vin） | 高（85~95%） |
| **发热** | 大 | 小 |
| **输出噪声** | 极低（μV级） | 有纹波（mV级） |
| **电路复杂度** | 简单（几个电容就行） | 复杂（电感+二极管+布局讲究） |
| **成本** | 低 | 较高 |
| **PCB布局难度** | 简单 | 难（高频开关的EMI控制） |

**什么时候用LDO，什么时候用Buck？**

```
输入和输出之间的压差大吗？（比如7.4V→5V，压差2.4V）
  │
  ├── 压差小（<1V）且电流小（<500mA）→ LDO（简单、干净、够用）
  │   例：5V→3.3V给传感器供电  → 662K（SOT-23-3小巧LDO）
  │
  └── 压差大（>1V）或电流大（>500mA）→ Buck（高效、不发热）
      例：7.4V→5V给MCU和舵机 → MP1584
```

### 4.4 巡线小车的完整供电架构

```
锂电池组（7.4V / 2S）
     │
     ▼
 ┌──────────┐
 │  DC-DC   │  MP1584
 │  降压    │  → 5V / 2A
 └──────────┘
     │
     ├──────────────────┐
     │                  │
     ▼                  ▼
  舵机供电           MCU供电
  (5V, ~500mA)      (5V → 板载LDO → 3.3V)
                     
                     还要再降一次：
                     ┌──────────┐
                     │  LDO     │  662K
                     │  5V→3.3V │  → 3.3V / 250mA（给传感器）
                     └──────────┘
                          │
                          ▼
                    灰度传感器等
                    （模拟电路需要干净的电源）
```

**为什么不是直接从电池降到3.3V？**

因为**压差越大，LDO效率越低、发热越厉害**：
- 7.4V→3.3V：效率=3.3/7.4=**44.6%**，一半多的电能变成热量
- 5V→3.3V：效率=3.3/5=**66%**，好多了

所以先用**高效的DC-DC**从电池降到5V，再用**低噪声的LDO**从5V降到3.3V——两级降压，既高效又干净。

## 5. MP1584降压电路详解

### 5.1 MP1584芯片简介

MP1584是一颗经典的SOP-8封装Buck芯片：

| 参数 | 值 | 说明 |
|------|-----|------|
| 输入电压 | 4.5~28V | 覆盖2S~6S锂电池 |
| 输出电流 | 最大3A | 够用（留30%余量） |

### 5.4 输出电压设定——反馈电阻计算

MP1584的内部基准电压是**0.8V**。FB引脚会努力把电压维持在0.8V。

```
Vout = 0.8 × (1 + R1/R2)

R1 = 上分压电阻（FB到Vout）
R2 = 下分压电阻（FB到GND）
```

**算一算：要输出5V**

选R2=10kΩ（一个常见值）：
```
R1 = (5/0.8 - 1) × 10k = (6.25 - 1) × 10k = 5.25 × 10k = 52.5kΩ
```

电阻没有52.5kΩ这个标称值——最接近的是 **52.3kΩ（1%精度）**，或者用51kΩ+1.5kΩ串联。

> 💡 **为什么R1、R2要用1%精度电阻？** 反馈电阻的精度直接决定输出电压的精度。用5%的电阻，输出可能偏5%——也就是5V可能变成4.75V或5.25V。对于要给MCU供电的电路，这不太安全。

### 5.5 电感选型

电感是Buck电路最重要的元件之一。它的值决定了电流的"纹波"大小（电流波动的幅度）。

**不用自己算——直接看数据手册推荐值：**

| 输入电压 | 输出电压 | 推荐电感值 |
|---------|---------|-----------|
| 7~12V | 5V | **10~22μH**（推荐15μH） |
| 12~24V | 5V | 22~33μH |
| 7~12V | 3.3V | 6.8~10μH |

还有一个重要的参数：**饱和电流**。

电感的饱和电流必须大于电路中的**峰值电流**。一般取输出电流的1.3倍以上：
- 输出2A → 电感饱和电流 ≥ 2.6A
- 本项目选 **10~15μH / 3A** 的SMD功率电感（数据手册5V输出推荐15~22μH，我们选10~15μH兼顾小体积和低纹波）

**封装**：CD54（5.8×5.2mm）或 CD75（7.3×7.3mm）。尺寸越大，饱和电流越大、内阻越小。

## 6. 开关电源PCB布局——四大铁律

> ⚠️ **非常重要**：这是整个教程中最关键的PCB布局章节。开关电源的PCB画不好，电路可能完全不工作、输出纹波巨大到影响后续电路、甚至烧芯片。

为什么开关电源的布局这么讲究？因为**1.5MHz的高频开关**带来了两个问题：
1. **EMI（电磁干扰）**——高频开关产生电磁波，会干扰板上其他电路
2. **寄生电感**——走线不是理想导线，高频下走线电感会产生电压尖峰

### 铁律一：输入电容必须紧贴芯片

```
                    ┌──────┐
       ┌────────────│ C1/C2 │← 输入电容（22μF×2）
       │            └──────┘
       │          ╱ │     │ ╲
       │         ╱  │     │  ╲
       │        ╱   │     │   ╲
       │       ▼    ▼     ▼    ▼
       │   ┌─────────────────────┐
Vin ───┤───│ IN              GND │──── MP1584
       │   │   MP1584            │
       │   │ SW                  │
       │   └─────────────────────┘
       │           │
       └───────────┘
```

**输入电容（C1/C2）的正极 → MP1584的IN脚**  
**输入电容（C1/C2）的负极 → MP1584的GND脚**  

距离越短越好——最好 **≤5mm**。走线要宽、要短。

**为什么？** 每次开关管导通时，输入电容要在极短时间内提供一个大电流尖峰。如果电容离得远，走线的寄生电感会让IN脚产生电压尖峰——可能直接击穿芯片。

> 说白了：这就像消防车救火——如果水管（走线）太长，水压（电压）还没到火场就损失了。电容要像消防员一样站在火场边（紧贴芯片引脚）。

### 铁律二：SW节点面积最小化

SW引脚连接三个东西：电感L1、续流二极管D1、MP1584的SW脚。这三个元件应该**挨着放**，走线短而粗。

但矛盾的是——SW节点的**铜箔面积要尽量小**（但走线又要够粗走大电流）。

**为什么？** SW节点的电压在0V和Vin之间以1.5MHz的频率高速跳变，它就像一个**微型无线电发射天线**——铜箔面积越大，辐射的电磁干扰越强。

**怎么做？**
- 电感L1紧挨SW引脚
- 二极管D1紧挨SW引脚
- 三者构成一个紧凑的三角形，面积尽可能小

### 铁律三：反馈走线远离SW节点和电感

FB引脚检测的是输出电压，信号很微弱（和0.8V基准比较）。如果FB走线靠近SW或电感，开关噪声会耦合到FB上→输出电压抖动。

**FB走线的三条规则：**

```
✅ 正确做法：
                        R1(52.3k)
Vout ─────────────────────┬──┬─── FB引脚
                          │  │
                          │  R2(10k)
                          │  │
                         GND GND
          ↑      ↑
    走线细(0.25mm)  走线在远离电感的区域

❌ 错误做法：
Vout ──┬── R1 ──┬── FB
       │        │
       │     R2 ── GND
       │
       ╲   ╲
        ╲   ╲  ← 走线经过电感或SW下方！
```

1. FB走线用细线（0.25mm就够，电流<1mA）
2. 走线远离电感和SW区域——不要在电感正下方走
3. R2靠近FB引脚放，让FB到芯片的走线尽量短

### 铁律四：底部完整连续地平面

在MP1584下方保持**完整、连续的地平面**（GND铺铜）。

**具体要求：**
- 芯片底部的散热焊盘必须焊接到地平面（散热+电气连接）
- 输入电容的GND、输出电容的GND、二极管的GND，都通过大面积铺铜低阻抗连通
- 不要在电源部分用"星形接地"——一整块连续的地铜皮才是最好的

  [输入端子] ── C1/C2 ── MP1584 ── L1 ── C3/C4 ── [输出端子]
                  │          │        │
                  └── D1 ────┘        │
                  (SS34)              │
                                      │

## 8. 焊接注意事项

### DC-DC部分的难点

**MP1584的SOP-8 EP封装**——除了8个引脚，底部还有一个大散热焊盘。

**焊接方法一（热风枪法）**：
1. 在PCB的散热焊盘上涂锡膏或预上锡
2. 放上芯片，用热风枪（~350°C，风速适中）从上方吹
3. 焊锡熔化后芯片会自动"下沉"——完成底部焊接
4. 再焊接8个引脚（用烙铁拖焊）

**焊接方法二（背面灌锡法—没有热风枪时）**：
1. 在PCB的散热焊盘位置打一个直径1mm的大过孔
2. 先焊好8个引脚
3. 从PCB背面通过过孔灌锡进去

**SS34二极管**——注意极性！**阴极（有横线标记的一端）**接SW或电感端，**阳极**接GND。焊反了电路会短路！

**电感**——不区分正负极，但焊接要牢固（电流大，虚焊会发热）。

### LDO部分的注意事项

**662K（SOT-23-3封装）**——只有三个小引脚，焊接方法：
1. 先给一个焊盘上锡
2. 用镊子夹着芯片放上去，对准引脚
3. 焊上那个预上锡的引脚固定位置
4. 焊剩下的两个引脚
5. **检查：用万用表通断档确保相邻引脚之间没短路**（SOT-23-3引脚间距很小，容易连锡）

## 10. 常见问题排查

| 现象 | 可能原因 | 先查什么 |
|------|---------|---------|

## 12. 开环vs闭环·电源视角总结

LDO和Buck都用了**闭环控制**（反馈），这是它们能稳定输出的根本原因。

但有趣的是——在整个教程中，不同模块用了不同的控制方式：

| 项目 | 什么控制方式 | 为什么 |
|------|-------------|--------|
| **项目1: NE555流水灯** | **开环**——NE555产生固定频率，CD4017计数，不管LED亮没亮 | 不需要知道结果，灯肯定会亮 |
| **项目2: 电机驱动** | **开环**——给PWM占空比，假设电机转到了对应的速度 | 巡线时由传感器反馈修正，电机本身不需要闭环 |
| **项目3: 灰度传感器** | **开环**——红外发射→接收→比较器输出，没有调节回路 | 传感器只负责"感知"，不负责"调整" |
| **本项目: 电源管理** | **闭环**——反馈电阻检测输出，调整占空比/电阻 | **不闭环就无法稳压！** 没有反馈就没有稳定电压 |
| **项目5: MCU最小系统** | 既有开环（跑程序）也有闭环（ADC+PID） | MCU本身不控制，它运行的程序可以控制 |

> 🔑 **记一句话**：**需要准确输出 → 闭环；只需要按流程走 → 开环。** LDO和Buck的反馈回路，是你在本教程中遇到的**第一个、也最重要的闭环控制系统**。

## 13. 拓展思考

### 13.1 如果只用LDO会怎样？

假设巡线小车用7.4V电池，需要5V/2A和3.3V/500mA：
- 7.4V→3.3V（LDO）：P=(7.4-3.3)×0.5=**2.05W**——小芯片热得可以煎鸡蛋了
- 7.4V→5V（LDO）：P=(7.4-5)×2=**4.8W**——需要加很大的散热片，电池续航只有Buck的60%

所以大压差、大电流下**必须用Buck**。

### 13.2 同步Buck vs 异步Buck

MP1584是**异步Buck**——续流用外部肖特基二极管。二极管有约0.3~0.5V的压降，会降低效率。

更先进的是**同步Buck**——续流用内部的一个MOSFET（同步整流管），导通电阻只有几十mΩ（≈0.01V压降），效率更高（>95%）。

为什么本项目选异步Buck？因为**二极管好理解、好观察**——你可以用示波器看到二极管两端在开关切换时的电压变化，学习价值更大。

### 13.3 输出纹波和ESR

输出纹波主要由两部分决定：
1. **电感纹波电流** × 电容**ESR**（等效串联电阻）
2. 开关切换瞬间的电压尖峰

MLCC（多层陶瓷电容）的ESR极低（几mΩ），所以纹波小。但MLCC有个问题：**DC Bias特性**——加上直流电压后，实际容量会下降（比如25V耐压的22μF电容在12V下可能只剩~10μF）。所以选电容时要留余量。

> **项目4完成标准**：输入7~12V，DC-DC输出稳定5V（±0.1V），LDO输出稳定3.3V（±0.1V），带载1A不掉电压，纹波<100mV。

## 🧩 拓展延伸 — 小故事

### 🧨 一次电容爆炸的教训

2000年代初，一家消费电子公司在量产一款新型电视机顶盒时，遇到了一个诡异的问题：**机顶盒在用户家里使用几周后，电源就会烧掉。** 而且故障率高达15%。

工程师拆开分析发现：**输入电容的容量几乎为零。** 电容外观完好，但用万用表一测，标称22μF实际只剩不到2μF。

原因是什么？**DC Bias效应**——MLCC（多层陶瓷电容）在加上直流电压后，实际容量会急剧下降。这批电容用的是Z5U材质，在12V电压下实际容量只剩标称的10%。

但设计电路时工程师是按照22μF来算的，实际只有2.2μF——纹波电流远超预期，电容过热→慢慢干涸→最终炸裂。

> 这就是为什么我们的BOM里选的是**X5R或X7R材质**的MLCC（耐DC Bias能力强），并且耐压值留了1.5倍余量——这是前人用炸电容换来的经验。

### 🛸 阿波罗登月舱的电源难题

1969年，阿波罗11号登月。但很少有人知道，登月舱的电源设计差点让整个项目泡汤。

登月舱需要一套既轻便又可靠的电源系统。当时的方案有两个：
- **方案A**：用线性稳压——简单可靠，但效率低，需要更大的散热片，**重量超标20%**
- **方案B**：用开关稳压——效率高、体积小，但1960年代的开关电源可靠性很差，**容易烧**

NASA最终选择了两个方案**同时用**：关键系统（导航计算机）用线性稳压保障可靠性，非关键系统（照明、通讯）用开关稳压减重。这套"混搭"方案让登月舱的重量刚好控制在预算内。

> 50多年后的今天，开关电源的可靠性已经远远超过线性电源。但这段历史告诉我们——**选LDO还是Buck，本质上是在可靠性、效率、体积之间做权衡，半个世纪前如此，今天依然如此。**
