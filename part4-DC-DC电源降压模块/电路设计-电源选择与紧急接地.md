# 电路设计：电源选择开关 + 紧急接地按钮

> **功能**：用滑动开关在电路A和电路B之间切换供电，按下按钮时两个电路同时接地（紧急停止/放电功能）。

---

## 功能说明

| 操作 | 效果 |
|:----|:------|
| 滑动开关 → 位置1 | 电源接入**电路A**，电路B断电 |
| 滑动开关 → 位置2 | 电源接入**电路B**，电路A断电 |
| **按下按钮**（任意位置） | **电路A和电路B同时接地**，即时断电 |

---

## 完整电路图

```
                     滑动开关（SPDT）
                   ┌──────────────────┐
                   │                  │
      电源正极 ────┤ 公共端(COM)      │
       Vin         │                  │
                   ├── 位置1 ──────── 电路A
                   │                  │
                   └── 位置2 ──────── 电路B
                   │                  │
                   └── 位置中间(悬空) ─ 两个都没电


         按钮（常开型，按下导通）
            ┌────┐
            │    │
            │  ○  │
            └──┬─┘
               │
               ├── D1(1N4148) ──── 电路A
               │     阴极←─────── 阳极
               │                   
               └── D2(1N4148) ──── 电路B
                     阴极←─────── 阳极
               │
               ▼
              GND（地）
```

## 元件清单

| 元件 | 数量 | 说明 |
|:----|:----:|:-----|
| SPDT滑动开关 | 1 | 单刀双掷，3个引脚（公共端+两个位置） |
| 常开按钮 | 1 | 按下导通，松开断开 |
| 1N4148二极管 | 2 | 隔离二极管，防止电流倒灌 |
| 10kΩ电阻 | 2 | 电路A和电路B的下拉电阻（可选） |

## 工作原理

### 模式一：滑动开关选择供电

```
电源正极 ──→ 滑动开关公共端

位置1：电源 → 电路A
       电路B ← 无电（悬空）

位置2：电源 → 电路B
       电路A ← 无电（悬空）
```

### 模式二：按钮按下→全部接地

```
按下按钮 → 按钮导通 → D1阳极 → D1阴极 → 电路A（接地）
                    → D2阳极 → D2阴极 → 电路B（接地）

无论滑动开关在哪个位置，两个电路都被拉到GND
```

### 二极管D1/D2的作用

如果没有二极管，按下按钮时：

```
按下按钮 → GND直接接到了电路A和电路B
         ↕
         如果此时滑动开关在位置1（电源正在给电路A供电）
         → 电源正极直接通过按钮到GND → **短路！烧电源！**
```

加了二极管后：
```
按下按钮 → 电路A → D1 → GND  ✓ 正常接地
           电路B → D2 → GND  ✓ 正常接地

但电源正极 → 滑动开关 → 电路A → D1 → GND
           ↑ 二极管反向截止！电流过不去！ ✓ 不会短路
```

## 波形示意

```
滑动开关在位置1（电路A供电）：

电路A电压 ────────|─── 供电中
                  |
电路B电压 ────────| 0V

         ← 按下按钮 →
电路A电压 ────────| 立刻降到0V
                  |
电路B电压 ────────| 立刻降到0V
                  |
         ← 松开按钮 →
电路A电压 ────────| 恢复供电
电路B电压 ────────| 保持0V
```

## 实际应用场景

| 场景 | 电路A | 电路B | 按钮作用 |
|:----|:-----|:-----|:---------|
| 双电机控制 | 左电机 | 右电机 | 急停，两个电机同时断电 |
| 信号源切换 | 信号源1 | 信号源2 | 短路所有信号到GND（静音） |
| 电池放电 | 电池组1 | 电池组2 | 同时短路放电（安全保护） |
| LED灯组 | 红灯组 | 蓝灯组 | 全部熄灭 |

---

> **提示**：如果被控电路电流较大（>100mA），1N4148二极管的电流不够，需要换成 **1N4007**（1A）或 **SS34**（3A）。
