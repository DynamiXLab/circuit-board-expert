## 2. 原理图绘制

### 2.1 新建工程

1. 打开立创EDA专业版
2. 新建工程 → 命名 `流水灯_NE555`
3. 左侧工程树会出现一个 `.eprj` 工程文件和空白原理图页

【图片】立创EDA新建工程界面截图

### 2.2 放置元件

立创EDA的元件搜索方式：左侧"元件库"面板 → 搜索框输入关键字。

| 要找的元件 | 搜索关键词 | 选择技巧 |
|-----------|-----------|---------|
| NE555 | `NE555` | 选 **DIP-8** 封装 |
| CD4017 | `CD4017` | 选 **DIP-16** 封装，"CD4017BE"是德州仪器的 |
| 电阻 | `RES` 或 `电阻` | 选 "R AXIAL-0.3" 系列（通用的插件电阻封装） |
| 电解电容 | `电解电容` | 选 "CAP-DIP" 封装，脚距2.54mm或5.08mm |
| 瓷片电容 | `瓷片电容` | 选 "CAP-DIP" 或 "C 插件" |
| LED | `LED 3mm` | 选 "LED-TH-3mm" 红色 |
| 排针 | `排针 1x2` | 选 2.54mm间距 |

【图片】立创EDA元件库搜索界面截图

**操作**：双击搜索结果中的元件，它就会粘在光标上，再点击画布放置。

【图片】放置元件到画布的操作GIF动图

### 2.3 连接导线

- 快捷键 **W**：进入连线模式
- 点击起点引脚，拖动到终点引脚，再点击完成
- 两条线交叉处如果有点 = 连接，没有点 = 交叉但不导通
- 可以用 **网络标签（Net Label）** 代替长距离走线——同名标签自动连通

【图片】连线操作演示GIF

### 2.4 原理图检查（ERC）

菜单 → 设计 → ERC检查。ERC会帮你检查：
- 有没有引脚悬空忘了接
- 有没有不该短接的地方短接了
- 电源网络是否正确

**ERC通过 = 原理图没有电气错误。** 有报错就按提示改。

## 3. PCB布局布线

### 3.1 原理图转PCB

菜单 → 设计 → 转为PCB → 在弹出的对话框中确认。立创EDA会自动生成一个PCB文件，所有元件带着封装出现在板框旁边，它们之间有飞线（鼠线）显示该连哪些引脚。

【图片】原理图转PCB后的初始界面（元件散落在板框外，有飞线）

### 3.2 PCB制造约束速查表

> 在开始布局布线之前，必须先知道嘉立创**能做什么、不能做什么**。

| 参数 | 嘉立创最低标准 | **建议设计值（新手）** |
|------|:----------:|:------------------:|
| 最小线宽 | 5mil (0.127mm) | **≥ 8mil (0.2mm)** |
| 最小线距 | 5mil | **≥ 8mil** |
| 最小过孔内径 | 0.2mm | **≥ 0.3mm** |
| 最小过孔外径 | 0.45mm | **≥ 0.6mm** |
| 焊环宽度 | ≥ 0.125mm | **≥ 0.15mm** |
| 最小字符线宽 | 0.15mm | 0.15mm |
| 板子尺寸 | — | **≤ 10cm×10cm**（免费打样！） |

**铜厚与载流能力（1oz铜厚，温升10°C时）：**

| 走线宽度 | 约可通过电流 |
|:--------:|:----------:|
| 0.25mm (10mil) | ~0.5A |
| 0.5mm (20mil) | ~0.8A |
| 1.0mm (40mil) | ~1.5A |
| 2.0mm (80mil) | ~3A |

> 本项目信号线（<50mA）用 0.3mm 够，电源线（<500mA）用 0.5mm。

### 3.3 布局

**布局顺序：先摆大件，再摆小件，最后调细节。**

**十大布局原则（本项目用到的标了✅）：**

| # | 原则 | 说白了就是 | 本项目 |
|---|------|-----------|:------:|
| 1 | **先大后小** | 先放连接器、IC座，再塞小电阻电容 | ✅ |
| 2 | **模块化布局** | 同一个功能的元件聚在一起——NE555的阻容跟着NE555，CD4017的LED跟着CD4017 | ✅ |
| 3 | **电源优先** | 电源入口、滤波电容先占好位置（靠板边） | ✅ |
| 4 | **去耦电容就近** | IC的电源脚旁边的0.1μF电容，距离≤5mm | ✅ |
| 5 | **信号流向** | 信号从左到右——NE555(左)→CD4017(中)→LED(右) | ✅ |
| 6 | **模拟/数字分离** | 高速数字电路远离模拟信号区——本项目全是数字信号，不用管 | — |
| 7 | **发热元件分散** | 大功率管、稳压IC不要挤在一起——本项目没有发热大户 | — |
| 8 | **接插件靠边** | 电源排针、DC座放在板边 | ✅ |
| 9 | **晶振紧贴MCU** | 晶振走线要短、等长——本项目没MCU | — |
| 10 | **预留走线通道** | 元件之间不要太挤，留≥0.3mm给走线通过 | ✅ |

**实际操作**：
1. 先把两个IC座摆好（比如NE555在左，CD4017在右中）
2. 10个LED排成一排（或两排交错、或圆形）——取决于你想让流水灯长什么样
3. 电阻放在LED和CD4017之间（靠近LED，和LED对齐）
4. 电源排针、电解电容放在板边
5. 调整到飞线交叉最少的状态

【图片】PCB布局示例（标注各模块位置：NE555区域、CD4017区域、LED阵列、电源区）

> "布局70%，布线30%"——好的布局让布线变得轻松，差的布局让你怎么走都走不通。

### 3.4 布线

**板层**：双层板（Top Layer + Bottom Layer）

| 走线类型 | 推荐宽度 | 原因 |
|----------|---------|------|
| 信号线（LED、时钟） | 0.3mm（12mil） | 电流小于50mA |
| 电源线（VCC 5V） | 0.5mm（20mil） | 给所有LED供电，可能数百mA |
| 地线 | 0.5mm | 同上 |

**布线技巧**：
- 优先布顶层（Top Layer），走不过去再切换到底层——用**过孔（Via）**切换
- 尽量走直线、45°角拐弯，不要直角
- 两条线之间保持至少 0.25mm 间距
- 不需要把所有线布完再处理地——地可以用**铺铜**搞定（见下一节）

【图片】PCB布线完成示例（顶层+底层分别截图）

### 3.5 铺铜

铺铜就是在PCB上大面积覆盖铜皮，通常连接GND网络：
- 好处1：所有GND自动连通，不用手动拉很多地线
- 好处2：大面积铜皮提供低阻抗回流路径，降低干扰
- 好处3：省蚀刻液（环保 😄）

**操作**：PCB工具栏 → 铺铜 → 画一个矩形覆盖整个板子 → 网络选 "GND" → 顶层一次、底层一次 → 确认。

【图片】铺铜操作GIF动图

### 3.6 DRC（设计规则检查）

菜单 → 设计 → DRC检查。如果通不过，按照报错逐个修改（线太近、焊盘和线距离不够等）。

**最低规则设置**（嘉立创双层板标准工艺）：
- 最小线宽：6mil（0.152mm），我们画 0.3mm 远远够
- 最小间距：6mil
- 最小过孔：0.3mm内径 / 0.6mm外径

> ✅ **DRC全绿 = PCB可以下单了。**

### 3.7 丝印调整

丝印是PCB上的白色文字，用来标注元件位置和接口功能：

- 元件编号（R1、R2、LED1...）不要被元件本体挡住
- 电源接口标注 "+" "GND"
- 板子上写上项目名、日期、你的名字（仪式感！）

## 🎯 拓展延伸：PCB的发明——一个"印刷"出来的革命

你可能会好奇：为什么电路板叫"印刷电路板"（**Printed** Circuit Board）？

因为在1920年代以前，电子设备里的所有连接都是**一根一根电线手工焊的**——收音机内部像鸟窝一样乱。到了二战期间，军用电台和雷达需要大量生产，手工接线效率太低。

一位奥地利工程师 **Paul Eisler** 在1936年提出了一个疯狂的想法：**把电路"印"在绝缘板上**——就像印报纸一样。他借用了印刷行业的工艺：先在铜箔上印抗腐蚀油墨（就是你要画的"走线"），然后用酸把没印油墨的铜腐蚀掉，留下的就是电路。

这个工艺一直用到今天——你现在用的立创EDA，本质上还是 Eisler 当年那个思路的数字化版本。

> 🖨️ 你今天对着屏幕画的每一条线，工厂会用激光或化学蚀刻的方式"印"到铜板上——和1936年的原理一模一样。
