## 2. 拓展思考（选读）

> 以下内容不要求掌握，是给想知道"为什么"的同学准备的。

### 2.1 NE555内部结构

NE555内部有三个 5kΩ 电阻（所以叫555）串联成分压器，把VCC分成 1/3 VCC 和 2/3 VCC 两个参考电压。此外还有两个比较器（上比较器检测 >2/3VCC，下比较器检测 <1/3VCC）、一个RS触发器、一个放电三极管。

<img src="image/day5-NE555内部框图.png" alt="NE555内部结构框图" style="max-width:100%;">

**多谐振荡器的工作过程**：

```
1. 上电时C1电压为0 → 下比较器输出使触发器置位 → OUT=高 → 放电管关断
2. 电源通过R1+R2给C1充电 → C1电压上升
3. 当C1电压 > 2/3 VCC → 上比较器使触发器复位 → OUT=低 → 放电管导通
4. C1通过R2和放电管放电 → C1电压下降
5. 当C1电压 < 1/3 VCC → 下比较器再置位 → 循环回到步骤1
```

| 阶段 | 路径 | 经过的电阻 |
|------|------|:---:|
| 充电 | VCC → R1 → R2 → C1 → GND | R1 + R2 |
| 放电 | C1 → R2 → 放电管 → GND | 仅 R2 |

<img src="image/day5-NE555充放电路径.png" alt="NE555多谐振荡器充放电路径示意图" style="max-width:100%;">

### 2.2 频率公式推导

- 充电时间：T_high = 0.693 × (R1+R2) × C1
- 放电时间：T_low = 0.693 × R2 × C1
- 周期：T = T_high + T_low = 0.693 × (R1+2R2) × C1
- 频率：f = 1/T = **1.44 / ((R1+2R2) × C1)**

> 0.693 来自 ln(2)，和RC电路的指数充放电特性有关。想知道为什么是 ln(2) 的，去学一下RC电路的一阶微分方程——这是大一下学期电路分析的内容。

### 2.3 CD4017内部原理

CD4017内部是一个5级约翰逊计数器（5个D触发器级联）加一个译码器，把5位二进制状态译成10个输出。

它的计数序列是：
```
00000 → 10000 → 11000 → 11100 → 11110 → 11111 → 01111 → 00111 → 00011 → 00001 → 00000 → ...
```

经过译码后每个状态对应一个 Q 输出为高，所以任何时候**有且仅有一个**LED亮。

## 🎯 拓展延伸：嘉立创——从华强北柜台到全球PCB巨头

你昨天一键下单的嘉立创（JLCPCB），总部就在深圳。它的故事很有意思。

2006年，丁氏兄弟在深圳华强北开了一家做PCB打样的小柜台——就是一个几平米的档口，帮人拼板、拼单。当时的PCB打样又贵又慢：做5片小板子要几百块，交期一两周。

他们发现了一个痛点：**工程师和爱好者只需要几片测试板，但工厂不愿意接小单**。于是他们做了几件事：
- 把大量小订单拼在一张大板上一起生产（拼板）
- 自建工厂，自动化报价
- 推"5片5块钱包邮"的体验价

结果呢？全球的创客、学生、工程师都涌过来了。今天嘉立创已经是全球最大的PCB打样厂商之一，每天出货数万单。

> 你手里那5片"5块钱包邮"的绿油板，背后是一个从华强北柜台起步、做到全球第一的中国制造故事 🇨🇳
