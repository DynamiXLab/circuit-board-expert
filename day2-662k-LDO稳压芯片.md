# 元件档案 · 662K — 3.3V低压差线性稳压器（LDO）

> **一句话认识它**：一颗只有三只脚、比米粒还小、花8分钱就能买到的3.3V稳压芯片。
> **封装**：SOT-23-3（三个引脚的小贴片，长得像三极管）
> **售价**：≈0.08元/颗（立创商城）

---

## 1. 长什么样？

### 外观与引脚定义

```
         俯视图（看芯片正面，有字朝上）
              __
             /  \
         ┌──┤662├──┐   ← 正面丝印写 "662K"
         │  \__/    │
         │          │
       ①脚       ②脚       ③脚
       GND      Vout      Vin
       (地)     (3.3V输出) (5V输入)
```

**引脚说明（记住这个顺序！）**

| 引脚 | 名称 | 功能 | 在电路中接什么 |
|:----:|:----:|------|----------------|
| **①** | **GND** | 地——0V参考点 | 接GND（大面积铺铜） |
| **②** | **Vout** | **3.3V稳压输出** | 接负载（传感器/MCU等） |
| **③** | **Vin** | 电源输入（**最高6.0V**） | 接5V（来自MP1584的输出） |

> ⚠️ **重要**：引脚顺序是 **①GND ②Vout ③Vin**，不是③①①的顺序！焊之前一定要用万用表通断档确认。

### 实物大小对比

```
    │662K│    │ AMS1117  │
    │SOT│    │  SOT-223 │    ← 662K只有AMS1117的1/4大
    │23 │    │          │
    └────┘    └──────────┘
    ~3mm       ~6.5mm
```

## 2. 内部原理——"智能可变电阻"

### 2.1 用比喻理解

**662K的内部就像一个自动调节的可变电阻：**

```
   Vin (5V) ──┬── [智能可变电阻] ──┬── Vout (3.3V)
               │                     │
               │      ↑              │
               │   [控制电路] ← 基准电压 0.8V
               │                     │
               └── GND ──────────── ─┴── GND
```

**工作过程**（闭环控制）：（复习：睁眼走路）

```
① 控制电路"看"输出电压（Vout）
② 和内部基准（0.8V）比较：
   - Vout太高？→ 增大电阻 → 压降变大 → Vout降回来
   - Vout太低？→ 减小电阻 → 压降变小 → Vout升回去
③ 回到①，不断循环
```

> 💡 整个过程每秒调整**成千上万次**，所以输出非常稳定。

### 2.2 核心参数

| 参数 | 值 | 什么意思 |
|------|:---:|---------|
| **输入电压范围** | ≤ **6.0V** | ⚠️ **不能接电池！** 必须接5V |
| **输出电压** | **3.3V（固定）** | 买回来就是3.3V，不需要调 |
| **最大输出电流** | **250mA** | 给传感器够用了（总共不到100mA） |
| **压差（Dropout）** | ~0.2V @ 90mA | 输入只比输出高0.2V就能工作 |
| **静态电流（Iq）** | **25μA** | 自己几乎不耗电，适合电池供电 |
| **精度** | ±3% | 输出3.3V时，实际在3.2~3.4V之间 |
| **输出噪声** | 极低（μV级） | 比Buck干净得多，适合模拟电路 |
| **工作温度** | -40°C ~ +85°C | 四季都能用 |

## 3. 典型应用电路

### 3.1 完整电路（就这几个元件）

```
               ┌──────────┐
   5V (来自Buck) ───┤③ Vin    │
                    │         │
   C1 ──┤├──┬──────┤① GND    │
   1μF     │      │         │
           │      │② Vout ──┼──┬── C2 ──┤├── GND
          GND     └──────────┘  │   1μF
                                │
                           Vout (3.3V)
```

**就这么多——输入一个电容、输出一个电容。**

### 3.2 元件说明

| 元件 | 规格 | 作用 | 为什么是这个值 |
|------|------|------|--------------|
| **C1 (输入电容)** | 1μF MLCC 0603 | 防止输入电压抖动 | 662K内部已稳压，外部1μF足够 |
| **C2 (输出电容)** | 1μF MLCC 0603 | 让3.3V输出更平滑 | 传感器电流小(<100mA)，1μF够用 |

> 对比：如果你用AMS1117，需要10μF电解+100nF各两个（4个电容）。662K只用**2个0603小电容**——省地方、省成本。

## 4. 为什么选662K而不是AMS1117？

### 4.1 对比表

| 对比项 | 662K（本模块选用） | AMS1117-3.3（传统方案） |
|-------|-------------------|----------------------|
| **封装** | **SOT-23-3**（极小） | SOT-223（较大） |
| **价格** | **~0.08元** | ~0.3元 |
| **静态电流** | **25μA**（几乎不耗电） | ~5mA（一直耗电） |
| **所需外围** | **2个1μF** | 4个电容（10μF+100nF×2） |
| **输出电流** | 250mA（够用） | 1A（大材小用） |
| **输入电压** | ≤6.0V（限制！） | ≤15V（更宽） |
| **PCB占板面积** | **~3mm²** | ~20mm² |

### 4.2 选择理由

| 理由 | 说明 |
|------|------|
| **① 够用≠浪费** | 传感器总共不到100mA，250mA绰绰有余 |
| **② 省电是关键** | 662K静态电流25μA vs AMS1117的5mA——差了**200倍**！巡线小车用电池，省电很重要 |
| **③ 越小越好** | SOT-23-3比SOT-223小得多，给巡线小车主板省下宝贵的板面空间 |
| **④ 外围简单** | 两个1μF电容搞定，焊接也简单 |
| **⑤ 便宜** | 8分钱一片，跟白送差不多 |

### 4.3 什么场景应该用AMS1117？

如果项目要求 **输出 > 250mA**（比如带大功率传感器、多个舵机），或者输入电压 > 6V，那就得用AMS1117。但在我们的巡线小车上，传感器电流总和 < 100mA——662K正好。

## 5. 容易犯的错误

| ❌ 错误 | 后果 | ✅ 正确做法 |
|---------|------|------------|
| 把662K的Vin直接接7.4V电池 | 芯片烧毁（最高6V！） | 必须接Buck的5V输出 |
| 引脚搞错（把Vin当GND焊） | 不工作或烧芯片 | 焊前用万用表确认①=GND ②=Vout ③=Vin |
| 省掉输入电容C1 | 输出电压抖动 | 必须焊C1（1μF），紧贴芯片 |
| 输出电流超过250mA | 芯片过热保护/输出电压下降 | 确认负载电流 ≤ 250mA |
| 把662K靠近电感放 | 输出噪声增大 | 662K尽量远离MP1584的电感和SW区域 |
